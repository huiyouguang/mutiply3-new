## 二、考点精讲（分模块拆解）
### 第一节 存货的确认和初始计量
#### 1. 存货的概念（判断“是否属于存货”是高频考点）
#存货概念
- **定义**：企业在日常活动中持有以备出售的产成品/商品、处在生产过程中的在产品、在生产/提供劳务过程中耗用的材料和物料。
- **具体范围**（计入“存货”项目）：<span class="cloze-span">原材料、产成品（库存商品）、周转材料、发出商品、委托加工物资、在途物资、在产品（通过“生产成本”核算）。</span>
- **排除项**：<span class="cloze-span">工程物资（用于在建工程，非日常出售/耗用）、合同资产（属于收款权利，非实物资产）。</span>
- **典例对应**：典例2-1（应在“存货”列报的项目为A.发出商品、B.生产成本、C.委托加工物资，D.合同资产不属于，答案：ABC）。

---

 

#### 2. 存货的确认条件（与“资产确认条件”逻辑一致，侧重存货场景）
需同时满足两个条件：
1. 与该存货有关的**经济利益很可能流入企业**（如已签订销售合同的存货，需判断履约可能性）；
2. 该存货的**成本能够可靠地计量**（如外购存货需有明确采购凭证，自行生产需能归集直接材料/人工等）。

---



#### 3. 存货的初始计量（★★，计算类考点核心）
存货初始计量原则：**按取得时的成本计量**，不同取得方式的成本构成不同，具体如下：

| 取得方式         | 成本构成公式                                             | 关键易错点（高频命题角度）                                                                                                                                          | 典例对应                                       |
| ------------ | -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------ |
| 外购存货         | 购买价款（含运输途中合理损耗）+ 相关税费 + 运输费 + 装卸费 + 保险费 + 入库前挑选整理费 | ① 合理损耗计入成本，非合理损耗（如意外灾害）暂记“待处理财产损溢”，查明后计入营业外支出/管理费用；<br>② 增值税：一般纳税人进项可抵扣（不计入成本），小规模纳税人进项不可抵扣（计入成本）；<br>③ 仓储费：入库前计入成本，入库后一般计入当期损益，生产必需的仓储费（如酒的窖藏）计入加工成本。 | 典例2-2（答案B：100万价款+1万运费=101万，增值税抵扣）          |
| 委托加工物资       | 发出存货成本 + 加工费 + 运输费 + 相关税费（增值税+消费税）                 | ① 增值税：一般纳税人进项可抵扣（不计入），小规模纳税人计入成本；<br>② 消费税：收回后**直接出售**计入成本，收回后**连续生产应税消费品**计入“应交税费——应交消费税”（可抵扣）。                                                       | 典例2-4（答案ABD：原材料30万+加工费6万+消费税4万，增值税0.38万抵扣） |
| 自行生产存货       | 直接材料 + 直接人工 + 制造费用（如车间折旧费、季节性停工损失）                 | 非正常消耗（如超定额废品损失、自然灾害损失）不计入成本，计入当期损益（营业外支出/管理费用）。                                                                                                        | -                                          |
| 其他方式（如投资者投入） | 按投资合同/协议约定的价值（约定不公允的，按公允价值）                        | 需同时确认增值税进项税额（如投资方开具专票），差额计入“资本公积——资本溢价”。                                                                                                               | -                                          |

### 第二节存货的期末计量（★★★，主观题核心考点）


---


#### 1. 期末计量原则
资产负债表日，存货按**成本与可变现净值孰低**计量：
- 若存货成本 > 可变现净值：计提存货跌价准备，计入当期损益（“资产减值损失”）；
- 若存货成本 ≤ 可变现净值：无需计提，按成本计量。

---


#### 2. 关键概念辨析
| 概念    | 定义                                            |
| ----- | --------------------------------------------- |
| 存货成本  | 存货的账面余额（即“原材料”“库存商品”等科目的期末余额，未扣备抵科目）          |
| 可变现净值 | 日常活动中，存货的估计售价 - 至完工时估计将要发生的成本 - 估计销售费用 - 相关税费 |
| 账面价值  | 存货成本 - 存货跌价准备（期末列报金额）                         |
|       |                                               |


---

#### 3. 可变现净值的计算（需严格区分存货用途，高频计算点）
| 存货用途        | 可变现净值计算公式                                  | 关键注意事项                                                   |
| ----------- | ------------------------------------------ | -------------------------------------------------------- |
| 用于直接销售的库存商品 | 库存商品估计售价 - 估计销售费用 - 相关税费                   | 有合同部分按“合同价格”算估计售价，无合同部分按“市场价格”算，**需分开计算**（避免增值覆盖减值）      |
| 用于生产产品的原材料  | 所生产产品的估计售价 - 产品进一步加工成本 - 产品估计销售费用 - 产品相关税费 | 不直接用原材料自身市场价格，需先判断“产品是否减值”：产品未减值→原材料不减值；产品减值→原材料才需算可变现净值 |
| 用于直接销售的原材料  | 原材料自身估计售价 - 估计销售费用 - 相关税费                  | 与“用于销售的库存商品”逻辑一致，直接用自身售价计算                               |
|             |                                            |                                                          |


---

#### 4. 存货跌价准备的会计处理
- **计提分录**：  
  借：资产减值损失  
  贷：存货跌价准备  
- **转回分录**（仅当“以前减记存货价值的影响因素消失”时，在原计提范围内转回）：  
  借：存货跌价准备  
  贷：资产减值损失  
- **结转分录**（存货出售/用于在建工程时，需同步结转已计提的跌价准备）：  
  ① 出售时：  
  借：主营业务成本  
      存货跌价准备  
  贷：库存商品  
  ② 用于在建工程时：  
  借：在建工程  
      存货跌价准备  
  贷：原材料/库存商品

## 三、本章核心易错点总结
1. **存货范围误区**：工程物资、合同资产不属于存货；“生产成本”（在产品）、“发出商品”（控制权未转移）属于存货。
2. **外购存货成本误区**：非合理损耗（如运输中被盗）不计入成本；入库后仓储费一般计入“管理费用”，生产必需的仓储费（如酸奶发酵）计入存货成本。
3. **委托加工消费税误区**：只有“收回后直接出售”的消费税才计入成本，“连续生产应税消费品”的消费税需计入“应交税费”（可抵扣）。
4. **期末计量误区**：生产用原材料需先判断“产品是否减值”，而非直接用原材料市场价格对比成本；有合同与无合同存货需分开计算可变现净值，不得合并。
