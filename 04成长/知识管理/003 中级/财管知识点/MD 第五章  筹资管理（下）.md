## 一、资金需要量预测
### （一）[[因素分析法]]
- **含义**：以有关项目基期年度的平均资金需要量为基础，依据预测年度的生产经营任务和资金周转加速要求，进行分析调整来预测资金需要量。
- **公式**：资金需要量＝(基期资金平均占用额－不合理资金占用额)×(1 + 预测期销售增长率)÷(1 + 预测期资金周转速度增长率)
- **示例**：甲企业上年度资金平均占用额2500万元，不合理部分300万元，预计本年度销售增长5%，资金周转减慢3%，则预测年度资金需要量＝(2500－300)×(1＋5%)/(1 - 3%)＝2381.44万元 。
- **特点**：计算简便易掌握，但预测结果不太精确，适用于品种繁多、规格复杂、资金用量较小的项目。

### （二）[[销售百分比法]]
- **基本原理**：假设部分资产和负债与销售额存在稳定百分比关系，并非所有资产和负债项目都如此。
- **基本步骤**
    - **确定敏感项目**：经营性资产如库存现金、应收账款、[[存货]]；经营性负债如应付票据、应付账款，不包括短期借款等筹资性负债。
    - **确定比例关系**：题中说明某项资产或负债占销售收入的比例，即视为敏感资产或负债。
    - **计算筹资数量**：外部融资需求量＝敏感性资产总额占销售收入的百分比×销售收入增长额 + 固定资产投资增加(如果有)－敏感性负债总额占销售收入的百分比×销售收入增长额－预计销售收入×销售净利率×收益留存率。若存在非敏感性资产增加，需加上Δ非敏感性资产。
    - **示例**：上年销售收入5000万元，现增至8000万元。上年敏感性资产5600万元(占销售收入的112%)，敏感性负债1800万元(占销售收入的36%)，则需要增加的资金量=(8000×112% - 5000×112%) - (8000×36% - 5000×36%) 。
- **优缺点**
    - **优点**：简便易用，能提供预计短期财务报表，为筹资决策提供依据，适用于外部筹资需要。
    - **缺点**：假定敏感资产和负债占比不变，若此假定不成立，需调整原销售百分比。

### （三）[[资金习性预测法]]
- **资金习性含义**：资金总额（y）＝不变资金（a）＋变动资金（bx），可利用回归分析法或高低点法估计参数a和b 。
    - **不变资金**：在一定产销量范围内，不受产销量变动影响的固定资金，如最低现金、保险储备、固定资产占用资金等。
    - **变动资金**：随产销量变动同比例变动的资金，如构成产品实体的原材料、最低储备外的现金等。
    - **半变动资金**：受产销量变化影响但不同比例变动，如辅助材料占用资金。
- **预测方法**
    - **根据资金占用总额与产销量的关系预测**：设产销量为自变量X，资金占用量为因变量Y，Y＝a＋bX。例如A企业相关数据计算得Y = 442 + 0.29X ，20×7年预计销售量2500万件，资金需要量为442 + 0.29×2500 = 1167万元 。
    - **采用逐项分析法预测**：根据高点和低点数据代入直线方程y＝a＋bx求a和b。

## 二、[[资本成本]]
### （一）资本成本概述
- **含义**：企业为筹集和使用资本付出的代价，包括筹资费用和用资费用。
    - **筹资费用**：资本筹措过程中获取资本的代价，视为筹资数额扣除，如发行费、手续费等。
    - **用资费用**：使用过程中占用资本的代价，是主要内容，如利息、股息等。
- **作用**
    - 是比较筹资方式、选择筹资方案的依据，应选资本成本率最低的方式。
    - 平均资本成本衡量资本结构合理性，最小时企业价值最大。
    - 评价投资项目可行性的主要标准，预期收益率超资本成本率时项目可行。
    - 评价企业整体业绩的重要依据，总资产税后收益率应高于平均资本成本率。
- **影响因素**
    - **总体经济环境**：经济健康稳定、通胀低，投资者风险小，必要收益率低，资本成本低。
    - **资本市场条件**：市场缺乏效率，证券流动性低，投资者风险大，必要收益率高。
    - **经营状况和融资状况**：企业经营和财务风险高，总体风险高，投资者要求收益率高。
    - **筹资规模和时限**：资金规模大、占用时限长，资本成本高，但非严格线性关系。

### （二）资本成本的计算
- [[个别资本成本]]的计算**
    - **计算模式**
        - **一般模式**：不考虑资金时间价值，主要用于债务资本成本计算，债务资本成本率＝年利息×(1－企业所得税税率)/(筹资总额－筹资费用) 。
        - **折现模式**：使现金流入现值与现金流出现值相等的折现率，考虑资金时间价值，适用于金额大、借贷时间长的情形。
    - **银行借款**
        - **示例**：某企业5年期长期借款500万元，年利率8%，每年付息一次，到期一次还本，借款费用率0.3%，企业所得税税率25%。
        - **一般模式**：银行借款的资本成本=8%×(1 - 25%)/(1 - 0.3%) =6.02%
        - **折现模式**：通过计算可得Kb＝6.07%
    - **公司债券**
        - **示例**：某企业溢价20%发行面值1000元、期限5年、票面利率8%的公司债券，每年付息，到期还本，发行费用率5%，所得税税率25%。
        - **一般模式**：（文档未完整给出计算结果，暂略）
        - **折现模式**：Kb = 2.95%
    - **优先股**
        - **一般模式**：固定股息率优先股，Ks = 年固定股息÷[发行价格×(1 - 筹资费用率)]
        - **示例**：某优先股相关计算，贴现模式下K = 2.48%
    - **普通股**
        - **股利增长模型法**：普通股资本成本=[D0×（1 + g）]/[P0×（1 - 3%）]+g 。示例中普通股资本成本 = 7.09%
        - **资本资产定价模型法**：普通股资本成本＝无风险利率＋β×(市场平均收益率－无风险利率) 。示例中普通股资本成本＝19%
    - **留存收益**：与普通股资本成本计算相同，不考虑筹资费用。
- **平均资本成本的计算**：以各项个别资本占总资本比重为权数加权平均，包括账面价值权数、市场价值权数、目标价值权数，各有优劣。
- **边际资本成本的计算**：企业追加筹资的加权平均成本，是追加筹资决策依据，权数采用目标价值权数。
- **项目资本成本的确定**
    - **使用公司当前综合资本成本**：需项目风险与公司当前资产平均风险相同且继续采用相同资本结构为新项目筹资。示例中综合资本成本＝10.2% 。
    - **运用可比公司法**：通过卸载、加载[[财务杠杆]]等步骤确定。

## 三、杠杆效应
### （一）[[经营杠杆]]效应
- **定义**：因固定性经营成本存在，资产收益（息税前利润）变动率大于业务量变动率。
- **经营杠杆系数的计算**
    - **定义公式**：经营杠杆系数=息税前利润变动率/业务量变动率
    - **推导公式**：经营杠杆系数=基期边际贡献/基期息税前利润 。

### （二）[[财务杠杆]]效应
- **定义**：因固定性资本成本（固定利息、优先股股利、固定租赁费）存在，普通股收益（或每股收益EPS）变动率大于息税前利润变动率。
- **[[财务杠杆]]系数的计算**
    - **定义公式**：[[财务杠杆]]系数=普通股收益变动率/息税前利润变动率
    - **推导公式**：[[财务杠杆]]系数=息税前利润/（息税前利润－利息－税前优先股股息） 。

### （三）[[总杠杆]]效应
- **定义**：因固定经营成本和固定资本成本存在，普通股每股收益变动率大于产销业务量的变动率。
- **总杠杆系数的计算**
    - **定义公式**：总杠杆系数=普通股每股收益变动率/产销业务量变动率
    - **推导公式（不存在优先股股息时）**：DTL＝M0/（M0 - F0 - I0） 。

## 四、[[资本结构]]
### （一）资本结构的基础知识
- **含义**
    - **广义**：全部债务与股东权益的构成比例。
    - **狭义**：长期负债与股东权益的构成比例。
    - **最优标准**：平均资本成本率最低、企业价值最大。
- **资本结构理论**
    - **MM理论**
        - **无税的MM理论**：企业价值不受资本结构影响，权益资本成本随负债比例增加而加大。
        - **修正的MM理论**：债务利息可抵税，企业价值随债务资本比例上升而增加。
    - **权衡理论**：有负债企业的价值=无负债企业的价值+税赋节约现值-财务困境成本现值。
    - **代理理论**：债务筹资减少股权代理成本，但增加债权代理成本，需平衡两者关系。
    - **优序融资理论**：融资顺序为先内后外，先债后股。
- **影响因素**
    - **经营状况和成长率**：稳定性好可多负担固定财务费用；增长率高可采用高负债资本结构。
    - **财务状况和信用等级**：良好则易获债务资金。
    - **资产结构**：固定资产多通过发行股票融资；流动资产多依赖流动负债；资产适用于抵押则负债多；技术研发为主负债少。
    - **投资人和管理当局的态度**：股权分散多采用权益资本筹资；股权集中避免普通股筹资；稳健管理当局偏好低负债比例。
    - **行业特征和企业发展周期**：成熟产业可提高债务资金比重；高新技术企业降低比重；初创阶段控制负债，成熟阶段适度增加，收缩阶段降低。
    - **经济环境的税务政策和货币政策**：所得税税率高时，债务资金抵税作用大；紧缩货币政策下，债务资金成本增大。

### （二）资本结构优化
- [[**每股收益分析法**]]：计算息税前利润平衡点，选择最优筹资方案。
- [[**平均资本成本比较法**]]：计算不同方案平均资本成本，选最低的方案。
- [[**企业价值分析法**]]：计算企业价值V和加权平均资本成本，以公司市场价值为标准优化。

### （三）[[双重股权]]结构
- **含义**：同股不同权，投票权和分红权分离。
- **优点**：保障创始人或管理层控制权，防止恶意收购，提高运行效率，利于长期发展。
- **缺点**：易导致独裁，损害非控股股东利益，加剧经营者道德风险和逆向选择。
