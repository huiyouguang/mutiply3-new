# 第一章 总论
## 1 一、法律行为   #熟悉
（一）法律行为的概念和特征(★)
法律行为是指**民事主体通过意思表示设立、变更、终止民事法律关系的行为**。其中，  “意思表示”是指行为人将其期望发生法律效果的内心意思，以定方式表达于外部的行为。
1. 需要与事实行为做对比
	1. 当事人决定的，如上面，是法律行为
	2. 法律规定的，是事实行为
2. 法律明确规定后果的
3. 例题
	1. 根据民事法律制度的规定，下列各项中，属于民事法律行为的是（       ）。
		1. A. 张某观测宇宙黑洞
		2. B. 王某与机器人对弈
		3. C. 李某购买考试教材
		4. D. 刘某食用新鲜草莓
			1. 斯尔解析   本题考查法律行为的辨识。在本题中，将法律行为与其他行为混在一起，要求考生选出其中的法律行为。面对这样的题目，解题时要分两步：
			2. 第一步，这个行为会法律关系造成影响（产生、改变、消灭权利义务）吗？
			3. 选项 AB 不当选，“张某观测宇宙黑洞”“王某与机器人对弈”一般不对法律关系造成影响，即不会导致法律上权利义务的产生、改变或者消灭。
			4. 第二步，这个行为对法律关系造成的具体影响受行为人意志的左右吗？
			5. （1）选项 C 当选，李某与书店店主之间构成买卖合同的法律关系， 即该行为导致法律关系的产生。进一步，该买卖合同法律关系中，李某的具体权利（买哪本书） 、具体义务（付多少钱） ，以及店主的具体权利（收多少钱） 、具体义务（卖哪本书）都是由李某与书店老板的内心意思决定的。
			6. （2）选项 D 不当选，刘某食用新鲜草莓后，导致刘某对于草莓拥有的所有权灭失，即消灭了物权法律关系。但是，  无论刘某吃掉该草莓时内心是怎么想的，只要其将草莓吃掉，就会导致草莓上物权法律关系的消灭。因此，刘某对草莓拥有的所有权灭失与刘某的内心意思并无关系。
			7. 在这个过程中，<span class="cloze-span">刘某内心意思不影响法律后果，该行为属于事实行为，其法律效果由法律直接规定。类似的行为还有建造房屋 、拾得遗失物等行为。</span>

## 2 （二）法律行为的分类（★）

| 分类标准 | 具体分类 | 基本概念 | 举例 |
| --- | --- | --- | --- |
| 法律行为的成立需要几方当事人的意思表示 | 单方法律行为 | 依一方当事人的意思表示而成立的法律行为 | 委托代理的撤销、无权代理的追认等 |
|  | 多方法律行为 | 依两个或两个以上当事人意思表示一致而成立的法律行为 | 订立合同、设立公司的协议等 |
| 法律行为一方当事人从对方当事人取得利益有无对价为标准 | 有偿法律行为 | 双方当事人各因给付而从对方取得利益的法律行为 | 买卖、租赁、承揽等 |
|  | 无偿法律行为 | 当事人一方无须为给付而获得利益的法律行为 | 赠与、无偿委托、借用等 |
| 法律行为的成立是否需要具备法律规定或当事人约定的形式 | 要式法律行为 | 法律明确规定或当事人明确约定必须采取一定形式或履行一定程序才能成立的法律行为 | 应采用书面形式订立的合同：<br>（1）融资租赁合同<br>（2）建设工程合同<br>（3）技术开发合同<br>（4）担保合同（抵押合同、质押合同、保证合同）<br>（5）除自然人之间借款合同以外的其他借款合同 |

| 法律行为之间的依存关系而进行的分类 | 主法律行为 | 不需要有其他法律行为的存在就可以独立成立的法律行为 | 借款合同 |
|  | 从法律行为 | 从属于其他法律行为而存在的法律行为 | 担保合同（抵押合同、质押合同、保证合同） |

## 3 （三）法律行为的成立要件与生效要件  #掌握
1. 成立要件
**意思表示即为法律行为的一般成立要件**。此外， 特定法律行为还要求具备特别成立要件，如对于要式法律行为，还需要具备法律规定或当事人约定的特定形式才能成立。
2. 生效要件
法律行为的成立是法律行为生效的前提，但是，已成立的法律行为不一定必然发生法律效力，只有具备一定生效条件的法律行为，才能产生预期的法律效果。
根据《民法典》的规定，法律行为应当具备下列生效要件：
3. （1）==行为人具有相应的民事行为能力。==
	1. 民事行为能力，指民事主体能够独立参加民事法律关系，以自己的行为取得民事权利或承担民事义务的能力，即独立实施法律行为的能力
4. （2）==意思表示真实。==
5. （3）==不违反强制性规定，不违背公序良俗。==
### 3.1 民事行为能力分类表
| 行为能力类型 | 标准（年龄） | 标准（心智） | 独立实施法律行为的效力 |
|--------------|--------------|--------------|------------------------|
| 无民事行为能力 | 不满8周岁的未成年人 | 不能辨认自己行为的人 | 无效 |
| 限制民事行为能力 | 8周岁以上、不满18周岁的未成年人 | 不能完全辨认自己行为的人 | 有效：<br>①纯获利益的法律行为<br>②与其年龄、智力或精神健康状况相适应的法律行为<br><br>效力待定：除上述行为以外的其他法律行为 |
| 完全民事行为能力 | 一般：18周岁以上的成年人<br>视为：16周岁以上不满18周岁但以自己的劳动收入为主要生活来源的未成年人 | 心智正常 | 一般有效 |

### 3.2 解读
民事行为能力是自然人独立实施法律行为的资格，核心是**年龄+心智状态**的双重判断：
1. **无民事行为能力人**：无法独立实施任何有效法律行为，其行为需由法定代理人代理。
2. **限制民事行为能力人**：仅能独立实施“纯获利”或“与自身能力匹配”的行为（如购买小额文具），超出范围的行为需经法定代理人追认才有效。
3. **完全民事行为能力人**：可独立实施各类法律行为（如签订合同、处分财产），需自行承担行为后果。

其中“视为完全民事行为能力人”的规定，是对以自己劳动独立生活的未成年人的特殊认可，体现了法律对实际行为能力的尊重。

## 4 （四）民事法律行为效力瑕疵（★★）

### 4.1 无效民事法律行为
无效的法律行为，是指当事人所追求的法律效果自始、当然、确定不发生的法律行为。
#### 4.1.1 （1）种类
| 记忆提示 | 具体规定                |
| ---- | ------------------- |
| “无”  | 无民事行为能力人独立实施的法律行为   |
| “假”  | 当事人通谋虚假表示实施的法律行为    |
| “串”  | 恶意串通，损害他人合法权益的法律行为  |
| “违”  | 违反强制性规定或违背公序良俗的法律行为 |

#### 4.1.2 （2）法律效力
无效民事法律行为自始无效。民事法律行为部分无效，不影响其他部分效力的，其他部分仍然有效。
#### 4.1.3 （3）法律后果
| 记忆提示 | 具体规定                                                    |
| ---- | ------------------------------------------------------- |
| 恢复原状 | 恢复到无效法律行为发生之前的状态，当事人因该行为取得的财产应当返还给受损失的一方                |
| 赔偿损失 | 有过错的一方应当赔偿对方因此所受的损失。如果双方都有过错的，应当各自承担相应的责任               |
| 返还收归 | 双方恶意串通，实施法律行为损害国家、集体或第三人利益的，应当追缴双方取得的财产，收归国家、集体所有或返还第三人 |
| 其他制裁 | 依法承担适当的行政责任或刑事责任                                        |

### 4.2 （2）可撤销民事法律行为效力。
1. ①撤销权人对撤销权的行使拥有选择权，其可以选择撤销或不撤销其法律行为。可撤销法律行为被撤销前，其效力已经发生；未经撤销，其效力不消灭。
2. ②可撤销法律行为被依法撤销后，法律行为从行为开始起无效。
3. ③如果撤销权人表示放弃撤销权或未在法定期间内行使撤销权的，则可撤销法律行为确定地成为完全有效的法律行为。
4. 效力待定法律行为
	法律行为成立后尚未生效，须经权利人追认才能生效的法律行为是效力待定法律行为，具体包括：
	（1）限制民事行为能力人超出自己的行为能力范围独立实施的法律行为。
	（2）行为人没有代理权、超越代理权或者代理权终止后以被代理人名义实施（“无权代理”）的法律行为。

### 4.4 解读
可撤销民事法律行为是指因存在法定瑕疵，当事人可通过**诉讼/仲裁**主张撤销的行为，核心规则特点：
1. **撤销权主体**：欺诈、胁迫的撤销权专属于受损害方（受欺诈方、受胁迫方），重大误解、显失公平的撤销权主体未在表中明确，通常为存在误解或受不利影响的一方。
2. **期间限制**：采用“主观期间+客观期间”双重限制——
   - 主观期间：需在“知道/应当知道撤销事由”（或胁迫终止）后的**90日/1年**内行使；
   - 客观期间：无论是否知晓，行为发生后**5年**内未行使的，撤销权直接消灭。
3. **效力状态**：行为在被撤销前是有效的，撤销后才溯及地自始无效，区别于“自始无效”的无效法律行为。

（五）附条件和附期限的法律行为(★)
（1）“ 附条件的法律行为”是指当事人在法律行为中约定一定的条件，并以将来该条件的成就（或发生）或不成就（或不发生）作为法律行为效力发生或消灭的根据。
（2）“ 附期限的法律行为”是指当事人在法律行为中约定一定的期限，并以该期限的到来作为法律行为生效或解除的根据。
1. ①期限是必然要到来的事实，而条件却不一定发生，这是期限与条件的根本区别。
2. ②法律行为所附期限可以是明确的期限，例如某年某月某日，也可以是不确定的期限，例如“某人死亡之日”。

### 0.5 （四）程序

;#### 0.5.1 基本流程
申请 → 受理 → 组仲裁庭 → 裁决 → 执行  
（1）仲裁庭可以由 **1 名** 仲裁员成立独任仲裁庭或 **3 名** 仲裁员组成合议仲裁庭。由 3 名仲裁员组成的合议仲裁庭，设首席仲裁员。  
（2）回避。  
仲裁员有下列情况之一的，必须回避，当事人也有权在首次开庭前提出回避申请：  
①是本案当事人，或者当事人、代理人的近亲属。  
②与本案有利害关系。  
③与本案当事人、代理人有其他关系，可能影响公正仲裁的。  
④私自会见当事人、代理人，或者接受当事人、代理人的请客送礼的。  
（3）仲裁庭不能形成多数意见时，应当按照 **首席仲裁员** 的意见作出裁决，即 “先多数后首席” 。  
（4）仲裁应当开庭进行，但当事人可以协议不开庭。仲裁一般 **不公开** 进行，当事人协议公开的，可以公开进行，但涉及国家秘密的除外。  
（5）一方当事人不履行仲裁裁决的，另一方当事人可以向人民法院申请执行。  

#### 0.5.1 受理后仲裁庭的处理  
申请仲裁后，当事人可以自行和解。仲裁庭在作出裁决前，可以先行调解，当事人自愿调解的，仲裁庭应当调解；调解不成的，应当及时作出裁决。具体规则如下：  
1. 和解：  
    - 达成和解协议 → 可请求仲裁庭根据和解协议作出裁决书，也可撤回仲裁申请  
    - 撤回仲裁申请后反悔 → 可根据仲裁协议 **重新仲裁**  
2. 调解：  
    - 调解达成协议 → 仲裁庭制作调解书（经双方当事人签收后生效 ），或根据协议结果制作裁决书  
    - 调解不成 → 及时作出裁决  
3. 裁决：  
    - 裁决书自作出之日起发生法律效力，实行 **一裁终局**  
    - 对裁决不服 → 不得就同一纠纷再申请仲裁或向法院起诉，但可依法向法院申请 **撤销裁决** 或 **不予执行** ，若经法院裁定撤销/不予执行，当事人可重新达成仲裁协议申请仲裁，或向法院 **提起民事诉讼**

## 1 二、民事诉讼




### 1.1 一、合议制度
合议制度是指由 3 名以上审判人员组成审判组织，代表人民法院行使审判权，对案件进行审理并作出裁判的制度。  
- （1）合议制度是一种基本的审判组织形式。  
- （2）合议庭的成员，应当是 <span class="cloze-span">3 </span>人以上的单数。  
- （3）与合议制相对的**独任制度**，由一名审判员独立对案件进行审理和裁判，适用于特定情形。我们在后文会介绍，适用简易程序审理的案件采用独任制。  


### 1.2 二、回避制度
审判人员、法官助理、书记员、司法技术人员、翻译人员、鉴定人、勘验人有下列情形之一的，应当自行回避，当事人有权用口头或者书面方式申请他们回避：  
- ①是<span class="cloze-span">本案当事人或者当事人、诉讼代理人近亲属的。  </span>
- ②与<span class="cloze-span">本案有利害关系的。  </span>
- ③与<span class="cloze-span">本案当事人、诉讼代理人有其他关系，可能影响对案件公正审理的。  
</span>
**提示**：证人不适用回避制度。  


### 1.3 三、公开审判制度
公开审判包括审判过程公开和审判结果公开两项内容，为方便同学们对比记忆，我们将在后文与行政诉讼一起介绍。  


### 1.4 四、两审终审制度
两审终审制度是指一个诉讼案件经过两级人民法院审判后即终结的制度。但下列情形，适用于一审终审：  
- （1）适用特别程序、督促程序、公示催告程序和简易程序中的小额诉讼程序审理的案件实行**一审**终审。  
- （2）最高人民法院所作的一审**判决、裁定，为终审判决、裁定。**

# （三）诉讼时效  



诉讼时效是指权利人不在法定期间内行使权利而失去诉讼保护的制度。  

## 1 特点  
- （1）诉讼时效期间届满，**债务人获得抗辩权**，但债权人的起诉权、实体权利并不消灭。由此：  
	  - ①诉讼时效期间届满后，当事人自愿履行义务的，不受诉讼时效限制。  
	  - ②义务人履行了义务后，又以诉讼时效期间届满为由抗辩的，法律不予支持。  
- （2）诉讼时效的期间、计算方法以及中止、中断的事由由**法律规定**，当事人约定无效。  
- （3）义务人未提出诉讼时效抗辩的，人民法院不应对诉讼时效问题进行释明及主动适用诉讼时效的规定进行裁判。  


## 2 诉讼时效的适用范围  
诉讼时效适用于**请求权**。下列情况不适用诉讼时效：  

| 记忆提示       | 具体规定                                                                 |
|----------------|--------------------------------------------------------------------------|
| 物权请求停排消 | 请求停止侵害、排除妨碍、消除危险                                       |
| 不动登记返财产 | 不动产物权和登记的动产物权的权利人请求返还财产                         |
| 人身相关找三养 | 请求支付抚养费、赡养费或者扶养费                                       |
| 存债本利加投资 | （1）支付存款本金及利息请求权；<br>（2）兑付国债、金融债券等本息请求权；<br>（3）基于投资关系的缴付出资请求权 |  


## 3 诉讼时效的长度及起算  



### 3.1 （1）普通时效 vs 长期时效  
| 种类       | 长度  | 起算时间                   | 备注            |     |
| -------- | --- | ---------------------- | ------------- | --- |
| 普通时效（主观） | 3年  | 权利人知道/应当知道权利受损 + 知道义务人 | 适用中止、中断，不适用延长 |     |
| 长期时效（客观） | 20年 | 权利受侵害之日                | 适用延长，不适用中止、中断 |     |


### 3.2 （2）特殊起算点规定  
| 分类                                   | 起算点                     |
|----------------------------------------|----------------------------|
| 无/限制民事行为能力人对法定代理人的请求权 | 法定代理终止之日           |
| 未成年人被性侵                         | 受害人年满18周岁之日       |
| 附条件/附期限之债                       | 条件成就/期限届满之日      |
| 不作为之债                             | 债权人知道债务人作为之时   |
| 侵权之债                               | 权利人知道侵权事实+加害人  |
| 分期履行之债                           | 最后一期履行期限届满       |  


## 4 诉讼时效期间的中止和中断  


### 4.1 （1）诉讼时效中止  
| 项目       | 具体规定                                                                 |
|------------|--------------------------------------------------------------------------|
| 原因（法定事由） | 不可抗力（如自然灾害、军事行动）；<br>其他障碍（无/限制行为能力人无代理人、继承未确定继承人等） |
| 发生时间   | 诉讼时效**最后6个月内**                                                   |
| 法律效力   | 法定事由消除后，**重新起算6个月**诉讼时效                                 |  


### 4.2 （2）诉讼时效中断  
发生于诉讼时效内，因法定事由使已过时效期间归无效，**重新计算时效**。  

#### 4.2.1 ① 中断事由  
| 项目   | 具体规定                                                                 |
|--------|--------------------------------------------------------------------------|
| 权利人 | 提出履行要求；提起诉讼/申请仲裁                                         |
| 义务人 | 同意履行（分期、部分履行、请求延期等承诺）                               |
| 其他   | 与诉讼/仲裁等效的情形                                                   |  

#### 4.2.2 ② 法律效力  
- a. 中断事由发生后，已过时效期间**归无效**；  
- b. 中断事由存续期间（如诉讼程序中），**时效不计算**；  
- c. 中断事由终止时，**重新计算时效期间**。

# 三、行政复议与行政诉讼  

## 1 （一）基本概念梳理（★）  
### 1.1 
1.1 行政复议与行政诉讼的概念  
- （1）**行政复议**：行政相对人对行政行为争议，向行政复议机关申请，复议机关审查行政行为**合法性、适当性**并作决定的活动。  
- （2）**行政诉讼**：行政相对人认为行政行为侵权，向法院请求司法保护，法院审查行政行为**合法性**并裁判的司法活动。  


### 1.2 是否可以同时申请行政复议或者提起行政诉讼  

#### 1.2.1 （1）或复议或诉讼（选择型）  
- ①申请复议且受理的，**复议期间不得起诉**；  
- ②起诉且受理的，**不得申请复议**。  

#### 1.2.2 （2）先复议后诉讼（复议前置）  
- ①未申请复议不得起诉；复议机关不予受理/逾期不答复的，可在**收到决定/复议期满15日内**起诉。  
- ②具体情形（如纳税争议、自然资源权属争议等，见原文列举）。  

#### 1.2.3 （3）只复议不诉讼（复议终局）  
- 复议后**不得起诉**（流程：行政机关→复议→再复议→国务院终局裁决 ，无法诉讼 ）。  

#### 1.2.4 （4）或诉讼或裁决  
- 涉及“自我”复议，可选择**起诉或国务院终局裁决** 。  


### 1.3 申请行政复议与提起行政诉讼的情形  
行政复议与行政诉讼**受案范围基本一致**，具体对应关系如下（表格梳理）：  

| 项目       | 行政复议受案范围                          | 行政诉讼受案范围                    |     |
| -------- | --------------------------------- | --------------------------- | --- |
| 行政处罚     | 对行政处罚决定不服                         | 对行政拘留、暂扣/吊销许可证等行政处罚决定不服     |     |
| 行政强制     | 对行政强制措施、强制执行决定不服                  | 对限制人身自由、财产查封扣押等行政强制决定不服     |     |

| 行政征收、征用  | 对征收、征用决定及补偿决定不服                   | 对征收、征用决定及补偿决定不服             |     |
| 侵权       | 认为侵犯经营自主权、农村土地承包经营权               | 同行政复议范围                     |     |
| 滥用权利     | 认为行政机关滥用权力排除/限制竞争                 | —                           |     |
| 违法要求履行义务 | 认为行政机关违法集资、摊派费用等                  | —                           |     |
| 行政不作为    | 申请行政机关履行保护人身/财产/受教育权等职责，机关拒绝/未履行  | 申请行政机关履行保护人身、财产权职责，机关拒绝/未答复 |     |
| 行政给付     | 申请行政机关给付抚恤金、社保待遇等，机关未依法给付         | 认为行政机关未依法支付抚恤金、最低生活保障等      |     |

| 政府信息公开   | 认为行政机关在政府信息公开中侵权                  | —                           |     |
| “兜底”     | 认为行政机关其他行政行为侵权                    | 认为行政机关侵犯其他人身权、财产权等合法权益      |     |

## 2 (五) 行政诉讼与民事诉讼程序的对比 (★)

### 2.1 一、起诉
|项目|民事诉讼|行政诉讼|
| ---- | ---- | ---- |
|起诉条件|（1）原告是与本案有直接利害关系的公民、法人或其他组织<br>（2）有明确的被告<br>（3）有具体的诉讼请求和事实、理由<br>（4）属于人民法院受理民事诉讼的范围和管辖范围|（1）原告是认为行政行为侵犯其合法权益的公民、法人或者其他组织<br>（2）有明确的被告<br>（3）有具体的诉讼请求和事实根据<br>（4）属于人民法院受案范围和受诉人民法院管辖|
|时间|教材中未明确列示，但要注意特殊规定，如请求权适用诉讼时效的规定|主观起算 6 个月<br>客观起算：不动产 20 年；其他案件 5 年|

#### 2.2.2 （2）简易程序
在民事诉讼、行政诉讼、行政复议中，均可能适用简易程序，以下比较三者异同：
##### 2.2.2.1 ①适用情形

### 2.4 四、民事诉讼的审判监督程序（再审程序）
在民事诉讼与行政诉讼中均可能适用再审程序，但在中级经济法的考试中，仅考查过民事诉讼再审程序，故以下仅介绍民事诉讼的再审程序。
#### 2.4.1 （1）再审程序的启动
|主体|前提|处理方式|
| ---- | ---- | ---- |
|各级人民法院院长|本院已经发生法律效力的判决、裁定、调解书，发现确有错误|提交审判委员会讨论决定|
|①最高人民法院对地方各级人民法院<br>②上级人民法院对下级人民法院|已经发生法律效力的判决、裁定、调解书，发现确有错误|有权提审或指令下级人民法院再审|
|当事人 - 一般情况|已经发生法律效力的判决、裁定，认为有错误|向上一级人民法院申请再审|
|当事人 - 一方人数众多或双方为公民|已经发生法律效力的判决、裁定，认为有错误|向上一级或原审人民法院申请再审|

#### 2.4.2 （2）当事人申请再审，有下列情形之一的，人民法院不予受理：
- 再审申请被驳回后再次提出申请的  
- 对再审判决、裁定提出申请的  
- 在人民检察院对当事人的申请作出不予提出再审检察建议或者抗诉决定后又提出申请的  

#### 2.4.3 （3）当事人申请再审的，不停止判决、裁定的执行。  

#### 2.4.4 （4）当事人对已经发生法律效力的调解书申请再审，应当在调解书发生法律效力后 6 个月内提出。  


### 2.5 （六）民事诉讼、行政复议、行政诉讼中的期限规定

### 2.6 （七）行政复议与行政诉讼对比
| 项目   | 行政复议                                | 行政诉讼                         |
| ---- | ----------------------------------- | ---------------------------- |
| 提起方式 | 可书面、可口头                             | 可书面、可口头                      |
