　　1.纵向分工结构的基本类型

|   |   |   |
|---|---|---|
|（1）高长型组织结构|**管理层次多**。  <br>在每个层次上，管理人员的**控制幅度较窄**。|有利于企业内部的控制，但对市场变化的反应较慢。|
|（2）扁平型组织结构|**管理层次较少**。  <br>在每个层次上，管理人员的**控制幅度较宽**。|可以及时反映市场变化，并做出相应的反应，但容易造成管理失控。|

　　在纵向分工中，基本有两种形式：一是**高长型组织结构**；二是**扁平型组织结构**。  
　　2.纵向分工结构组织内部的管理问题

|   |   |
|---|---|
|问题|结论（了解）|
|（1）集权与分权|**集权**是指企业的高层管理人员拥有最重要的决策权力。  <br>优点：  <br>（1）**易于协调**各职能间的决策；  <br>（2）易于对上下沟通的形式进行规范；  <br>（3）能与企业的目标达成一致；  <br>（4）**危急情况**下能够做出**快速决策**；  <br>（5）有助于实现规模经济；  <br>（6）比较适用于由外部机构实施密切监控的企业，因为所有的决策都能得以协调。  <br>缺点：  <br>（1）高级管理层可能不会重视**个别部门的要求**；  <br>（2）**决策时间过长**；  <br>（3）级别较低的管理者**职业发展有限**。|
|分权型结构一般包含更少的管理层次，并将决策权分配到较低的层级，从而具有较宽的管理幅度并呈现出扁平型结构。  <br>分权型结构减少了信息沟通的**障碍**，提高了企业**反应能力**，能够为决策提供更多的信息并对员工产生激励效应。|
|（2）中层管理人员人数|企业为了**降低成本**，使其结构效率化，应尽量**减少管理层次**。|
|（3）信息传递|内部管理**层次越多**，信息在传递的过程中就越容易发生不同程度扭曲，不可能完整地到达信息传递的目的地，**管理费用也会相应增加**。|
|（4）协调与激励|应尽量**减少管理层次**。|

　　1.横向分工结构的基本类型  
　　企业组织结构有8种基本类型：创业型组织结构、职能制组织结构、事业部制组织结构、M型组织结构（多部门结构）、战略业务单位组织结构（SBU）、矩阵制组织结构和H型结构（控股企业/控股集团组织结构）和国际化经营企业的组织结构。  
　　 ![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps237.jpg)  
　　（1）创业型组织结构。  
　　(Entrepreneurial Startup Structure/Simple Structure)

|   |   |
|---|---|
|基本含义|企业的所有者或管理者对若干下属实施**直接控制**，并由其下属执行一系列工作任务。企业的战略计划（若有）由中心人员完成，该中心人员还负责所有**重要的经营决策**。  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps238.jpg)|
|特点|**弹性较小**并**缺乏专业分工**，其成功主要依赖于该**中心人员**的个人能力。|
|适用情况|**通常应用于小型企业。**|

|   |   |
|---|---|
|基本含义|按**职能**进行**专业化分工**。  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps239.jpg)  <br>【例子】1990年之前的卡特彼勒（Caterpillar）职能制结构图  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps240.jpg)|
|适用情况|**单一业务企业**。|
|优点|（1）能够通过在单一部门内集中所有某一类型的活动来实现**规模经济**；  <br>（2）有利于培养**职能专家**；  <br>（3）由于职能管理是常规和重复性任务，因而**工作效率**得到提高；  <br>（4）董事会易于监控各个部门。|
|缺点|（1）由于对战略的重要流程进行了过度细分，因而在**协调**不同职能时可能出现问题；  <br>（2）难以确定各项产品产生的盈亏；  <br>（3）职能之间容易发生**冲突**、各自为政；  <br>（4）集权化的决策制定机制会放慢反应速度。|

**第04讲 M型组织结构、战略业务单位组织结构**  
　　（4）M型组织结构（多部门结构）。

|   |   |
|---|---|
|M型组织结构（多部门结构）|通过产品线的增加，企业会不断扩张。随着企业规模的扩大，上述事业部制组织结构就会演变为更复杂的M 型组织结构。在M 型组织结构中，原来的事业部一般由拥有更大经营权的**公司**所代替；每个公司比**以前的事业部**负责更多的产品线；有的公司下设**若干事业部**分别管理不同的产品生产线。  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps247.jpg)  <br>事业部制和M型定义内容区别：  <br>事业部制：企业内部组织管理方面的概念，无需注册登记、无需单独纳税  <br>M型：（1）分公司，法律上的概念（不具有独立的法人资格），需注册登记、总公司统一缴纳企业所得税  <br>（2）子公司，法律上的概念（独立法人），需注册登记、单独缴纳企业所得税  <br>优点：  <br>（1）有利于企业持续成长；  <br>（2）由于每一个公司都有其自身的高层管理者，总经理及总部员工的工作量会有所减轻，这样，总经理可以有更多的时间分析各个公司的经营情况并进行资源配置；  <br>（3）由于职权被分派到总部下面的每个公司，并在每个公司内部进行再次分派，因而有利于调动各层管理者的积极性；  <br>（4）能够使用诸如资本回报率等指标对各个公司的绩效进行财务评估和比较。  <br>缺点：  <br>（1）为公司之间分摊企业的管理成本比较困难；  <br>（2）各个公司之间经常会由于争取更多的企业资源而产生有损职能的竞争和摩擦；  <br>（3）当一个公司生产另一公司所需的部件或产品时，确定转移价格也会产生冲突。  <br>适用情况：具有**多个产品线**的企业。|

  
　　（5）战略业务单位组织结构（SBU）。

|   |   |
|---|---|
|基本含义|企业的成长最终需要将相关产品线归类为事业部，然后再将这些事业部归类为战略业务单位。（按照战略业务单位建立组织结构）  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps248.jpg)  <br>【举例】某集团战略业务单位组织结构图  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps249.jpg)|
|适用情况|规模较大的**多元化**经营企业。|
|优点|（1）降低了企业总部的**控制跨度**（管理范围、管理幅度）；  <br>（2）控制幅度的降低也减轻了总部的信息过度情况；  <br>（3）有利于具有类似战略的产品、市场或技术之间实现更好的协调和配合；  <br>（4）由于无须在各个战略业务单位之间分摊管理成本，因而易于监控、评估每个战略业务单位的绩效。|
|缺点|（1）由于增加了一个垂直管理层，因此总部与事业部和产品层的关系变得更疏远；  <br>（2）各个战略业务单位的总裁之间为了取得更多的企业资源会产生竞争和摩擦，从而对企业的总体绩效产生不利影响。|

|   |   |
|---|---|
|基本含义|矩阵结构是一种具有**两个**或多个命令通道的结构，包含**两条**预算权力线以及**两个**绩效和奖励来源。  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps250.jpg)  <br>【例子】1990年之后的卡特彼勒  <br>![](file:////Users/guochenfa/Library/Containers/com.kingsoft.wpsoffice.mac/Data/tmp/wps-guochenfa/ksohtml//wps251.jpg)|

|   |   |
|---|---|
|适用情况|非常复杂**项目**中的控制问题。|
|优点|（1）项目经理与项目的关系更紧密，能够更**直接地参与到**与其产品相关的战略中来，从而有利于激发项目经理成功的动力；更多资料V：283348610  <br>（2）能够有效地**优先考虑**关键项目，加强对产品和市场的关注；  <br>（3）**项目组**与**产品经理**、**职能经理**之间的联系更加直接，因而能够做出**更优**的决策；  <br>（4）实现了各个**部门**之间的协作以及各项技能和专门技术的**相互交融**；  <br>（5）**双重权力**使得**职能经理**不再**只关注**自身的业务范围。|
|缺点|（1）可能导致**权力划分不清晰**（例如谁来负责预算），甚至造成职能工作和项目工作之间的冲突；  <br>（2）双重权力容易使**管理者**之间**产生冲突**；  <br>（3）**管理层**可能难以接受双重权力结构，并且管理者可能会觉得另一名管理者会**争夺其权力**，从而产生**危机感**；  <br>（4）**协调**所有的**产品和职能**会**增加**时间成本和财务**成本**，并导致决策的**时间**过长。|

  
**第06讲 H型结构和国际化经营企业组织结构**  
　　（7）H型结构（控股企业/控股集团组织结构）——企业集团。

|   |   |
|---|---|
|基本含义|成立控股企业，其下属子企业具有独立的法人资格。  <br>控股企业的类型：  <br>（1）**纯粹控股公司**不直接从事某种实际的生产经营活动，其目的只是掌握子公司的股份，控制其股权；  <br>（2）**混合控股公司**除了利用控股权支配子公司的生产经营活动，还从事自身的生产经营。|
|适用情况|**业务领域涉及多个方面，甚至上升到全球化竞争层面。**|
|主要特点|（1）其业务单元的自主性强；  <br>（2）企业**无须负担高额的母公司管理费**，因为母公司的职员数量很可能非常少；业务单元能够自负盈亏并从母公司**取得较便宜的投资成本**；  <br>（3）在某些国家如果将这些企业看成一个整体，业务单元还能够获得**一定的节税收益**；  <br>（4）控股企业可以将**风险分散**到多个企业中，但是有时也很容易撤销对个别企业的投资。|

|任务导向型|管理者关心的是不断成功地解决问题。采用的组织结构往往是矩阵式。**实现目标**是任务导向型企业的**主导思想**。企业强调的是速度和灵活性，专长是个人权力和职权的主要来源，并且决定一个人在给定情景中的相对权力。  <br>具有很强的适应性，**个人能高度控制自己分内的工作**，**在十分动荡或经常变化的环境中会很成功。也会给企业带来很高的成本。**|常见于新兴产业中的企业，特别是一些高科技企业。|
|人员导向型|企业存在的目的主要是**为其成员的需要服务**，员工通过示范和助人精神来互相影响，而不是采用正式的职权。这类文化中的人员不易管理，企业能给他们施加的影响很小。|常见于俱乐部、协会、专业团体和小型咨询公司。|

|使用比率来进行绩效评价的主要原因  <br>（细小知识点，通常是客观题）|（1）通过比较**不同时期的比率**可以很容易地发现它们的变动。  <br>（2）相对于**实物数量或货币的绝对数值**，比率对企业业绩的衡量更为适合。  <br>（3）比率适合用作业绩目标。  <br>（4）比率提供了总结企业业绩和经营成果的工具、方法，并可在**同类企业**之间进行比较。|
|比率评价的局限性  <br>（细小知识点，通常是客观题）|（1）信息**获取存在困难**。  <br>（2）信息的使用存在局限性。  <br>（3）比率在各个行业的理想标准不同，而且理想标准会随着时间推移发生改变，这给不同行业或同一行业中不同企业的业绩比较带来困难。  <br>（4）比率有时不能准确反映真实情况。  <br>（5）比率有时体现的是被扭曲的结果。  <br>（6）可能鼓励短期行为。  <br>（7）忽略其他战略要素。  <br>（8）激励、控制的人员范围有限。|

　　提出者：卡普兰和诺顿。分为四个角度审视自身业绩：财务角度、顾客角度、内部流程角度、创新与学习角度。四个角度指标所包含的内容体现了五个方面的平衡：

　　● 企业内部利益相关者（员工：创新与学习角度）与外部利益相关者（股东：财务角度、客户：顾客角度）的平衡；  
　　● 领先指标即预期性指标（创新与学习角度、顾客角度、内部流程角度）与滞后指标即结果性指标之间的平衡（财务角度）。

**第16讲 例题讲解**  
　　【例题·简答题】（2021年）2015年成立的盟塑公司是国内一家工业电子商务服务平台，相对于国内综合性B2B（企业与企业之间交易）电商平台，盟塑公司专注于塑化行业，围绕塑化原材料贸易，运用数字化技术，为采购商和供应商提供专业的塑化材料采购和配套的供应链服务。具体运营模式如下：  
　　（1）建立塑化原材料上下游对接通道。针对塑化原料贸易信息不透明、交易流程长、效率低下等关键行业痛点，盟塑平台提供大量供应商的信息数据，采购商通过专业搜索可拿到大品牌厂家的直供原料，满足其专业化采购的需求；上游企业也因此获得大量消费者、提高了毛利、打造了品牌。  
　　（2）整合关联服务方，提供一站式互联网服务。针对塑化原材料贸易双方对于仓储公司、物流公司以及金融机构等关联服务方的需求问题，盟塑平台将这些关联服务方整合进来，贸易双方不需要再费时费力地去跟各个环节打交道，大大降低了贸易成本。而对于关联服务方而言，也因此增加了客源，并能通过盟塑平台积累的大量用户的数据，增加对用户信用的了解，降低了经营风险。  
　　（3）引入C端（消费者）参与贸易服务。盟塑平台引入包括行业的采购师、分销人员、司机车主等个人提供对应的贸易服务，有资源的其他消费者也能通过盟塑平台参与贸易服务，从而加速塑化产业的货物流通和货款流通，解决成本高企、效率低下的难题。  
　　2019年，盟塑公司荣获“中国B2B电子商务平台50强”称号。  
　　要求：  
　　（1）简要分析盟塑公司建立工业电子商务服务平台所依据的产业市场细分的变量。  
　　（2）简要分析盟塑公司是如何运用价值链分析构筑自身竞争优势的。  
　　（3）简要分析盟塑电子商务平台的运营模式所体现的数字化技术对企业经营模式的影响。  
　　『正确答案』  
　　（1）盟塑公司建立工业电子商务服务平台所依据的产业市场细分的变量是“用户的行业类别”。“相对于国内综合性B2B（企业与企业之间交易）电商平台，盟塑公司专注于塑化行业，围绕塑化原材料贸易，运用数字化技术，为采购商和供应商提供专业的塑化材料采购和配套的供应链服务”。  
　　（2）盟塑公司运用价值链分析构筑其竞争优势体现如下：  
　　①确认那些支持企业竞争优势的关键性活动。“针对塑化原料贸易信息不透明、交易流程长、效率低下等关键行业痛点，……”。
