# 第三章 战略选择

## 学习要点

本章节是战略科目的核心章节之一,主要包括:
- 市场营销战略
- 总体战略(公司层战略)
- 业务单位战略
- 职能战略
- 国际化经营战略

---

## 第一节 市场营销战略

### 一、市场细分

#### 1. 消费者市场细分的依据

| 细分变量 | 具体指标 |
|---------|---------|
| **地理细分** | 国家、地区、城市农村、地形气候、交通运输条件等<br>举例:中国市场分为南方/北方;一线城市/二线城市等 |
| **人口细分** | 年龄、性别、收入水平、职业、受教育程度、家庭规模、家庭生命周期阶段、社会阶层、宗教信仰、民族及国籍等 |
| **心理细分** | 个性、爱好、价值观念、生活方式、购买动机、追求的利益等<br>购买动机:求美、求廉、求实、求新、求名、求便、炫耀、情感、理智 |
| **行为细分** | 购买或使用时机、使用率(轻度/中度/重度)、进入市场程度(未曾/曾经/潜在/初次/常规使用者)、品牌忠诚度等 |

#### 2. 产业市场细分的依据

| 细分变量 | 具体指标 |
|---------|---------|
| **用户的行业类别** | — |
| **用户规模** | 公司规模:大型、中型、小型 |
| **用户的地理位置** | — |
| **购买行为因素** | 用户追求的利益、使用频率、品牌忠诚度、使用者地位、购买方式等 |

**波诺马和夏皮罗细分变量:**

| 变量类别 | 具体内容 |
|---------|---------|
| **经营变量** | 技术、使用者/非使用者情况、顾客能力 |
| **采购方法变量** | 采购职能组织、权力结构、现有关系、总采购政策、采购标准 |
| **情境因素变量** | 紧急、特别用途、订货量 |
| **企业特征变量** | 购销双方相似点、对待风险的态度、忠诚度 |
| **其他变量** | 行业、公司规模、地理位置 |

### 二、目标市场选择

| 目标市场选择策略 | 含义 | 优缺点 |
|----------------|------|--------|
| **无差异性营销** | 企业把整个市场作为目标市场,运用一种产品、一种价格、一种推销方法 | 【优点】有利于标准化及大规模生产,发挥规模经济优势,降低生产、存货、运输、研发成本<br>【缺点】单一产品难以满足所有消费者需求,应变能力较差 |
| **差异性营销** | 选择两个或两个以上的细分市场作为目标市场,分别设计不同产品和营销组合策略 | 【优点】满足不同消费者需要,扩大销售量,增强竞争力,分散风险<br>【缺点】增加了企业经营管理难度,生产成本、研发成本、存货成本增加 |
| **集中性营销** | 集中全部资源,以一个或少数几个性质相似的细分市场为目标 | 【优点】容易取得深入的市场了解,提高产品和企业知名度,节省营销费用<br>【缺点】对单一目标市场依赖性大,面临极大风险 |

**选择需考虑的因素:**
1. 企业资源和能力
2. 产品同质性
3. 产品所处生命周期阶段
4. 市场同质性
5. 竞争对手的战略

### 三、市场定位

| 定位策略 | 具体内容 |
|---------|---------|
| **避强定位** | 主动回避与强力竞争对手直接对抗,抢占或填补市场空位<br>举例:七喜汽水定位"非可乐型饮料" |
| **迎头定位** | 将产品定位于与现有竞争者产品重合的市场位置<br>举例:雪碧与七喜竞争 |
| **并存定位** | 将产品定位在竞争对手产品附近,共同满足同一目标市场<br>条件:目标市场还有未被满足的需求;产品能与竞争产品相媲美 |
| **取代定位** | 将竞争对手赶出原有位置,并取而代之 |
| **重新定位** | 变更产品特色,使目标顾客群重新认识原有产品的个性和形象 |
| **领先定位** | 开辟新细分市场或对已有产品再创造成为市场领先者<br>举例:王老吉重新定位成预防上火的功能性饮料 |

### 四、市场营销组合(4P)

#### 1. 产品策略

**产品组合的四个维度:**

| 维度 | 含义 |
|-----|------|
| **宽度** | 产品组合中所包含的产品线或产品大类的数量 |
| **长度** | 所有产品大类中所包含的产品项目的总数 |
| **深度** | 所有产品项目中所包含的花色、品种、规格的数量 |
| **关联度** | 各个产品大类在最终使用、目标市场、生产条件、分销渠道等方面相互关联的密切程度 |

**产品组合策略类型:**
1. 扩大产品组合(拓展宽度、增加长度、加强深度、加强关联度)
2. 缩减产品组合
3. 产品延伸
   - 向下延伸:原产高档产品→增加低档产品(如iPhone SE)
   - 向上延伸:原产低档产品→增加高档产品(如比亚迪仰望)
   - 双向延伸:原定中档→同时增加高档和低档产品

**品牌策略:**

| 类型 | 具体内容 |
|-----|---------|
| **品牌归属策略** | ①企业品牌/生产者品牌/自有品牌<br>②中间商品牌(如沃尔玛惠宜、京东京造)<br>③贴牌生产 |
| **品牌统分策略** | ①统一品牌:所有产品使用一个品牌(如飞利浦)<br>②个别品牌:不同产品使用不同品牌(如宝洁)<br>③分类品牌:不同类别产品使用不同品牌<br>④复合品牌:同时采用两个或两个以上品牌名称 |

**新产品开发策略:**

| 新产品分类 | 说明 |
|-----------|------|
| 全新产品 | 运用新原理、新技术、新材料研制的市场上从未有过的产品 |
| 替代产品 | 采用新技术、新材料、新元件,使原有产品性能有重大改进的产品 |
| 改进产品 | 从造型、结构、质量、性能等对原有产品作出适当改进的产品 |
| 模仿性新产品 | 通过模仿市场上已有的产品制造出来的产品 |

**新产品开发成功条件:**
1. 具有独特性的优质产品
2. 与顾客保持密切沟通,深入了解真正的需求
3. 采用开放式新产品开发模式
4. 合理配置资源

#### 2. 价格策略

**基本的定价方法:**
1. 成本导向定价法(成本加成、目标利润、固定报酬、收支平衡)
2. 需求导向定价法(感知价值定价、反向定价)
3. 竞争导向定价法(按行业平均价、低于/高于市场价、拍卖投标)

**主要定价策略:**

| 策略类型 | 具体内容 |
|---------|---------|
| **心理定价策略** | ①声望定价(利用品牌声望制定高价)<br>②尾数定价(以零头数或吉利数结尾)<br>③整数定价(结算方便)<br>④招徕定价(明显低于市价吸引顾客) |
| **产品组合定价策略** | ①系列产品定价 ②备选产品定价 ③副产品定价<br>④关联产品定价(主产品低价,关联品高价)<br>⑤分部定价 ⑥产品捆绑定价(套餐定价) |
| **折扣与折让策略** | ①现金折扣 ②数量折扣 ③功能折扣(交易折扣)<br>④季节折扣 ⑤折让(以旧换新、促销折让) |
| **差别定价策略** | ①顾客差别定价 ②地点差别定价<br>③时间差别定价 ④样式差别定价 |

**新产品定价策略:**

| 策略 | 价格 | 条件 |
|-----|------|------|
| **渗透定价** | 低价(市场占有率) | 需求弹性大;规模效应显著;低价不会导致过度竞争 |
| **撇脂定价** | 高价(利润) | 市场有足够需求;高价不抵消利润;高价产生高质印象;不影响市场优势 |
| **温和定价** | 中等价格 | 价格符合消费者预期;企业能正常经营;有调整空间;产品独特性不受影响 |
| **免费定价** | 免费 | 主要适用于数字化产品。包括限制使用免费、产品部分免费、捆绑式免费、完全免费 |

#### 3. 分销策略

**分销渠道结构:**

| 类型 | 含义 | 优缺点 |
|-----|------|--------|
| **直接渠道** | 产品从生产者向消费者转移不经过中间环节 | 【优点】保持良好客户关系、快速响应市场、减少流通费用<br>【缺点】增加分销负担和成本,不适合分散购买 |
| **间接渠道** | 经过中间环节(一级、二级、三级渠道) | 【优点】专业化协作、广泛分销<br>【缺点】沟通不密切、反应滞后、控制力减弱 |

**渠道宽度策略:**

| 分销策略 | 优势 | 劣势 | 适用范围 |
|---------|------|------|---------|
| **独家分销** | 对中间商控制强;中间商获优惠价格 | 合作紧密要求高;满意度可能受影响;反控制力强 | 技术含量高、需售后服务的专门产品 |
| **选择性分销** | 获得中间商支持;比独家分销购物方便 | 中间商竞争较激烈 | 选购品和特殊品 |
| **密集分销** | 市场覆盖率高;便利消费者 | 价格竞争激烈;市场混乱;管理成本高 | 日用消费品、家具、家用电器 |

**渠道广度策略:**
- 单渠道
- 多渠道
- 跨渠道(线上线下交叉,数据无缝对接)
- 全渠道(整合销售,包括实体店铺、网上商城、信息媒体)

**基于成员关系的渠道系统:**

| 类型 | 含义 |
|-----|------|
| **松散型渠道系统** | 渠道成员各自为政、各行其是,缺乏密切协作 |
| **垂直渠道系统** | 生产者、批发商和零售商组成的联合体 |
| **水平渠道系统** | 两个或两个以上企业横向联合,互相利用对方渠道 |

#### 4. 促销策略

| 促销组合要素 | 含义 |
|-------------|------|
| **广告促销** | 以付费方式,通过电视、报纸、广播、户外广告、手机短信、微信、微博、短视频等媒体进行宣传 |
| **公关营销** | 通过有效的公共关系策略,建立良好的公众形象和关系来促进销售 |
| **营业推广** | 采用非媒体促销手段,如赠券、价格折扣、赠送样品、奖品、竞赛抽奖、会员俱乐部等 |
| **人员推销** | 销售人员直接与潜在购买者面对面交流,说服其购买 |

**促销组合策略:**

| 策略 | 含义 |
|-----|------|
| **推式策略** | 通过销售人员和中间商将产品"推"向消费者 |
| **拉式策略** | 开展直接针对消费者的促销活动,吸引、激励他们主动购买 |
| **推拉结合策略** | 综合采用推式策略和拉式策略 |

## 第二节 总体战略(公司层战略)

### 一、总体战略的主要类型

企业总体战略分为三大类:
1. **发展战略**
2. **稳定战略**
3. **收缩战略**

### 二、发展战略

#### 1. 一体化战略

**产业链:** 供应商(上游) → 生产商 → 中间商(下游)

| 战略类型 | 含义 | 优点 | 缺点/风险 |
|---------|------|------|----------|
| **纵向一体化** | 沿产业链的纵向或横向延展业务 | 节约交易成本,控制稀缺资源,保证关键投入质量或获得新客户 | 增加内部管理成本;不熟悉新业务领域风险;退出成本高 |
| **前向一体化** | 向下游环节扩张(进入销售环节) | 控制销售过程和渠道,掌握市场,增强对消费者需求变化的敏感性 |  |
| **后向一体化** | 向上游环节扩张(进入供应环节) | 有效控制关键原材料的成本、质量及供应可靠性,确保生产经营活动稳步进行 |  |
| **横向一体化** | 向同一环节扩张(并购竞争对手) | 实现规模经济以获取竞争优势 |  |

**一体化战略适用条件:**

| 战略类型 | 适用条件 |
|---------|---------|
| **前向一体化** | ①现有销售商销售成本高或可靠性较差<br>②产业增长潜力较大<br>③具备所需资金和人力资源<br>④销售环节利润率较高 |
| **后向一体化** | ①现有供应商供应成本高或可靠性较差<br>②供应商数量少而需求方竞争者众多<br>③产业增长潜力较大<br>④具备所需资金和人力资源<br>⑤供应环节利润率较高<br>⑥产品价格稳定对企业十分关键 |
| **横向一体化** | ①产业竞争较为激烈<br>②规模经济较为显著<br>③符合反垄断法律法规<br>④产业增长潜力较大<br>⑤具备所需资金和人力资源 |

#### 2. 密集型战略(安索夫矩阵)

**安索夫矩阵:**

||产品||
|---|---|---|
|**市场**|现有产品|新产品|
|现有市场|市场渗透|产品开发|
|新市场|市场开发|多元化|

**密集型战略包括:** 市场渗透、市场开发、产品开发

**市场渗透战略**(现有产品+现有市场)
- 含义:强调发展单一产品,通过更强的营销手段获得更大的市场占有率
- 适用条件:
  ①整个市场正在增长
  ②企业决定将利益局限在现有产品或市场领域
  ③其他企业因各种原因离开了市场
  ④企业拥有强大的市场地位和独特竞争优势
  ⑤市场渗透战略风险较低、管理者参与度高、所需投资较少

**市场开发战略**(现有产品+新市场)
- 新市场包括:地理区域市场、消费者细分市场
- 原因:
  ①企业发现现有产品生产过程的性质导致难以转而生产全新的产品
  ②市场开发往往与产品改进结合在一起
  ③现有市场或细分市场已经饱和
- 适用条件:
  ①存在未开发或未饱和的市场
  ②可得到新的、可靠的、经济的和高质量的销售渠道
  ③企业在现有经营领域十分成功
  ④企业拥有扩大经营所需的资金和人力资源
  ⑤企业存在过剩的生产能力
  ⑥企业的主业属于正在迅速全球化的产业

**产品开发战略**(新产品+现有市场)
- 原因:
  ①充分利用企业对市场的了解
  ②保持相对于竞争对手的领先地位
  ③从现有产品组合的不足中寻求新的机会
  ④使企业能继续在现有市场中保持稳固的地位
- 适用条件:
  ①企业产品具有较高的市场信誉度和顾客满意度
  ②企业所在产业属于适宜创新的高速发展的高新技术产业
  ③企业所在产业正处于高速增长阶段
  ④企业具有较强的研究和开发能力
  ⑤主要竞争对手以近似价格提供更高质量的产品

#### 3. 多元化战略

多元化战略是指企业进入与现有产品和市场不同的领域。

**采用多元化战略的原因:**
1. 在现有产品或市场中持续经营不能达到目标
2. 企业资金超过了在现有产品或市场中的财务扩张所需要的资金
3. 与在现有产品或市场中的扩张相比,多元化战略意味着更高的利润

**多元化战略的类型:**

| 战略类型 | 战略描述 | 适用条件 |
|---------|---------|---------|
| **相关多元化战略**(同心多元化) | 以现有业务或市场为基础进入相关产业或市场,获得融合优势(范围经济) | 企业在产业或市场内具有较强的竞争优势,而该产业或市场的成长性或吸引力逐渐下降 |
| **非相关多元化战略**(离心多元化) | 进入与当前产业和市场均不相关的领域,获得新的利润增长点,规避发展风险 | 企业当前产业或市场缺乏吸引力,而企业也不具备较强的能力和技能转向相关产品或市场 |

**多元化战略优点:**
1. 分散风险
2. 能更容易地从资本市场中获得融资
3. 当企业在原产业无法增长时找到新的增长点
4. 利用未被充分利用的资源
5. 运用盈余资金
6. 获得资金或其他财务利益
7. 运用企业在某个产业或市场中的形象和声誉来进入另一个产业或市场

**多元化战略风险:**
1. 来自原有经营产业的风险
2. 市场整体风险
3. 产业进入风险
4. 产业退出风险
5. 内部经营整合风险

### 三、稳定战略

| 项目 | 内容 |
|-----|------|
| **含义** | 又称维持战略,指企业在战略期所期望达到的经营状况基本保持在战略起点的范围和水平上的战略 |
| **适用情况** | 适用于对战略期环境的预测变化不大,而在前期经营相当成功的企业 |
| **优点** | ①充分利用原有生产经营领域中的各种资源<br>②避免开发新产品和新市场所需的巨大资金投入和开发风险<br>③避免资源重新配置和组合的成本<br>④防止由于发展过快、过急造成的失衡状态 |
| **风险** | 一旦外部环境发生较大变动,战略目标、外部环境、企业实力三者之间就会失去平衡;容易减弱风险意识,形成惧怕风险、回避风险的企业文化,降低对风险的敏感性和适应性 |

### 四、收缩战略

| 项目 | 内容 |
|-----|------|
| **含义** | 也称撤退战略,指企业从目前的经营领域和基础上收缩,在一定时期内缩小原有经营范围和规模的战略 |
| **原因** | **主动原因:** 满足企业战略重组的需要<br>**被动原因:** 外部环境原因(宏观经济、产业周期、技术、政策、社会价值观等变化);内部环境原因(内部经营机制不顺、决策失误、管理不善) |
| **方式** | **紧缩与集中战略:** 机制变革、财政和财务战略、削减成本战略<br>**转向战略:** 重新定位或调整现有产品和服务、调整营销策略<br>**放弃战略:** 将企业的一个或若干个部门出售、转让或停止经营 |
| **退出障碍** | ①固定资产的专用性程度<br>②退出成本(劳工协议、重新安置等)<br>③内部战略联系<br>④感情障碍<br>⑤政府与社会约束 |

## 第三节 发展战略的主要途径

### 一、并购战略

#### 1. 并购的类型

| 分类标准 | 类型 | 含义 |
|---------|------|------|
| **按并购双方所处产业分类** | 横向并购 | 并购方与被并购方处于同一产业 |
| | 纵向并购 | 在经营对象上有密切联系,但处于不同产销阶段的企业之间的并购(前向/后向) |
| | 多元化并购 | 处于不同产业、在经营上也无密切联系的企业之间的并购 |
| **按被并购方态度分类** | 友善并购 | 通过友好协商确定并购条件 |
| | 敌意并购(恶意并购) | 遭到拒绝后采取强制手段强行收购 |
| **按并购方身份分类** | 产业资本并购 | 由非金融企业进行,目的是获得产业利润 |
| | 金融资本并购 | 由投资银行或非银行金融机构进行,目的是获得投资利润 |
| **按收购资金来源分类** | 杠杆收购 | 70%及以上的资金来源是对外负债 |
| | 非杠杆收购 | 主体资金来源是自有资金 |

#### 2. 并购的动机
1. 避开进入壁垒,迅速进入,争取市场机会
2. 获得协同效应(经营协同、财务协同、技术或资产协同、管理协同、市场或产品协同)
3. 克服企业负外部性,减少竞争,增强对市场的控制力

#### 3. 并购失败的原因
1. 决策不当(未认真分析目标企业成本和效益、高估并购对象产业吸引力和管理能力)
2. 并购后不能很好地进行企业整合
3. 支付过高的并购费用
4. 跨国并购面临政治风险

### 二、内部发展(新建)战略

#### 1. 企业采取内部发展战略的动因
1. 开发新产品的过程使企业能深刻地了解市场及产品
2. 不存在合适的收购对象
3. 保持统一的管理风格和企业文化
4. 为管理者提供职业发展机会
5. 代价较低,无须为商誉支付额外的金额
6. 并购通常会产生隐藏的或无法预测的损失,而内部发展不太可能
7. 这可能是唯一合理的、实现真正技术创新的方法
8. 可以有计划地进行,容易获得财务支持,成本可以按时间分摊
9. 风险较低
10. 内部发展的成本增速较慢

#### 2. 内部发展的缺点
1. 在市场上增加了竞争者,可能激化某一市场内的竞争
2. 企业不能接触到其他企业的知识及系统
3. 从一开始就缺乏规模经济或经验曲线效应
4. 当市场发展非常快时,内部发展显得过于缓慢
5. 进入新市场可能要面对非常高的障碍

#### 3. 内部发展战略的应用条件
1. 产业处于不均衡状况,结构性障碍还没有完全建立起来
2. 产业内现有企业的行为性障碍容易被制约
3. 企业有能力克服结构性与行为性障碍,或者克服障碍的代价小于企业进入后的收益

### 三、企业战略联盟

#### 1. 企业战略联盟的基本特征
1. 从经济组织形式来看,是介于企业与市场之间的一种"中间组织"
2. 从企业关系来看,是在资源共享、优势相长、相互信任、相互独立的基础上通过事先达成协议而结成的一种平等的合作伙伴关系
3. 从企业行为来看,是一种战略性的合作行为

#### 2. 企业战略联盟形成的动因

| 动因 | 基本原理 | 典型案例线索 |
|-----|---------|------------|
| **促进技术创新** | 分担投入 | 通过合作获得先进技术(专利) |
| **避免经营风险** | 信息沟通 | 合作双方各有短板,通过合作弥补 |
| **避免或减少竞争** | 竞合,避免过度竞争 | 合作双方是现实(或潜在)竞争对手 |
| **实现资源互补** | — | 合作双方各有优势,实现互补 |
| **开拓新的市场** | 产品增加,市场扩张 | 获得另一方新产品或帮助进入新市场 |
| **降低协调成本** | 不用整合(相对于并购) | 原文"协调成本" |

#### 3. 企业战略联盟的主要类型

| 从股权参与和契约联结的角度 | 具体内容 |
|---------------------------|---------|
| **股权式联盟** | 合资企业(体现战略意图,强调股权对等)<br>相互持股投资(少量持股) |
| **契约式联盟** | 技术交流协议<br>合作研究开发协议<br>生产营销协议<br>产业协调协议 |

#### 4. 战略联盟的管控
1. 订立协议(严格界定联盟目标、周密设计联盟结构、准确评估投入资产、规定违约责任和解散条款)
2. 建立合作信任的联盟关系

#### 5. 企业战略联盟的新发展——网络合作联盟

**网络合作联盟的类型:**

| 类型 | 出现背景 | 优缺点 |
|-----|---------|--------|
| **稳定网络合作联盟** | 在成熟的行业中,创新发生不频繁,竞争优势可以存续较长的时间,顾客需求相对固定且可预测 | 通过网络合作延伸竞争优势,实现规模经济或范围经济,持续创造价值,提升经济效益 |
| **动态网络合作联盟** | 在产品创新频繁、产品生命周期较短的行业中 | 实现更敏捷、更灵活的聚散,通过资源共享不断探索新想法,实现产品与服务创新,通过进入或开发新市场保证经济利益 |

## 第四节 业务单位战略

### 一、基本竞争战略(波特三种基本竞争战略理论)

#### 1. 成本领先战略

| 项目 | 内容 |
|-----|------|
| **含义** | 企业通过在内部加强成本控制,在研究开发、生产、销售、服务和广告等领域把成本降到最低限度,成为产业中的成本领先者 |
| **优势** | ①形成进入障碍<br>②增强讨价还价能力<br>③降低替代品的威胁<br>④保持领先的竞争地位 |
| **实施条件-市场情况(外部条件)** | ①产品具有较高的价格弹性,市场中存在大量价格敏感用户<br>②产业中所有企业的产品都是标准化产品<br>③购买者不太关注品牌<br>④价格竞争是市场竞争的主要手段<br>⑤消费者的转换成本较低 |
| **实施条件-资源和能力(内部条件)** | ①在规模经济显著的产业中装备相应的生产设施<br>②降低各种要素成本<br>③提高生产率(采用最新技术、工艺或流程,充分利用学习曲线)<br>④改进产品工艺设计(简单产品设计)<br>⑤提高生产能力利用程度<br>⑥选择适宜的交易组织形式(外购/外包VS自行生产)<br>⑦资源集中配置 |
| **风险** | ①技术的变化可能使过去用于降低成本的投资一笔勾销<br>②产业的新加入者或追随者通过模仿或使用更高技术水平的设施,达到同样的甚至更低的产品成本<br>③市场需求从注重价格转向注重产品的品牌形象,使企业原有的优势变为劣势 |

#### 2. 差异化战略

| 项目 | 内容 |
|-----|------|
| **含义** | 企业向顾客提供的产品和服务在产业范围内独具特色,这种特色可以给产品带来额外的加价 |
| **优势** | ①形成进入障碍<br>②降低顾客对价格的敏感程度<br>③增强讨价还价能力<br>④抵御替代品威胁 |
| **实施条件-市场情况(外部条件)** | ①产品能够充分地实现差异化,且为顾客所认可<br>②顾客的需求是多样化的<br>③企业所在产业技术变革较快,创新成为竞争的焦点 |
| **实施条件-资源和能力(内部条件)** | ①具有强大的研发能力和产品设计能力<br>②具有很强的市场营销能力<br>③有能够确保激励员工创造性的激励体制、管理体制和良好的创造性文化<br>④具有从总体上提高某项经营业务的质量、树立产品形象、保持先进技术和建立完善分销渠道的能力 |
| **风险** | ①企业形成产品差别化的成本过高<br>②市场需求发生变化<br>③竞争对手的模仿和进攻使已建立的差异缩小甚至转向 |

#### 3. 集中化战略

| 项目 | 内容 |
|-----|------|
| **含义** | 针对某一特定购买群体、产品细分市场或区域市场,采用成本领先或产品差异化来获取竞争优势的战略。分为集中成本领先战略和集中差异化战略 |
| **优势** | ①成本领先和差异化战略抵御产业五种竞争力的优势也都能在集中化战略中体现出来<br>②对于中小企业来说,可以增强相对的竞争优势<br>③对于大企业来说,能够避免与竞争对手正面冲突,处于竞争缓冲地带 |
| **实施条件** | ①购买者群体之间在需求上存在着差异<br>②目标市场在市场容量、成长速度、获利能力、竞争强度等方面具有相对的吸引力<br>③在目标市场上,没有其他竞争对手采用类似的战略<br>④企业资源和能力有限,难以在整个产业实现成本领先或差异化 |
| **风险** | ①狭小的目标市场导致高成本<br>②购买者群体之间需求差异变小<br>③竞争对手的进入与竞争 |

### 二、基本战略的综合分析——"战略钟"(克利夫·鲍曼)

**混合战略(途径3):** 企业可以在为顾客提供更高的认可价值的同时,获得成本优势。

混合战略实现因素:
1. 提供高质量产品的公司会增加市场份额,通过规模经济降低平均成本
2. 生产高质量产品的经验累积和降低成本的速度比生产低质量产品快(经验曲线)
3. 注重提高生产效率可以在高质量产品的生产过程中降低成本

### 三、中小企业竞争战略

#### 1. 零散产业中的竞争战略

**造成产业零散的原因:**
1. 进入障碍低或存在退出障碍
2. 市场需求多样导致高度产品差异化
3. 不存在规模经济或难以达到经济规模
4. 政府政策和地方法规限制
5. 新产业中还没有企业掌握足够的技能和能力

**零散产业的战略选择:**

| 战略选择 | 具体内容 |
|---------|---------|
| **克服零散——获得成本优势** | ①连锁经营或特许经营<br>②技术创新以创造规模经济<br>③尽早发现产业趋势 |
| **增加附加价值——提高产品差异化程度** | — |
| **专门化——目标集聚** | ①产品类型或产品细分的专门化<br>②顾客类型专门化<br>③地理区域专门化 |

**谨防潜在的战略陷阱:**
1. 避免寻求支配地位
2. 保持严格的战略约束力
3. 避免过分集权化
4. 了解竞争者的战略目标与管理费用
5. 避免对新产品做出过度反应

#### 2. 新兴产业中的竞争战略

**新兴产业内部结构的共同特征:**
1. 技术的不确定性
2. 战略的不确定性
3. 成本的迅速变化
4. 萌芽企业和另立门户
5. 首次购买者

**新兴产业发展障碍:**
- 专有技术选择、获取与应用的困难
- 原材料、零部件、资金与其他供给的不足
- 顾客的困惑与等待观望
- 被替代产品的反应
- 缺少承担风险的胆略与能力

**新兴产业发展机遇:**
- 进入障碍低
- 竞争结构未建立

**新兴产业的战略选择:**
1. 塑造产业结构(建立产业游戏规则)
2. 正确对待产业发展的外在性(产业整体利益与企业个体利益的协调)
3. 注意产业机会与障碍的转变,在产业发展变化中占据主动地位
4. 选择适当的进入时机与领域

### 四、蓝海战略

**红海 vs 蓝海:**

||红海战略|蓝海战略|
|---|---|---|
|竞争|在已经存在的市场内竞争,参与竞争|拓展非竞争性市场空间,规避竞争|
|需求|争夺现有需求|创造并攫取新需求|
|价值与成本|遵循价值与成本互替定律|打破价值与成本互替定律,同时追求差异化和低成本|

#### 蓝海战略的六项原则

| 战略制定原则 | 降低的风险因素 |
|-------------|----------------|
| 重建市场边界 | 搜寻的风险 |
| 注重全局而非数字 | 规划的风险 |
| 超越现有需求 | 规模的风险 |
| 遵循合理的战略顺序 | 商业模式风险 |

| 战略执行原则 | 降低的风险因素 |
|-------------|----------------|
| 克服关键组织障碍 | 组织的风险 |
| 将战略执行建成战略的一部分 | 管理的风险 |

#### 重建市场边界的基本法则(六条路径)

| 路径 | 红海思维 | 蓝海观点 |
|-----|---------|---------|
| **1. 审视他择产业** | 接受现有产业界定,成为其中最优 | 与他择产品或服务的产业对手竞争 |
| **2. 跨越战略群组** | 接受现有战略群组划分,在群组中技压群雄 | 突破现有群组划分,寻找市场空白点 |
| **3. 重新界定产业的买方群体** | 只关注单一买方(购买者) | 买方由购买者、使用者和影响者组成,关注点转向使用者 |
| **4. 放眼互补性产品或服务** | 雷同方式为产品或服务的范围定界 | 互补性产品或服务蕴含着未经发掘的需求 |
| **5. 重设客户的功能性或情感性诉求** | 接受现有产业固化的功能或情感导向 | 改变现有功能或情感导向,发现新空间 |
| **6. 跨越时间,参与塑造外部潮流** | 只关注现阶段的竞争威胁 | 洞悉技术与政策潮流如何改变顾客获取的价值 |

### 五、商业模式

#### 1. 商业模式画布

商业模式画布的九个要素:

**客户板块:**
1. 客户细分(大众市场、利基市场、区隔化市场、多元化市场、多边平台)
2. 客户关系(个人助理、自助服务、专用个人助理、自动化服务、社区、共同创作)
3. 渠道通路(自有渠道、合作伙伴渠道或两者混合)

**供给板块:**
4. 价值主张(通过定制新元素组合为群体创造价值)

**基础设施板块:**
5. 核心资源(实体资产、知识资产、人力资源、金融资产)
6. 关键业务(制造产品、平台/网络、问题解决)
7. 重要合作

**财务板块:**
8. 成本结构(成本驱动型、价值驱动型)
9. 收入来源

#### 2. 商业模式创新

商业模式创新的四个构成要素:
1. 价值主张(对客户真实需求的深入描述)
2. 价值创造(企业创造客户价值的方式)
3. 价值获取(生产、供应满足目标客户需要的产品或服务的一系列业务活动)
4. 价值实现(成本结构与收入来源)

#### 3. 商业模式创新类型

**平台商业模式:** 连接两个或两个以上的特定群体,为其提供互动机制和交流平台

**长尾商业模式:** 众多小市场的汇聚可产生与主流市场相匹敌甚至更多的收益

**免费商业模式:**

| 盈利模式 | 说明 | 例子 |
|---------|------|------|
| 增值服务收费模式 | 提供免费基础服务,将少数需要个性化服务的用户转化为付费用户 | 淘宝网免费建店,天猫网收费;奇虎360免费杀毒,企业服务收费 |
| 广告模式(三方市场模式) | 用户免费使用,吸引足够用户量后向企业收取广告费 | 电视广告,消费者免费观看,向做广告的企业收费 |
| 交叉补贴模式 | 免费提供一款产品或服务,对互补品收费或对后续服务收费 | 美国吉利公司剃须刀架免费,刀片收费;通讯运营商送手机 |
| 非货币市场模式 | 用免费产品和服务换取用户的相关劳务 | 鼓励用户参与改进产品或服务的活动,以免费产品回报 |

## 第五节 职能战略

### 一、研究与开发战略

#### 1. 研发的层次
- 基础性研究(科学概念、原理和理论)
- 应用型研究(针对实际问题,将理论知识转化为可应用技术)
- 开发型研究(将应用型研究成果转化为实际产品、服务或解决方案)

#### 2. 研发的类型
- 产品研发(新产品开发)
- 技术研发(探索、发现、采用或改进技术)
- 工艺研发(对产品生产过程的研究和改进)
- 流程研究(对生产、管理过程进行分析评估和改进)

#### 3. 研发的模式

| 模式 | 优点 | 缺点 |
|-----|------|------|
| **自主研发** | 一旦研发成功,在行业内容易做到技术产品领先;产品差异化程度高,避免行业产品同质化;有利于技术秘密和知识产权保护 | 研发成本较高;研发周期较长;研发失败风险较大;研制成功的新产品可能被同业竞争对手仿制 |
| **合作研发** | 发挥各自优势,完成单个企业难以承担的项目;减轻研发压力和负担;减少并分散研发失败风险;缩短研发周期 | 利益不一致或沟通不畅导致效率低下;难以取得技术产品独家领先地位;技术秘密和知识产权保护存在困难 |
| **委托研发** | 企业不需要对研发投入太多精力;依靠具有研发优势的单位或机构,较快提升市场竞争力和行业地位 | 对提高企业自身研发能力帮助不大;获得或保护技术秘密和知识产权存在较大困难 |
| **开放研发** | 实现知识共享,加速知识传播和扩散;鼓励跨组织、跨领域合作与协作;研发过程十分透明,提高研发可靠性;降低研发成本;确保研发成果得到认可和保护 |  |

#### 4. 研发的动力来源
1. 市场需求
2. 技术进步
3. 市场竞争
4. 法规政策
5. 创新文化
6. 社会责任

#### 5. 研发定位
1. 成为向市场推出新技术产品的企业
2. 成为成功产品的创新模仿者
3. 成为成功产品的低成本生产者
4. 成为成功产品低成本生产者的模仿者

### 二、生产运营战略

#### 1. 运营流程的四个因素

| 因素 | 影响分析 |
|-----|---------|
| **批量** | 大规模生产→低成本(可实现专业化分工);小规模生产→高成本 |
| **种类** | 多品种→成本高(要求灵活性);少品种→成本低(标准化生产) |
| **需求变动** | 需求波动→产能利用率低→成本高;需求稳定→产能利用率较高→成本低 |
| **可见性** | 可见性高(服务型)→员工技巧要求高→单位成本高;可见性低(生产型)→员工技巧要求低→单位成本低 |

#### 2. 生产运营战略的竞争重点(TQCF)

| 竞争重点 | 含义 |
|---------|------|
| **交货期(Time)** | 比竞争对手更快捷地响应顾客的需求,体现在新产品推出、交货期等方面。包括快速交货和按约交货 |
| **质量(Quality)** | 全面质量,既包括产品本身的质量,也包括生产过程的质量 |
| **成本(Cost)** | 包括生产成本、制造成本、流通成本和使用成本等 |
| **制造柔性(Flexibility)** | 快速而又低成本地适应市场需求,反映生产运作系统对外部环境做出反应的能力 |

#### 3. 生产流程与产能计划:平衡生产能力与市场需求

| 产能计划的类型 | 含义 |
|---------------|------|
| **领先策略(进攻型)** | 根据对需求增长的预期增加产能 |
| **滞后策略(保守型)** | 仅当企业因需求增长而满负荷生产或超额生产后才增加产能 |
| **匹配策略(稳健型)** | 少量地增加产能来应对市场需求的变化 |

| 平衡产能与需求的方法 | 顺序 |
|---------------------|------|
| **资源订单式生产** | 订单→资源→生产 |
| **订单生产式生产** | 资源→订单→生产 |
| **库存生产式生产** | 资源→生产→订单 |

#### 4. 供应链与配送网络选择

| 类型 | 含义 |
|-----|------|
| 制造商存货加直送 | 产品绕过零售商直接从制造商发送到最终顾客 |
| 制造商存货、直送加在途并货 | 将来自不同地点的订单组合起来,使顾客只需接收一次交付 |
| 分销商存货加承运人交付 | 由分销商或零售商存放在中间仓库里,使用包裹承运人运送到最终顾客 |
| 分销商存货加到户交付 | 分销商或零售商将产品交付到顾客家门而不通过承运人 |
| 制造商或分销商存货加顾客自提 | 存货存放在制造商或分销商的仓库,顾客到指定提货点领取商品 |
| 零售商存货加顾客自提 | 库存存放在零售店,顾客走进零售店购货或在线下订单后提货 |

### 三、采购战略

#### 1. 货源策略

| 策略 | 优点 | 缺点 |
|-----|------|------|
| **少数或单一货源策略** | 建立稳固关系;信息保密;产生规模经济,享受价格优惠;获得高质量供应品 | 单一供应商议价能力增强;容易遭受供应中断风险 |
| **多货源少批量策略** | 保证稳定供应;获得更多知识和技术;议价能力增强 | 联系不够稳固;不能产生规模经济;不能获得高质量供应品 |
| **平衡货源策略** | 在少数货源和多货源之间寻求均衡 |  |

#### 2. 交易策略

| 策略类型 | 含义 | 适用条件 |
|---------|------|---------|
| **市场交易策略** | 通过与供应商签订买卖合同在市场上取得所需供应品 | ①供应品技术含量较低或生产技术成熟<br>②供应品在企业产品生产和销售中不具有重要性<br>③企业不需要供应商提供售后服务<br>④供应商所处市场较为成熟<br>⑤供应商数量较多,竞争激烈 |
| **短期合作策略** | 为应对一定市场需求对供应商采取短期合作 | ①产品面临急剧变化的市场机会和灵活的客户需求<br>②供应品供给具有较高的适应性<br>③供应品有较高技术含量,对企业产品设计、生产、销售有重要影响 |
| **功能性联盟策略** | 与供应商通过订立协议结成联盟 | ①供应品在企业产品生产经营中起着重要作用<br>②企业对供应品的需求量比较大<br>③供应品生产技术成熟,可替代性较高<br>④供应商具有较强生产能力和实现规模经济的能力 |
| **创新性联盟策略** | 为了产品、业务的创新并取得长期竞争优势而与供应商结成联盟 | 从某种新产品概念提出就开始与供应商合作,产品设计、试制、改进、定型、生产与供应商技术创新基本同步进行 |

#### 3. 采购模式

| 模式类型 | 特点 |
|---------|------|
| **传统采购模式** | 根据库存情况制定采购计划;信息沟通不充分;简单的供需关系;以补充库存为目的;管理简单粗放,采购成本居高不下 |
| **MRP采购模式** | 以生产为导向,根据生产计划计算推导需要购买的原材料、零部件数量和进货时间;生产计划和采购计划精细;采购计划计算和编制复杂,需要借助计算机技术 |
| **JIT采购模式(准时化采购)** | 供应商把适当数量、适当质量的物品在适当的时间送达适当的地点;供应商数量少甚至是单一;建立长期稳定的合作关系;采购批量小,送货频率高;关心对方产品改进和创新,信息共享快速可靠 |
| **VMI采购模式(供应商管理库存)** | 由供应商管理企业库存,确定最佳库存量,制定并执行库存补充措施;建立长期稳定的深层次合作关系;打破各自为政的采购和库存管理模式,共享实时信息;按照利益共享、风险共担原则协商确定费用分担和利润分成比例 |
| **数字化采购模式** | 通过人工智能、物联网、云端协同等技术,实现采购全流程的智慧管理;以数字化平台为基础建立新型合作关系;自动化技术淘汰大量人工操作;采购管理的科学性、便捷性、精细性、准确性空前提高,"降本增效"极为显著 |

### 四、人力资源战略

#### 1. 人力资源规划的步骤
1. 调查、收集和整理涉及企业战略决策和经营环境的各种信息
2. 确定人力资源规划的期限、范围和性质,建立人力资源信息系统
3. 在分析人力资源供给和需求影响因素的基础上,对企业未来人力资源供求进行预测
4. 制订人力资源供求平衡的总计划和各项业务计划

#### 2. 人力资源获取策略

| 策略 | 成本领先 | 差异化 | 集中化 |
|-----|---------|--------|--------|
| 员工来源 | 外部 | 内部 | 两者兼顾 |
| 晋升阶梯 | 狭窄、不宜转换 | 广泛、灵活 | 狭窄、不宜转换 |
| 甄选决策 | 人力资源部 | 业务部门 | 结合两者 |
| 甄选方法 | 简历和面试为主 | 多重方法 | 心理测试 |
| 甄选标准 | 强调技能 | 强调与文化契合 | 结合两者 |
| 社会化过程 | 正式的雇佣和社会化过程 | 非正式的雇佣和社会化过程 | 结合两者 |

#### 3. 人力资源培训与开发

| 战略类型 | 匹配特点 |
|---------|---------|
| 成本领先战略 | 强调个人能力,范围有限的知识和技巧,实施个人在职培训;通过自己设立企业大学或定期培训提升员工知识和能力 |
| 差异化战略 | 强调公司与其他企业的不同之处,要求具有广泛的知识、技巧和创造性;传递外部新颖信息、购买所需技能或利用外部培训机构对团队进行培训 |
| 集中化战略 | 对专门领域的知识需求更迫切,强调应用范围适中的知识和技巧(专有知识,不易转换和共享);利用在职培训或外部培训,自己培养技能或购买技能 |

#### 4. 人力资源绩效评估

| 战略类型 | 匹配特点 |
|---------|---------|
| 成本领先战略 | 强调结果导向,以控制成本为目的,评估范围狭窄,评估信息来源单一,上级作为考核主要考官 |
| 差异化战略 | 评估内容涉及行为和结果两种指标,评估范围宽广,评估信息丰富,主要用于员工的发展和素质提升 |
| 集中化战略 | 绩效管理目的、内容、范围及其结果应用等倾向于二者的结合 |

#### 5. 人力资源薪酬激励

**薪酬的组成:**
1. 基本薪酬(根据工作或技能支付的稳定经济收入)
2. 可变薪酬(根据绩效支付的变动性质经济收入)
3. 间接薪酬(给员工提供的各种福利)

**公平性原则:**
1. 外部公平性(不同企业类似职位薪酬基本相同)
2. 内部公平性(同一企业不同职位薪酬与各自对企业贡献成正比)
3. 个体公平性(同一企业相同或类似职位员工,薪酬与其能力、贡献成正比)

**薪酬水平策略:**
1. 领先型策略(薪酬水平高于市场平均水平)
2. 匹配型策略(薪酬水平与市场平均水平保持一致)
3. 拖后型策略(薪酬水平明显低于市场平均水平)
4. 混合型策略(针对企业内部不同职位采用不同策略)

**企业竞争战略与薪酬策略:**

| 战略类型 | 工资基础 | 使用薪酬类型 | 决策过程 |
|---------|---------|------------|---------|
| 成本领先战略 | 岗位或年资 | 固定薪酬(基本薪酬) | 强调集权,高层作出决策 |
| 差异化战略 | 能力或绩效 | 浮动薪酬(可变薪酬) | 授权中层或子公司进行决策 |
| 集中化战略 | 能力与绩效的结合 | 固定薪酬和浮动薪酬一起使用 | 将授权与分权统一 |

### 五、财务战略

#### 1. 财务战略的概念
财务战略分为筹资战略和资金管理战略。狭义的财务战略仅指筹资战略,包括资本结构决策、筹资来源决策和股利分配决策等。

#### 2. 融资渠道与方式

| 融资方式 | 优点 | 缺点 |
|---------|------|------|
| **内部融资** | 管理层自主性强,降低融资成本 | 融资数量有限,对企业盈利能力要求高 |
| **股权融资** | 没有固定的股利支付压力,适宜大量资金需求 | 引起控制权变更,融资成本比较高 |
| **债权融资-贷款** | 融资成本较低、速度较快,方式隐蔽 | 限制较多,额度有限,需按期还本付息,压力大 |
| **债权融资-租赁** | 不需要额外融资,享有税收优惠,增加资本回报率 | 使用权有限,资产所有权不属于企业 |
| **资产销售融资** | 简单易行,不用稀释股东权益 | 无回旋余地,销售时机不准可能低于资产价值 |

#### 3. 财务风险与经营风险的搭配

| 搭配类型 | 经营风险 | 财务风险 | 说明 |
|---------|---------|---------|------|
| **高经营风险+低财务风险** | 非常高 | 非常低 | 反向搭配,可同时符合权益投资人和债权人期望,是现实搭配 |
| **低经营风险+高财务风险** | 低 | 高 | 反向搭配,是现实搭配 |
| **高经营风险+高财务风险** | 高 | 高 | "双高搭配",符合风险投资人要求,不符合债权人要求,难以实现 |
| **低经营风险+低财务风险** | 低 | 低 | "双低搭配",对债权人是理想资本结构,不符合权益投资人期望,不是现实搭配 |

#### 4. 产品生命周期不同阶段的财务战略

| 生命周期阶段 | 导入期 | 成长期 | 成熟期 | 衰退期 |
|-------------|-------|-------|-------|-------|
| 经营风险 | 非常高 | 高 | 中等 | 低 |
| 财务风险 | 非常低 | 低 | 中等 | 高 |
| 资本结构 | 股东权益 | 主要是股东权益 | 股东权益+债务 | 股东权益+债务 |
| 资金来源 | 风险资本 | 权益投资增加 | 保留盈余+债务 | 债务 |
| 股利 | 不分配 | 分配率很低 | 分配率高 | 全部分配 |
| 市盈率 | 非常高 | 高 | 中 | 低 |
| 股价 | 迅速增长 | 增长并波动 | 稳定 | 下降并波动 |

#### 5. 基于价值创造或增长率的财务战略选择(财务战略矩阵)

| 情况 | 财务战略 |
|-----|---------|
| **增值型现金短缺(第一象限)**<br>(投资资本回报率-资本成本)>0<br>(销售增长率-可持续增长率)>0 | ①如果高速增长是暂时的,通过借款(周转短期借款)筹集资金<br>②如果高速增长是长期的,提高可持续增长率(提高经营效率、改变财务政策)或增加权益资本(增发股份、兼并成熟企业) |
| **增值型现金剩余(第二象限)**<br>(投资资本回报率-资本成本)>0<br>(销售增长率-可持续增长率)<0 | 首选战略是利用剩余现金加速增长。途径:内部投资、收购相关业务。若仍有剩余,增加股利支付、回购股份 |
| **减损型现金剩余(第三象限)**<br>(投资资本回报率-资本成本)<0<br>(销售增长率-可持续增长率)<0 | 首选战略是提高投资资本回报率(提高税后经营利润率、提高经营资产周转率)或将企业出售 |
| **减损型现金短缺(第四象限)**<br>(投资资本回报率-资本成本)<0<br>(销售增长率-可持续增长率)>0 | ①如果盈利能力低是本公司独有的问题,选择"彻底重组"<br>②如果盈利能力低是整个行业衰退引起的,选择"尽快出售" |

## 第六节 国际化经营战略

### 一、企业国际化经营动因

| 动因 | 案例材料表现 |
|-----|-------------|
| **寻求市场** | — |
| **寻求效率** | 投资国生产成本上升,特别是劳动力成本;来自于低成本生产商的竞争 |
| **寻求资源** | 取得更多自然资源/原材料的供应,如石油、天然气、金属和非金属矿产 |
| **寻求现成资产** | 从发达国家获得品牌、专利、销售渠道、生产设施、管理经验、先进技术、资金、规模经济等 |

### 二、国际化经营的主要方式

#### 1. 出口贸易

**目标市场选择:**

| 目标市场的区域路径 | 说明 |
|-------------------|------|
| **传统方式(连续方式)** | 高新技术产品:先到经济技术发展水平类似的发达国家,再到发展中国家<br>发展中国家农产品、矿产品等初级产品和劳动密集型产品:主要流向发达国家 |
| **新型方式(不连续方式)** | 不论发达国家还是发展中国家,高新技术产品出口路径都是先到发达国家(特别是美国),然后再走向发展中国家 |

#### 2. 对外直接投资

| 类型 | 优点 | 缺点 |
|-----|------|------|
| **全资子公司** | ①管理者可以完全控制子公司日常经营活动<br>②确保有价值的技术、工艺和其他无形资产保留在子公司<br>③避免合资经营各方利益、目标冲突<br>④使国外子公司经营战略与总体战略融为一体 | ①耗费大量资金<br>②难以得到当地政策与经营资源支持,规避政治风险能力小 |
| **合资经营** | ①减少国际化经营的资本投入<br>②弥补跨国经营经验不足,吸引和利用东道国合资方资源 | 协调成本可能过大 |

#### 3. 非股权形式

非股权形式包括合约制造(OEM、ODM)、服务外包、订单农业、特许经营、许可经营、管理合约及其他类型的合约关系。

- **OEM(原始设备制造商):** A设计,B生产,A品牌,A销售(代工,代生产)
- **ODM(原始设计制造商):** B设计,B生产,A品牌,A销售(贴牌,别人的品牌)

### 三、全球价值链中的企业国际化经营

#### 1. 全球价值链中企业的角色定位

| 角色定位 | 说明 |
|---------|------|
| **领先企业** | 核心、主导,承担核心技术研发和营销渠道构建等功能 |
| **一级供应商** | 技术能力较强、具有较高成本优势,承担部件生产、组装、物流等外围管理工作 |
| **其他层级供应商** | 微弱比较优势,承接非关键环节的非核心生产活动,如简单组装、初始设备制造(OEM)等 |
| **合同制造商** | 具备一定技术能力,能够承接领先企业对技术有一定要求的产品生产,也可独立完成产品部分结构生产 |

#### 2. 全球价值链的分工模式

| 分工模式 | 产生条件 | 领先企业与供应商关系实现方式 | 协调性 |
|---------|---------|---------------------------|--------|
| **科层型(等级型)** | 非常复杂,不易编码;缺乏外部供应商,供应商垂直一体化 | 对外直接投资并购或新建 | 高 |
| **市场型** | 结构简单,容易编码;充足外部供应商,交易伙伴间没有正式合作 | 国际贸易 | 低 |
| **俘获型(领导型)** | 非常复杂,很难编码;锁定能力不足的供应商 | 非股权形式 | 较高 |
| **模块型** | 模块化特征,标准化 | 非股权形式 | 较低 |
| **关联型(关系型)** | 非常复杂,难以编码;供应商能力较强,相互依赖平等合作 | 非股权形式 | 中等 |

#### 3. 全球价值链与发展中国家企业升级

| 升级类型 | 含义 | 例子 |
|---------|------|------|
| **工艺升级** | 通过生产技术的改进和生产组织管理效率提升实现升级 | 为满足农业生产更高标准,供应商采用"良好农业规范"(GAP) |
| **产品升级** | 通过改进产品设计(甚至开发突破性产品)提高产品竞争力 | 旅游业企业提供更高品质酒店或增加生态旅游和医疗旅游等高档次产品 |
| **功能升级** | 通过占领价值链更高附加值的环节实现升级 | 从初始设备制造商(OEM)到初始设计制造商(ODM) |
| **价值链升级** | 通过进入技术壁垒或资本壁垒更高的价值链或获取价值链中更高地位提升盈利能力和竞争力 | 汽车零部件供应商进入整车制造产业;从OEM到ODM再到自有品牌制造商(OBM) |

### 四、国际化经营的战略类型

| 类型 | 主要特征 |
|-----|---------|
| **国际战略** | 研发在母国,推向各国市场;国外分支机构负责制造、营销,当地没有决策权;适应性较差,经营成本高 |
| **多国本土化战略** | 针对各国需求研发,产品在当地生产和销售,当地具有决策权;不同国家生产销售的产品不一样;适应性较好,经营成本高,高度分权 |
| **全球化战略** | 生产什么由总部统一决定,产品生产的不同环节配置在不同国家;不同国家生产销售的产品是标准化的;适应性较差,经营成本低,高度集权 |
| **跨国战略** | 综合多国本土化战略和全球化战略的做法;适应性较好,经营成本低 |

### 五、新兴市场的企业战略

#### 本土企业的战略选择

| 本土企业类型 | 核心策略 | 具体措施 |
|-------------|---------|---------|
| **"防御者"** | 利用本土优势进行防御 | ①把目光集中于喜欢本国产品的客户<br>②频繁调整产品和服务,适应客户特别需求<br>③加强分销网络的建设和管理 |
| **"扩张者"** | 向海外延伸本土优势 | 寻找在消费者偏好、地缘关系、分销渠道或政府管制方面与本国市场相类似的市场 |
| **"躲闪者"** | 避开跨国竞争对手的冲击 | ①与跨国竞争对手建立合资、合作企业<br>②将企业出售给跨国竞争对手<br>③重新定义核心业务,避开与跨国竞争对手直接竞争<br>④专注细分市场,将业务重心转向价值链的某些环节<br>⑤生产与跨国竞争对手产品互补的产品 |
| **"抗衡者"** | 在全球范围内对抗 | ①不要拘泥于成本上竞争,比照行业领先公司衡量实力<br>②找到一个定位明确又易于防守的市场<br>③在全球化产业中找到合适突破口<br>④学习从发达国家获取资源,克服自身技能不足和资本匮乏 |
