### 作业成本法的概念（含挖空 ）
|对比维度|作业成本法|传统成本法|
| ---- | ---- | ---- |
|直接成本计入产品| $\boldsymbol{直接计入}$ | $\boldsymbol{直接计入}$ |
|间接成本分配 - 第一步| 先按 $\boldsymbol{资源成本动因}$ 分配计入 $\boldsymbol{各项作业}$ | 先将资源分配计入 $\boldsymbol{部门}$ |
|间接成本分配 - 第二步| 再按 $\boldsymbol{作业成本动因}$ 计入各项产品 | 再按 $\boldsymbol{产量或工时}$ 计入各项产品 |
|资源成本动因| 引起 $\boldsymbol{作业成本}$ 增加，将 $\boldsymbol{资源}$ 成本分配给 $\boldsymbol{作业}$ | - |
|作业成本动因| 引起 $\boldsymbol{产品成本}$ 增加，将 $\boldsymbol{作业}$ 成本分配给 $\boldsymbol{产品}$ | - |
|成本分配方式| 用 $\boldsymbol{追溯}$、$\boldsymbol{动因分配}$ ，减少 $\boldsymbol{分摊}$ | - |
|成本分配精度| 最高：$\boldsymbol{追溯}$ ；最低：$\boldsymbol{分摊}$ | - |  


### 作业成本的计算步骤（含挖空连线 ）
#### 作业成本库设计（连线 ）
|成本/动因描述|对应作业成本库层级|
| ---- | ---- |
|财产税、工厂安保成本、行政管理成本、服务于整个工厂| $\boldsymbol{生产维持级成本}$ |
|产品设计成本、产品更新成本、工艺改造成本、随品种数成比例变动、同时服务于一种型号| $\boldsymbol{品种(产品)级成本}$ |
|产品抽检成本、与批数成比例变动、同时服务于一批产品、生产前的机器调试成本| $\boldsymbol{批次级成本}$ |
|直接人工成本、机器加工组装成本、每件产品至少执行一次、与产量成比例变动| $\boldsymbol{单位级成本}$ | 

#### 作业成本动因（挖空 ）
| 类型                   | 定义               |
| -------------------- | ---------------- |
| $\boldsymbol{业务}$ 动因 | 以执行次数为作业动因       |
| $\boldsymbol{持续}$ 动因 | 以执行一项作业所需时间为作业动因 |
| $\boldsymbol{强度}$ 动因 | 单独归集资源，直接计入特定产品  |

### 作业成本法计算（含挖空，与表格对应 ）
## 已知条件（生产数量、制造费用明细 ）
### 生产数量（甲、乙产品 ）
|产品|月初在产品|本月投产|本月完工|月末在产品|
| ---- | ---- | ---- | ---- | ---- |
|甲产品| $\boldsymbol{M}$ | $\boldsymbol{N}$ | $\boldsymbol{O}$ | $\boldsymbol{P}$（完工程度 50% ）|
|乙产品| $\boldsymbol{Q}$ | $\boldsymbol{R}$ | $\boldsymbol{S}$ | $\boldsymbol{T}$（完工程度 50% ）| 

### 制造费用明细
- 机器调试：费用 $\boldsymbol{X}$ ，甲执行 $\boldsymbol{x_1}$ 次、乙执行 $\boldsymbol{x_2}$ 次  
- 工艺改造：费用 $\boldsymbol{Y}$ ，甲执行 $\boldsymbol{y_1}$ 次、乙执行 $\boldsymbol{y_2}$ 次  
- 工厂安保：费用 $\boldsymbol{Z}$  


## 传统成本法 - 成本分配（以直接人工比例分配制造费用 ）
### 甲产品成本计算单
|项目|直接材料|直接人工|制造费用|
| ---- | ---- | ---- | ---- |
|月初在产品成本| $\boldsymbol{A_0}$ | $\boldsymbol{B_0}$ | $\boldsymbol{C_0}$ |
|本月发生的成本| $\boldsymbol{A_1}$ | $\boldsymbol{B_1}$ | $\boldsymbol{\frac{X + Y + Z}{B_1 + E_1}×B_1}$ |（$E_1$ 为乙产品本月直接人工 ）
|完工产品成本| $\boldsymbol{\frac{A_0 + A_1}{O + P}×O}$ | $\boldsymbol{\frac{B_0 + B_1}{O + 50\%P}×O}$ | $\boldsymbol{C_0 + \frac{X + Y + Z}{B_1 + E_1}×B_1 \over O + 50\%P}×O$ |
|在产品成本| $\boldsymbol{\frac{A_0 + A_1}{O + P}×P}$ | $\boldsymbol{\frac{B_0 + B_1}{O + 50\%P}×50\%P}$ | $\boldsymbol{C_0 + \frac{X + Y + Z}{B_1 + E_1}×B_1 \over O + 50\%P}×50\%P$ | 

### 乙产品成本计算单
|项目|直接材料|直接人工|制造费用|
| ---- | ---- | ---- | ---- |
|月初在产品成本| $\boldsymbol{D_0}$ | $\boldsymbol{E_0}$ | $\boldsymbol{F_0}$ |
|本月发生的成本| $\boldsymbol{D_1}$ | $\boldsymbol{E_1}$ | $\boldsymbol{\frac{X + Y + Z}{B_1 + E_1}×E_1}$ |
|完工产品成本| $\boldsymbol{\frac{D_0 + D_1}{S + T}×S}$ | $\boldsymbol{\frac{E_0 + E_1}{S + 50\%T}×S}$ | $\boldsymbol{F_0 + \frac{X + Y + Z}{B_1 + E_1}×E_1 \over S + 50\%T}×S$ |
|在产品成本| $\boldsymbol{\frac{D_0 + D_1}{S + T}×T}$ | $\boldsymbol{\frac{E_0 + E_1}{S + 50\%T}×50\%T}$ | $\boldsymbol{F_0 + \frac{X + Y + Z}{B_1 + E_1}×E_1 \over S + 50\%T}×50\%T$ |

# 作业成本法的管理及优缺点（含挖空）
## 管理 - 作业类型连线
|作业|类型|
| ---- | ---- |
|产品检验| 非增值作业 |
|材料堆积| 非增值作业 |
|迂回运送| 非增值作业 |
|废品清理| 非增值作业 |
|零件组装| 增值作业 |
|次品处理| 非增值作业 |
|次品返工| 非增值作业 |
|核对订单| 非增值作业 |
|寻找客户订单| 非增值作业 |
|修剪产成品边角余料| 增值作业 | 

## 优点
1. 成本计算更 $\boldsymbol{准确}$ ，扩大 $\boldsymbol{追溯}$ 分配比例  
2. 成本控制与管理更 $\boldsymbol{有效}$  
3. 为 $\boldsymbol{价值链分析}$、$\boldsymbol{成本领先战略}$ 提供支持  


## 缺点
1. $\boldsymbol{开发和维护费用}$ 较高  
2. 不符合对外 $\boldsymbol{财务报告}$ 需求  
3. 确定 $\boldsymbol{成本动因}$ 较难  
4. 成本库与企业 $\boldsymbol{组织结构}$ 不一致，不利组织控制
