# 考点1 重要 5种资本结构理论梳理
| 理论名称   | 财务杠杆升高时变化                       | 最优资本结构/价值公式                                                                                 | 特殊要点                          |     |
| ------ | ------------------------------- | ------------------------------------------------------------------------------------------- | ----------------------------- | --- |
| 无税MM理论 | 企业价值不变，权益成本升高，负债成本不变，加权平均资本成本不变 | 最优资本结构：无<br>有负债企业价值 = <span class="cloze-span">无负债企业价值 + 无</span>                           | 不考虑税盾，资本结构不影响企业价值             |     |
| 有税MM理论 | 企业价值升高，权益成本升高，负债成本不变，加权平均资本成本降低 | 最优资本结构<span class="cloze-span">：全部是负债融资时<br></span><br>有负债企业价值 = 无负债企业价值 + 利息抵税收益的现值        | 税盾提升企业价值，负债越多价值越高             |     |
| 权衡理论   | -（未单独列财务杠杆影响，聚焦价值公式修正 ）         | 有负债企业价值 = <span class="cloze-span">无负债企业价值 + 利息抵税收益的现值 - 财务困境成本的现值</span>                   | 平衡税盾收益与财务困境成本                 |     |
| 代理理论   | -（未单独列财务杠杆影响，聚焦价值公式修正 ）         | 有负债企业价<span class="cloze-span">值 = 无负债企业价值 + 利息抵税收益的现</span>值 - 财务困境成本的现值 - 债务代理成本 + 债务代理收益 | 引入代理成本/收益，细化价值权衡              |     |
| 优序融资理论 | -（未列财务杠杆影响，聚焦融资顺序 ）             | 筹资优先顺<span class="cloze-span">序</span>：留存收益＞普通债务＞可转换公司债券＞优先股＞普通股                            | 遵循 “内源-债务-权益” 顺序，避开发行成本与信息不对称 |     |

**核心逻辑**：从无税到有税，MM理论奠定资本结构与企业价值关系基础；权衡、代理理论逐步纳入税盾外的成本（财务困境、代理 ）；优序融资理论从融资决策顺序补充，共同构建资本结构理论体系 。

# 考点3 普通 企业价值比较法梳理
| 类别         | 内容                                                                 |
| ------------ | -------------------------------------------------------------------- |
| 最优资本结构 | 以 $\boldsymbol{企业价值}$ 最大化为标准确定最佳资本结构，此时 $\boldsymbol{加权平均资本成本}$ 最小 |
| 假设         | ① 经营利润永续；② 股东要求的回报率不变；③ 股利支付率为100%                            |
| 公式         | 股权价值 = $\boldsymbol{\dfrac{(EBIT - I)×(1 - T) - PD}{R_s}}$                        |  

**核心逻辑**：通过假设经营利润永续等条件，用特定公式计算股权价值，以企业价值最大、加权平均资本成本最小为标准，确定最优资本结构 。

# 考点4 重要 三大杠杆（经营、财务、联合）梳理
| 杠杆类型     | 经营杠杆（DOL）                                                                                                                                                                                                                                       | 财务杠杆（DFL）                                                                              | 联合杠杆（DTL）                                                                                                                                                                                         |     |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| **含义**   | 固定经营成本是根源，销量小变动→息税前利润（EBIT）大变动                                                                                                                                                                                                                  | 固定融资成本是根源，息税前利润（EBIT）小变动→每股收益（EPS）大变动                                                  | 固定经营成本 + 固定融资成本是根源，销量小变动→每股收益（EPS）大变动                                                                                                                                                             |     |
| **定义公式** | $DOL = \dfrac{息税前利润变动率}{销量变动率} = \dfrac{\triangle EBIT / EBIT}{\triangle Q / Q}$                                                                                                                                                                | $DFL = \dfrac{每股收益变动率}{息税前利润变动率} = \dfrac{\triangle EPS / EPS}{\triangle EBIT / EBIT}$ | $DTL = \dfrac{每股收益变动率}{销量变动率} = \dfrac{\triangle EPS / EPS}{\triangle Q / Q}$                                                                                                                     |     |
| **计算公式** | $DOL = \dfrac{基期边际贡献}{基期息税前利润} = \dfrac{基期M}{基期EBIT}$ <br>$= \dfrac{基期边际贡献}{基期边际贡献 - 固定经营成本} = \dfrac{基期M}{基期M - F}$ <br>$= \dfrac{基期息税前利润 + 固定经营成本}{基期息税前利润} = 1 + \dfrac{F}{基期EBIT}$ <br>本量利：$DOL = \dfrac{销量敏感系数}{安全边际率} = \dfrac{1}{安全边际率}$ | $DFL = \dfrac{基期息税前利润}{归属于普通股股东的税前利润} = \dfrac{基期EBIT}{基期EBIT - I - PD÷(1 - T)}$       | $DTL = 经营杠杆×财务杠杆$ <br>$= \dfrac{基期边际贡献}{基期归属于普通股股东的税前利润} = \dfrac{基期M}{基期M - F - I - PD÷(1 - T)}$ <br>$= \dfrac{基期息税前利润 + 固定经营成本}{基期归属于普通股股东的税前利润} = \dfrac{基期EBIT + F}{基期EBIT - I - PD÷(1 - T)}$ |     |
| **影响因素** | 单价↓→经营风险↑；销量↑→经营风险↓；变动成本↓→经营风险↓；固定经营成本↑→经营风险↑；利息/所得税率↓→经营风险不变                                                                                                                                                                                   | 单价↓→财务风险↑；销量↑→财务风险↓；变动成本↓→财务风险↓；固定经营成本↑→财务风险↑；利息↓→财务风险↓；所得税率（有优先股时）↑→财务风险↑             | 单价↓→总风险↑；销量↑→总风险↓；变动成本↓→总风险↓；固定经营成本↑→总风险↑；利息↓→总风险↓；所得税率（有优先股时）↑→总风险↑                                                                                                                              |     |

**核心逻辑**：经营杠杆聚焦 “经营成本 - 销量 - EBIT” 联动，财务杠杆聚焦 “融资成本 - EBIT - EPS” 联动，联合杠杆整合两者，体现 “销量 - EPS” 总杠杆效应，用于衡量风险与经营/融资决策的关联 。

杠杆类型	=定义公式（预测未来）=	计算公式（计算当期）
经营杠杆	息税前利润变动率 / 销量变动率	=基期边际贡献 / 基期息税前利润
财务杠杆	每股收益变动率 / 息税前利润变动率	=基期息税前利润 / 基期归普税前利润
联合杠杆	每股收益变动率 / 销量变动率	=基期边际贡献 / 基期归普税前利润
> （注：“归普税前利润” 指归属于普通股股东的税前利润 ，用于快速区分三类杠杆的定义与计算逻辑 ，辅助记忆不同场景下的公式应用 ）
