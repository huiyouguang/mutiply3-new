# 生产决策（含挖空，与表格对应 ）
## 保留或关闭生产线
- 前提：剩余生产能力无法转移  
- 决策：$\boldsymbol{亏损产品的边际贡献 \boldsymbol{> 0}}$ 时，不应停产  


## 零部件自制或外购（决策指标：相关成本 ）
|场景|相关成本包含|
| ---- | ---- |
|产能可转移 + 需新增投资| <u>变动成本</u>、<u>机会成本</u>、<u>专属成本</u> |
|产能无法转移 + 无需新增投资| <u>变动成本</u> |
|产能无法转移 + 需新增投资| <u>变动成本</u>、<u>专属成本</u> |
|产能可转移 + 无需新增投资| <u>变动成本</u>、<u>机会成本</u> |  


## 特殊订单是否接受（决策指标：相关损益 ）
|场景|相关成本包含|
| ---- | ---- |
|减少正常订单 + 产能可转移 + 需新增投资| <u>变动成本</u>、<u>机会成本</u>、<u>专属成本</u> |
|减少正常订单 + 产能无法转移 + 不新增投资| <u>变动成本</u>、<u>机会成本</u> |
|不影响正常订单 + 产能可转移 + 不新增投资| <u>变动成本</u>、<u>机会成本</u> |
|不影响正常订单 + 产能无法转移 + 需新增投资| <u>变动成本</u>、<u>专属成本</u> |
|减少正常订单 + 产能可转移 + 不新增投资| <u>变动成本</u>、<u>机会成本</u> |
|减少正常订单 + 产能无法转移 + 需新增投资| <u>变动成本</u>、<u>机会成本</u>、<u>专属成本</u> |
|不影响正常订单 + 产能无法转移 + 不新增投资| <u>变动成本</u> |
|不影响正常订单 + 产能可转移 + 需新增投资| <u>变动成本</u>、<u>机会成本</u>、<u>专属成本</u> |  


## 约束资源最优利用
- 决策：优先安排 $\boldsymbol{单位约束资源的边际贡献}$ 最大的产品  
- 计算公式：$\boldsymbol{\frac{单位产品边际贡献}{该单位产品耗用的约束资源量}}$  


## 出售或深加工（决策：深加工增加收入 - 增加成本 ）
|场景|相关成本包含|
| ---- | ---- |

|产能可转移 + 无需新增投资| <u>变动成本</u>、<u>机会成本</u> |

# 定价决策（含挖空，与表格对应 ）
## 成本加成定价法
|方法|成本基数|加成内容|
| ---- | ---- | ---- |
|完全成本加成法| <u>制造成本</u> | <u>非制造成本 + 利润</u> |
|变动成本加成法| <u>变动成本</u> | <u>固定成本 + 利润</u> |  


## 新产品定价
|方法|策略周期|适合产品生命周期|
| ---- | ---- | ---- |
|撇脂定价法| <u>短期</u> | 较 <u>短</u> |
|渗透定价法| <u>长期</u> | 较 <u>长</u> |  


## 有闲置能力条件下的定价
- 前提：生产能力无法转移、不追加投资、不影响原有订单  
- 价格区间：介于 $\boldsymbol{变动成本}$ 与 $\boldsymbol{目标价格}$ 之间
