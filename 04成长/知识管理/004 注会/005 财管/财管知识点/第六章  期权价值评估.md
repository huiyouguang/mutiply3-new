# 期权策略特点梳理
|                 | 策略一                                                        | 策略二                                                     |     |
| --------------- | ---------------------------------------------------------- | ------------------------------------------------------- | --- |
| 特点 - 锁定内容       | 锁定了 $\underline{\text{最低净收入和最低净损益}}$ ；                     | 锁定了 $\underline{\text{最高净收入和最高净损益}}$ ；                  |     |
| 特点 - 对净损益影响     | 降低了 $\underline{\text{净损益的预期}}$                            | 是机构还是个人投资者常用的投资策略 $\underline{\text{机构}}$               |     |
| 情景① - 极端结果      | 最坏的结果是股价 $\underline{\text{与执行价格一致}}$ ，白白损失两个期权的购买成本       | 最好的结果是股价 $\underline{\text{与执行价格一致}}$ ，白白赚取两个期权的出售价格    |     |
| 情景① - 净收益条件     | 股价偏离执行价格的差额必须 $\underline{\text{超过两份期权费}}$ ，才能给投资者带来净收益    | 股价偏离执行价格的差额必须 $\underline{\text{小于两份期权费}}$ ，才能给投资者带来净收益 |     |
| 情景② - 适用市场预期    | 对于预计市场价格将 $\underline{\text{发生剧烈变动，但是不知道升高还是降低}}$ 的投资者非常有用 | 对于预计市场价格将 $\underline{\text{相对比较稳定}}$ 的投资者非常有用          |     |
| 补充说明（可结合实际知识完善） | 多为**跨式期权（Straddle）** 等偏激进、博弈价格大幅波动的策略逻辑                    | 类似**反向跨式/勒式期权（反向 Strangle 等）** ，适合预判行情震荡收敛场景            |     |

# 复制原理和风险中性原理（挖空版）
假设 $S_u > X$，$S_d < X$，请填写以下表格横线处内容：

|          | 复制原理                                                                                                                                                                      | 风险中性原理                                                                                                                                                                  |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **计算方法** | 股价上行时<br>期权价值 $C_u = S_u - X$<br>投资组合价值 $= S_u×H - B×(1 + r)$<br>令二者相等，得到 $S_u - X = S_u×H - B×(1 + r)$                                                                   | 股价上行时<br>期权价值 <span class="cloze-span">$C_u = S_u - X$</span>                                                                                                           |
|          | 股价下行时<br>期权价值 $C_d = \underline{0}$<br>投资组合价值 $= S_d×H - B×(1 + r)$<br>令二者相等，得到 $0 = S_d×H - B×(1 + r)$                                                                   | 股价下行时<br>期权价值 <span class="cloze-span">$C_d = \underline{0}$</span>                                                                                                     |
|          | 公式<br>套期保值率 $H = \frac{C_u - C_d}{S_u - S_d} = \frac{S_u - X}{S_u - S_d}$<br>借款额 $B = \frac{S_d×H}{1 + r}$<br>期权价值 $C_0 = S_0×H - B$                                      | 无风险利率 $r =$ 股票上行收益率 $×$ 上行概率 $+$ 股票下行收益率 $×(1 -$ 上行概率 $)$，可求出上行概率<br>期权价值 $C_0 = \frac{C_u×\text{上行概率} + C_d×\text{下行概率}}{1 + r} = \frac{(S_u - X)×\text{上行概率}}{1 + r}$ |
