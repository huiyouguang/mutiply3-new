# 企业价值评估对象与价值类型连线对应关系

| 价值类型      | 所属类别                                       |     |
| --------- | ------------------------------------------ | --- |
| 整体价值      | 是企业价值评估的对象                                 |     |
| 实体价值      | 是企业价值评估的对象                                 |     |
| 股权价值      | 是企业价值评估的对象                                 |     |

| 少数股权价值    | <u>可能是</u>企业价值评估的对象                        |     |

可能是企业价值评估的对象|     |

| 未来现金净流量现值 | 是企业价值评估的对象                                 |     |

说明：企业价值评估聚焦于能反映企业整体经营、权益等层面的价值，像整体价值、实体价值等直接关乎企业运营与权益的，是典型评估对象；会计价值多基于会计核算，现时市场价值受短期市场波动影响大，各项资产价值汇总未体现企业整体协同，一般不属评估对象；而公允价值、清算价值等，因评估目的（如处置、特殊交易场景 ）等情况，<u>可能成为</u>评估对象，体现其场景化的应用特点 。

# 考点2 控股权溢价及净现值分享
用 $\boldsymbol{NPV_{\text{前}}}$ 代表收购或增资前的股权价值，$\boldsymbol{NPV_{\text{后}}}$ 代表收购或增资后的股权价值，<span style="background:#affad1">$\boldsymbol{P}$ 表示收购价或增资额</span>，相关计算如下：  

| 交易场景          | 控股权溢价                                              | 原股东分享的净现值                                               | 新股东分享的净现值                              |     |
| ------------- | -------------------------------------------------- | ------------------------------------------------------- | -------------------------------------- | --- |
| 收购100%股权      | $\boldsymbol{NPV_{\text{后}} - NPV_{\text{前}}}$     | $\boldsymbol{P - NPV_{\text{前}}}$                       | $\boldsymbol{NPV_{\text{后}} - P}$      |     |
| 收购30%股权，能够控制  | $\boldsymbol{NPV_{\text{后}} - NPV_{\text{前}}}$     | $\boldsymbol{P + 70\%×NPV_{\text{后}} - NPV_{\text{前}}}$ | $\boldsymbol{30\%×NPV_{\text{后}} - P}$ |     |
| 增资后持股30%，能够控制 | $\boldsymbol{NPV_{\text{后}} - NPV_{\text{前}} - P}$ | $\boldsymbol{70\%×NPV_{\text{后}} - NPV_{\text{前}}}$     | $\boldsymbol{30\%×NPV_{\text{后}} - P}$ |     |


**说明**：核心是利用 $\boldsymbol{NPV_{\text{前}}}$、$\boldsymbol{NPV_{\text{后}}}$、$\boldsymbol{P}$ 三个变量，区分不同交易场景（收购全股权、部分控股权、增资控股 ），计算控股权溢价及新旧股东净现值分配，体现股权交易中价值变动与收益分割逻辑。

# 考点3 重要 现金流量折现模型

| --------------- | -------------- | ------------------------------------------------------------------------------------- | --- |
| 实体价值            | 公式             | 实体价值 = $\boldsymbol{\dfrac{下一期的实体现金流}{加权平均资本成本 - g}}$                                 |     |
|                 | 实体现金流          | 仅默写核心公式即可，其他公式见第二章“管理用现金流量表”<br>实体现金流 = $\boldsymbol{\underline{税后经营净利润 - 净经营资产的增加}}$ |     |

# 考点4 重要 相对价值评估法（市盈率、市净率、市销率）
| 评估方法         | 市盈率估价（盈利驱动）                                                                                                                           | 市净率估价（资产驱动）                                                                                                                                                               | 市销率估价（收入驱动）                                                                                                                                                                |     |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| **定义公式**     | 本期市盈率 = $\boldsymbol{\dfrac{每股市价}{每股收益}}$                                                                                             | 本期市净率 = $\boldsymbol{\dfrac{每股市价}{每股净资产}}$                                                                                                                                | 本期市销率 = $\boldsymbol{\dfrac{每股市价}{每股营业收入}}$                                                                                                                                |     |
| 公式（驱动因素） | 本期市盈率 = $\boldsymbol{\dfrac{股利支付率×(1 + 增长率)}{股权资本成本 - 增长率}}$                                                                          | 本期市净率 = $\boldsymbol{本期市盈率×本期权益净利率}$                                                                                                                                      | 本期市销率 = $\boldsymbol{本期市盈率×本期营业净利率}$                                                                                                                                       |     |
| 内在（预期）市盈率  | 内在市盈率 = $\boldsymbol{\dfrac{股利支付率}{股权资本成本 - 增长率}}$                                                                                    | 内在市净率 = $\boldsymbol{内在市盈率×预期权益净利率}$                                                                                                                                      | 内在市销率 = $\boldsymbol{内在市盈率×预期营业净利率}$                                                                                                                                       |     |
| 驱动因素     | 包括增长潜力、股利支付率、风险；<br>关键驱动因素：$\boldsymbol{增长潜力}$                                                                                        | 包括权益净利率、股利支付率、增长潜力、风险；<br>关键驱动因素：$\boldsymbol{权益净利率}$                                                                                                                     | 包括营业净利率、股利支付率、增长潜力、风险；<br>关键驱动因素：$\boldsymbol{营业净利率}$                                                                                                                      |     |
| 基本模型     | 目标企业股价 = 可比企业市盈率 × 目标企业每股收益                                                                                                           | 目标企业股价 = 可比企业市净率 × 目标企业每股净资产                                                                                                                                              | 目标企业股价 = 可比企业市销率 × 目标企业每股营业收入                                                                                                                                              |     |
| 修正模型     | 修正市盈率 = $\boldsymbol{\dfrac{可比企业市盈率}{可比企业增长率×100}}$ <br>目标企业股价 = 修正市盈率 × 目标企业增长率 × 100 × 目标企业每股收益                                     | 修正市净率 = $\boldsymbol{\dfrac{可比企业市净率}{可比企业权益净利率×100}}$ <br>目标企业股价 = 修正市净率 × 目标企业权益净利率 × 100 × 目标企业每股净资产                                                                    | 修正市销率 = $\boldsymbol{\dfrac{可比企业市销率}{可比企业营业净利率×100}}$ <br>目标企业股价 = 修正市销率 × 目标企业营业净利率 × 100 × 目标企业每股营业收入                                                                    |     |
| 修正多个因素   | 调整增长率、股利支付率的影响：<br>$\boldsymbol{调整系数 = \dfrac{目标企业增长率}{可比企业增长率}×\dfrac{目标企业股利支付率}{可比企业股利支付率}}$ <br>目标企业股价 = 可比企业市盈率 × 调整系数 × 目标企业每股收益 | 调整增长率、权益净利率、股利支付率的影响：<br>$\boldsymbol{调整系数 = \dfrac{目标企业增长率}{可比企业增长率}×\dfrac{目标企业权益净利率}{可比企业权益净利率}×\dfrac{目标企业股利支付率}{可比企业股利支付率}}$ <br>目标企业股价 = 可比企业市净率 × 调整系数 × 目标企业每股净资产 | 调整增长率、营业净利率、股利支付率的影响：<br>$\boldsymbol{调整系数 = \dfrac{目标企业增长率}{可比企业增长率}×\dfrac{目标企业营业净利率}{可比企业营业净利率}×\dfrac{目标企业股利支付率}{可比企业股利支付率}}$ <br>目标企业股价 = 可比企业市销率 × 调整系数 × 目标企业每股营业收入 |     |
| 取平均值的方法  | 修正平均市盈率法：需平均 $\boldsymbol{可比企业市盈率、可比企业增长率}$ <br>股价平均法：需平均 $\boldsymbol{目标企业股价}$                                                       | 修正平均市净率法：需平均 $\boldsymbol{可比企业市净率、可比企业权益净利率}$ <br>股价平均法：需平均 $\boldsymbol{目标企业股价}$                                                                                         | 修正平均市销率法：需平均 $\boldsymbol{可比企业市销率、可比企业营业净利率}$ <br>股价平均法：需平均 $\boldsymbol{目标企业股价}$                                                                                          |     |
| 缺点       | 若企业 $\boldsymbol{收益是0或负值}$，市盈率无意义                                                                                                     | ① 会计政策差异影响可比性；<br>② 固定资产少的服务性/高科技企业，市净率无实际意义；<br>③ 若 $\boldsymbol{净资产是0或负值}$，市净率无意义                                                                                       | 不能反映 $\boldsymbol{成本}$ 变化对现金流的影响                                                                                                                                           |     |
| 适用       | $\boldsymbol{连续盈利的企业}$                                                                                                                | $\boldsymbol{拥有大量资产、净资产为正的企业}$                                                                                                                                            | $\boldsymbol{销售成本率较低的服务类企业/销售成本率趋同的传统行业企业}$                                                                                                                                |     |

**核心逻辑**：相对价值法通过可比企业 “市场乘数（市盈率/市净率/市销率 ）” 评估目标企业价值，需匹配驱动因素（盈利、资产、收入 ），修正模型消除可比企业与目标企业的差异，最终根据企业特征（盈利、资产、成本结构 ）选择适用方法 。
