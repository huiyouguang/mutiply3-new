# 基本生产费用分配（含挖空）
## 已知条件（企业仅甲、乙两种产品，定额工时为分配标准 ）
|项目|甲产品|乙产品|
| ---- | ---- | ---- |
|年预计制造费用| - | - |（合计为 $\boldsymbol{A}$ ）|
|本月实际制造费用| - | - |（合计为 $\boldsymbol{B}$ ）|
|年计划产量| $\boldsymbol{M_{甲}}$ | $\boldsymbol{M_{乙}}$ |
|月实际产量| $\boldsymbol{N_{甲}}$ | $\boldsymbol{N_{乙}}$ |
|单位产品定额工时| $\boldsymbol{X_{甲}}$ | $\boldsymbol{X_{乙}}$ | 

## 制造费用分配公式（实际分配率法、预计分配率法 ）
### 实际分配率法
- 甲产品分配的制造费用：$\boldsymbol{\frac{B}{N_{甲}X_{甲} + N_{乙}X_{乙}}×N_{甲}X_{甲}}$  
- 乙产品分配的制造费用：$\boldsymbol{\frac{B}{N_{甲}X_{甲} + N_{乙}X_{乙}}×N_{乙}X_{乙}}$  

### 预计分配率法
- 甲产品分配的制造费用：$\boldsymbol{\frac{A}{M_{甲}X_{甲} + M_{乙}X_{乙}}×N_{甲}X_{甲}}$  
- 乙产品分配的制造费用：$\boldsymbol{\frac{A}{M_{甲}X_{甲} + M_{乙}X_{乙}}×N_{乙}X_{乙}}$

## 约当产量法（加权平均 ）
### 分配率及成本计算（以主要材料、辅助材料、人工费用为例 ）
#### 主要材料（生产开始时一次性投入 ）
- 分配率：$\boldsymbol{\frac{A_0 + A_1}{S + T}}$  
- 月末完工产品成本：$\boldsymbol{\frac{A_0 + A_1}{S + T}×S}$  
- 月末在产品成本：$\boldsymbol{\frac{A_0 + A_1}{S + T}×T}$  

#### 辅助材料（陆续均衡投入 ）
- 分配率：$\boldsymbol{\frac{B_0 + B_1}{S + T×t\%}}$  
- 月末完工产品成本：$\boldsymbol{\frac{B_0 + B_1}{S + T×t\%}×S}$  
- 月末在产品成本：$\boldsymbol{\frac{B_0 + B_1}{S + T×t\%}×T×t\%}$  

#### 人工费用（陆续均衡发生 ）
- 分配率：$\boldsymbol{\frac{C_0 + C_1}{S + T×t\%}}$  
- 月末完工产品成本：$\boldsymbol{\frac{C_0 + C_1}{S + T×t\%}×S}$  
- 月末在产品成本：$\boldsymbol{\frac{C_0 + C_1}{S + T×t\%}×T×t\%}$  


## 约当产量法（先进先出 ）
### 分配率及成本计算（以主要材料、辅助材料、人工费用为例 ）
#### 主要材料（生产开始时一次性投入 ）
- 分配率：$\boldsymbol{\frac{A_1}{S - Q + T}}$  
- 月末完工产品成本（方法1 ）：$\boldsymbol{A_0 + A_1 - \text{主要材料分配率}×T}$  
- 月末完工产品成本（方法2 ）：$\boldsymbol{A_0 + \text{主要材料分配率}×(S - Q)}$  
- 月末在产品成本：$\boldsymbol{\text{主要材料分配率}×T}$  

#### 辅助材料（陆续均衡投入 ）
- 分配率：$\boldsymbol{\frac{B_1}{Q×(1 - q\%) + (S - Q) + T×t\%}}$  
- 月末完工产品成本（方法1 ）：$\boldsymbol{B_0 + B_1 - \text{辅助材料分配率}×T×t\%}$  
- 月末完工产品成本（方法2 ）：$\boldsymbol{B_0 + \text{辅助材料分配率}×[Q×(1 - q\%) + (S - Q)]}$  
- 月末在产品成本：$\boldsymbol{\text{辅助材料分配率}×T×t\%}$  

#### 人工费用（陆续均衡发生 ）
- 分配率：$\boldsymbol{\frac{C_1}{Q×(1 - q\%) + (S - Q) + T×t\%}}$  
- 月末完工产品成本（方法1 ）：$\boldsymbol{C_0 + C_1 - \text{人工费用分配率}×T×t\%}$  
- 月末完工产品成本（方法2 ）：$\boldsymbol{C_0 + \text{人工费用分配率}×[Q×(1 - q\%) + (S - Q)]}$  
- 月末在产品成本：$\boldsymbol{\text{人工费用分配率}×T×t\%}$

## 成本分配方法公式
### 实物数量法
- 联产品甲分配成本：$\boldsymbol{\frac{R}{B_{甲} + B_{乙}}×B_{甲}}$  
- 联产品乙分配成本：$\boldsymbol{\frac{R}{B_{甲} + B_{乙}}×B_{乙}}$

- 联产品甲分配成本：$\boldsymbol{\frac{R}{B_{甲}Z_{甲} + B_{乙}Z_{乙}}×B_{甲}Z_{甲}}$  
- 联产品乙分配成本：$\boldsymbol{\frac{R}{B_{甲}Z_{甲} + B_{乙}Z_{乙}}×B_{乙}Z_{乙}}$  

### 可变现净值法
#### 分配的联产品成本
- 联产品甲：$\boldsymbol{\frac{R}{B_{甲}(Y_{甲}-X_{甲}) + B_{乙}(Y_{乙}-X_{乙})}×B_{甲}(Y_{甲}-X_{甲})}$  
- 联产品乙：$\boldsymbol{\frac{R}{B_{甲}(Y_{甲}-X_{甲}) + B_{乙}(Y_{乙}-X_{乙})}×B_{乙}(Y_{乙}-X_{乙})}$  

#### 产品总成本
