# 负债筹资对比记忆
## 1 银行借款 vs 发行债券
| 对比项  | 银行借款     | 发行债券     |
| ---- | -------- | -------- |
| 资本成本 | <u>低</u> | <u>高</u> |
| 速度   | <u>快</u> | <u>慢</u> |
| 弹性   | <u>大</u> | <u>小</u> |
| 范围   | <u>小</u> | <u>大</u> |
| 规模   | <u>小</u> | <u>大</u> |
口诀：借债成本低快弹，范围规模小；发债成本高慢弹，范围规模大

# 保护性条款连线记忆
## 1 一般性保护条款
- 限制高管薪金  
- 限制股权再融资  
- 限制其他长期债务  
- 限制支付现金股利  
- 限制租赁固定资产的规模  
- 对流动资金保持量的规定  
- 定期向银行提交财务报表  
- 不准在正常情况下出售较多资产  
- 如期缴纳税费和清偿其他到期债务  
- 限制净经营性长期资产总投资的规模  
- 不准以任何资产作为其他承诺的担保或抵押  
口诀：一般保护多限制，薪金、股权、债务、股利、租赁、资金量，报表、资产、税费、投资、担保抵押齐 

## 2 特殊性保护条款
- 专款专用  
- 不准出售应收账款  
- 不准投资于短期内不能收回资金的项目  
- 要求主要领导人在合同有效期间担任领导职务  
口诀：特殊保护有专项，专款、应收、短投、领导职务管 

## 3 不属于保护性条款
无对应口诀，单独区分即可 ，即上述未归类到“一般性”“特殊性”的，就是“不属于保护性条款” ，实际按连线对应，这里无额外内容，记住分类逻辑 。

# 附认股权证债券与可转债筹资对比（含挖空）
## 1 投资人现金流
### 1.1 附认股权证债券
|阶段|现金流内容|
| ---- | ---- |
|期初| - 购买价款 |
|期中| ①行权前： + <u>利息</u><br>②行权时： + 行权价差收入<br>③行权后： + <u>利息</u> |
|期末| + <u>本金</u> | 

### 1.2 可转债
|阶段|现金流内容|
| ---- | ---- |
|期初| - 购买价款 |
|期中| ①行权前： + <u>利息</u><br>②行权时： + <u>底线价值</u><br>③行权后： 无 |
|期末| 无 | 

## 2 提高投资人收益的方法
### 2.1 附认股权证债券
1. 降低发行价格  
2. 提高票面利率  
3. 降低行权价格  
4. 增加每张债券附带的认股权证数量  

### 2.2 可转债
1. 降低发行价格  
2. 提高票面利率  
3. 降低转股价格/提高转换比率  
4. 延长不可赎回期  


## 3 区别
|对比项|附认股权证债券|可转债|
| ---- | ---- | ---- |
|行权时是否增加新的资本| <u>是</u> | <u>否</u> |
|灵活性| 较 <u>差</u> | 较 <u>强</u> |
|主要目的| 发行 <u>债券</u> 而非股票 | 发行 <u>股票</u> 而非债券 |
|承销费用| <u>高</u> | <u>低</u> | 

## 4 计算
### 4.1 附认股权证债券
- 纯债券价值 = 利息和本金 按照 <u>等风险普通债券的市场利率</u> 折现  
- 每张认股权证价值 = （附认股权证债券的价格 - 纯债券价值）÷ 一张债券附带的认股权证数量  
- 每张债券的行权价差收入 = （行权后股价 - 行权价）× 一张债券附带的认股权证数  

### 4.2 可转债
- 纯债券价值 = 利息和本金 按照 <u>等风险普通债券的市场利率</u> 折现  
- 转换价值 = 转股后股价 × 转换比例  
- 底线价值 = Max（纯债券价值，转换价值）  


## 5 合理的资本成本范围
附认股权证债券：介于 <u>等风险债务的市场利率</u> 和 <u>普通股的税前资本成本</u> 之间  


## 6 其他（与看涨期权的区别等）
### 6.1 附认股权证债券
|对比项|内容|
| ---- | ---- |
|看涨期权来源（行权时）| 看涨期权来源于 <u>二级市场</u> ，认股权证来源于 <u>新发股票/一级市场</u> |
|交割标的结算| 看涨期权一般以 <u>净额</u> 结算，认股权证一般以 <u>股票/实物</u> 结算 |
|稀释性| 看涨期权是否稀释股权收益 <u>否</u> ，认股权证是否稀释每股收益 <u>是</u> |
|期限| 看涨期权期限 <u>短</u> ，认股权证期限 <u>长</u> | 

### 6.2 可转债
|条款|对谁有利|
| ---- | ---- |
|赎回条款| 对 <u>发行企业</u> 有利 |
|不可赎回期| 对 <u>债券持有人</u> 有利 |
|回售条款| 对 <u>债券持有人</u> 有利 |
|转换条款| 对 <u>债券持有人</u> 有利 |
|强制转换条款| 对 <u>发行企业</u> 有利 |

# 租赁相关概念（含挖空）
## 1 租赁存在原因
|类别|原因|
| ---- | ---- |
|长期租赁| <u>节税</u> 是主要原因 |
|短期租赁| <u>降低交易成本</u> 是主要原因 |
|其他| 还可 <u>减少不确定性</u> | 

## 2 租赁费用经济内容
包括4部分，分别是 <u>购置成本</u> 、 <u>营业成本</u> 、 <u>相关的利息</u> 、 <u>利润</u>  

## 3 租赁分类（按不同标准）
### 3.1 按当事人关系
|类型|定义|
| ---- | ---- |
|直接租赁| 出租人直接向承租人提供租赁资产 |
|杠杆租赁| 出租人只支付购买款的一部分，其余款项以资产或出租权等为抵押，向另外的贷款者借入 |
|售后租回| 承租人先将资产卖给出租人，再将该资产租回 | 

### 3.2 按租赁期
|类型|定义|
| ---- | ---- |
|短期租赁| 租赁时间明显少于租赁资产的经济寿命 |
|长期租赁| 租赁时间接近租赁资产的经济寿命 | 

### 3.3 按全部租赁投资金额
|类型|定义|
| ---- | ---- |
|不完全补偿租赁| 租赁费不足以补偿租赁资产的全部成本 |
|完全补偿租赁| 租赁费≥资产全部成本 | 

### 3.4 按是否随时解除
|类型|定义|
| ---- | ---- |
|可以撤销租赁| 合同注明 <u>承租人</u> 可随时解除 |
|不可撤销租赁| 合同到期前不能单方面解除 | 

### 3.5 按谁提供资产维修
| 类型  | 负责维修方             |
| --- | ----------------- |
| 毛租赁 | <u>出租人</u> 负责资产维修 |
| 净租赁 | <u>承租人</u> 负责资产维修 |
