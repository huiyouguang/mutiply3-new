# 债务资本成本计算方法梳理表
| 类别                 | 适用情形                       | 计算逻辑与公式                                                                                                                                                                                                                                                                                                                                                                                        | 关键说明                                                                    |
| ------------------ | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **到期收益率法**（税前债务成本） | 公司目前有上市的长期债券               | 1. 若债券按面值发行（每年付息、不考虑交易费用 ）<br>设税前债务成本为$w_d$，公式：$P_0 = 1000×(P/A, w_d, n) + 1000×(P/F, w_d, n)$（$P_0$为债券当前价格，$n$为剩余付息期数 ）<br>2. 若债券为平息债券（每年付息、考虑交易费用 ）<br>公式：$P_d - f = 1000×r×(P/A, y, n) + 1000×(P/F, y, n)$（$P_d$为发行价，$f$为发行费用，$y$为用插值法求解的成本 ）<br>3. 若债券为贴现债券（每年付息、考虑交易费用 ）<br>设每期折现率为$z$，公式：$P_d - f = 1000×\frac{r}{m}×(P/A, z, mn) + 1000×(P/F, z, mn)$（$m$为付息次数 ），再换算年成本：$(1 + z)^m - 1$ | 通过债券现金流（利息、本金偿还），用折现思想，令债券当前价值（考虑发行费用时为发行价减费用）等于未来现金流现值，用插值法求解折现率，即债务成本 |
| **可比公司法**（税前债务成本）  | 公司目前没有上市债券，但能找到一个债务上市的可比公司 | Step1：选可比公司，需满足：<br> -<span class="cloze-span"> 可制作交易债</span>券<br> - <span class="cloze-span">与本公司处于同一行业</span><br> - <span class="cloze-span">具有类似商业模式</span><br> - <span class="cloze-span">规模、负债率和财务状况类似</span><br>Step2：据可比公司长期债券的到期收益率，作为本公司长期债务成本                                                                                                                                        | 找经营、财务特征匹配的可比公司，借鉴其上市债券的到期收益率，简化自身债务成本估算                                |
| **风险调整法**（税前债务成本）  | 没有上市债券也没有可比公司，但有信用评级资料     | 本公司税前债务成本 = <span class="cloze-span">无风险收益率 + 平均信用风险补偿率</span><br>（信用风险补偿率：用上市的与本公司信用级别相同的公司债券的到期收益率，减去与这些上市公司债券同期的长期政府债券到期收益率，取平均 ）                                                                                                                                                                                                                                                           | 利用信用评级关联风险，通过市场中同类信用级别债券与无风险债券的收益差，估算自身债务成本                             |
| **财务比率法**（税前债务成本）  | 没有上市债券、可比公司、信用评级资料         | 需知道目标公司关键财务比率，凭此大体判断信用级别，再用“风险调整法”确定债务成本                                                                                                                                                                                                                                                                                                                                                       | 通过财务数据间接评估信用风险，进而估算成本，较粗略                                               |
| **税后债务资本成本**       | -                          | 税后债务成本 =<span class="cloze-span"> 税前债务成本×（1 - 所得税税率）</span>                                                                                                                                                                                                                                                                                                                                    | 因债务利息可抵税，投资者要求收益（即公司债务成本）需考虑税盾效应，税后成本更低                                 |

# 普通股资本成本计算方法梳理
## 一、资本资产定价模型（CAPM）
### （一）公式  
$R_s = R_f + \beta×(R_m - R_f)$  

### （二）参数说明  
| 参数                  | 释义与取值规则                                                                                                                                                                          |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $R_f$（无风险利率）        | 期限：选**长期政府债券**利率，通常用10年期；<br>利率类型：用**到期收益率**（而非票面利率 ）                                                                                                                            |
| $\beta$（贝塔系数）       | 期限：公司风险无重大变化时，可用5年或更长数据；风险变化大时，用**变化后年份**数据；<br>收益计量间隔：选**每周或每月**数据；<br>使用历史$\beta$条件：**经营风险、财务风险均无显著改变**；<br>计算公式：$\beta_{i,r_m} = \frac{Cov(R_i,R_m)}{\sigma_m^2}$（协方差与市场方差比 ） |
| $R_m - R_f$（市场风险溢价） | 期限：用**长期**数据，覆盖经济繁荣、衰退周期；<br>平均值计算：用**几何平均**，公式 $\bar{g} = \sqrt[n]{\frac{P_n}{P_0}} - 1$ 或 $P_n = P_0×(1 + g)^n$                                                                |


## 二、股利增长模型  
### （一）基础公式  
$R_s = \frac{D_1}{P_0} + g$ ，其中 $D_1 = D_0×(1 + 增长率)$  

### （二）股利增长率（$g$）估算方法  
| 方法      | 说明                                         |
| ------- | ------------------------------------------ |
| 历史增长率   | 依据公司历史股利增长数据推算                             |
| 可持续增长率  | 基于可持续增长假设（$g = $ 销售净利率×总资产周转率×权益乘数×利润留存率 ） |
| 证券分析师预测 | 参考专业机构对公司股利增长的预期                           |


## 三、债券收益率风险调整模型  
### （一）公式  
$r_s = r_{dt} + RP_c$  

### （二）参数说明  
| 参数             | 释义                                          |
| -------------- | ------------------------------------------- |
| $r_{dt}$（债务成本） | 用**税后债务成本**（考虑利息抵税 ），且取**本公司债务成本**（而非市场平均 ） |
| $RP_c$（风险溢价）   | 基于**相对于债券的风险**，用**本公司**风险溢价（对比可比公司或自身历史 ）   |


## 四、考虑发行费的调整（发行新股时）  
公式：$R_s = \frac{D_1}{P_0×(1 - F)} + g$ （$F$ 为发行费率 ）  


**备注**：各模型适用场景不同，需结合公司是否发股利、数据可得性（如$\beta$、风险溢价等）灵活选择，核心是估算投资者要求的必要报酬率，反映普通股资本成本 。

# 其他资本成本
这是**其他资本成本及加权平均资本成本权重相关知识**，实体信息如下：
- **优先股资本成本公式**：$r_p = \frac{D_p}{P_p×(1 - F)}$（$D_p$为优先股股利，$P_p$为发行价格，$F$为发行费用 ） 
- **永续债资本成本公式**：$r_{pd} = \frac{I_{pd}}{P_{pd}×(1 - F)}$（$I_{pd}$为永续债利息，$P_{pd}$为发行价格，$F$为发行费用 ） 
- **加权平均资本成本权重**：
    - 按**账面价值**加权，反映**历史**时期结构 
    - 按**目标资本结构**加权，反映**未来**时期结构 
    - 反映当前资本结构的是按**实际市场价值**加权 
属于财务管理中资本成本计算及权重确定的知识点 。
