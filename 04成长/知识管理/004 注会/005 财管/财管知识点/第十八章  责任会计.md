# 考点1 重要 责任成本（可控成本）梳理
## 一、可控成本判定  
### （一）同时满足条件（3 个“能够” ）  
1. 能够预测  
2. 能够计量  
3. 能够调控  


### （二）满足其一条件（3 种情形 ）  
1. 通过行动有效影响成本 $\boldsymbol{数额}$  
2. 有权决定 $\boldsymbol{是否使用}$ 某种资产/劳务  
3. 虽不直接决定成本，但上级要求参与，对成本支出施加 $\boldsymbol{重要影响}$  


## 二、制造费用的分摊方式连线  
| 制造费用项目     | 分摊方式          |
|------------|---------------|
| 折旧         | 按责任基础分配      |
| 动力费        | 按责任基础分配      |
| 维修费        | 按责任基础分配      |
| 运输费用       | 按受益基础分配      |
| 机物料消耗      | 直接计入责任中心     |
| 试验检验费用     | 归入某特定责任中心    |
| 低值易耗品的领用   | 直接计入责任中心     |
| 按装机功率分配电费  | 按受益基础分配      |
| 分配的公司的管理费用 | 不分配           |  


# 考点2 重要 内部转移价格梳理
## 一、市场型内部转移价格  
### （一）适用中心  
一般适用于 $\boldsymbol{利润}$ 中心  


### （二）情形与价格连线  
| 情形描述               | 对应内部转移价格   |
|---------------------|------------|
| 没有外部市场            | 模拟市场价       |
| 经常外销且外销比例较大      | 外销价格        |
| 管理层认为不需要频繁变动价格   | 生产成本加一定毛利   |
| 一般不对外销且外部市场没有可靠报价 | 模拟市场价       |  


## 二、成本型内部转移价格  
### （一）适用中心  
一般适用于 $\boldsymbol{成本}$ 中心  


## 三、协商型内部转移价格  
### （一）适用企业  
主要适用于 $\boldsymbol{分权程度较高}$ 的企业  


### （二）价格区间  
协商价格上限为 $\boldsymbol{市场价}$，下限为 $\boldsymbol{变动成本}$  


**核心逻辑**：考点 1 围绕可控成本 “判定条件 + 制造费用分摊规则” 构建责任成本框架；考点 2 区分市场型、成本型、协商型内部转移价格，明确适用中心、场景连线及协商区间，体现责任会计中成本管控与内部交易定价逻辑。

# 考点3 重要 成本中心梳理
## 一、责任范围与分类  
### （一）责任范围  
只对其 $\boldsymbol{成本或费用}$ 承担责任的责任中心  


### （二）分类连线（标准成本中心 vs 费用中心 ）  
| 部门/场景       | 对应成本中心类型   |
|-------------|------------|
| 工段          | 标准成本中心     |
| 班组          | 标准成本中心     |
| 餐饮制作间       | 标准成本中心     |
| 会计部门        | 费用中心       |
| 行政部门        | 费用中心       |
| 制造车间        | 标准成本中心     |
| 研发部门        | 费用中心       |
| 广告部门        | 费用中心       |
| 医院放射科       | 费用中心       |
| 投入产出关系明确    | 标准成本中心     |
| 投入产出无密切关系   | 费用中心       |
| 产出物数量可以衡量   | 标准成本中心     |
| 产出物无法用财务指标衡量 | 费用中心       |  


## 二、考核指标  
### （一）标准成本中心  
1. 基本规则：使用 $\boldsymbol{既定产品质量和数量}$ 条件下的 $\boldsymbol{可控标准成本}$ 作为考核指标  
2. 成本差异责任判定（以生产车间为例 ）：  
   - 变动制造费用耗费差异：$\boldsymbol{是}$（可控 ）  
   - 变动制造费用效率差异：$\boldsymbol{是}$（可控 ）  
   - 固定制造费用耗费差异：$\boldsymbol{是}$（可控 ）  
   - 固定制造费用效率差异：$\boldsymbol{是}$（可控 ）  
   - 固定制造费用闲置能力差异：$\boldsymbol{否}$（不可控 ）  
   - 直接材料用量差异：$\boldsymbol{是}$（可控 ）  
   - 直接材料价格差异：$\boldsymbol{否}$（不可控 ）  
   - 直接人工工资率差异：$\boldsymbol{否}$（不可控 ）  
   - 直接人工效率差异：$\boldsymbol{是}$（可控 ）  


### （二）费用中心  
使用 $\boldsymbol{可控费用预算}$ 作为考核指标，同时需评价 $\boldsymbol{工作质量和服务水平}$  


# 考点4 重要 利润中心梳理
## 一、责任范围与分类  
### （一）责任范围  
对其 $\boldsymbol{收入、成本和费用}$ 承担责任的责任中心  


### （二）分类  
| 类型   | 定义/特征                     |
|------|---------------------------|
| 人为的利润中心 | 企业内部按内部转移价格出售产品        |
| 自然的利润中心 | 直接向企业外部出售产品，参与市场购销      |  


## 二、考核指标（计算公式 + 作用 ）  
| 指标名称       | 公式                         | 评价经理业绩？ | 评价对企业利润和管理费用贡献？ |
|------------|----------------------------|----------|-------------------|
| 部门边际贡献     | $\boldsymbol{部门销售收入 - 部门变动成本总额}$ | -        | -                 |
| 部门可控边际贡献   | $\boldsymbol{部门边际贡献 - 部门可控固定成本}$ | $\boldsymbol{能}$ | $\boldsymbol{否}$      |
| 部门税前经营利润   | $\boldsymbol{部门可控边际贡献 - 部门不可控固定成本}$ | $\boldsymbol{否}$ | $\boldsymbol{能}$      |  


**核心逻辑**：考点 3 从成本中心 “责任范围→分类（标准/费用中心 ）→考核指标（可控成本/预算 + 质量服务 ）” 构建管控逻辑；考点 4 围绕利润中心 “责任范围→分类（人为/自然 ）→考核指标（边际贡献分层 ）” 体现盈利责任与业绩评价，区分不同中心的责任边界与考核重点。

# 考点5 重要 投资中心梳理
## 一、责任范围  
对其 $\boldsymbol{收入、成本和费用、资产（投资）}$ 承担责任的责任中心  


## 二、考核指标  
### （一）部门投资报酬率  
1. 公式：$\boldsymbol{\dfrac{部门税前经营利润}{部门平均净经营资产}}$  
2. 特点判定（是/否 ）：  
   - 便于不同部门比较：$\boldsymbol{是}$  
   - 导致部门与企业目标相悖：$\boldsymbol{是}$  
   - 依赖会计数据质量：$\boldsymbol{是}$  
   - 可直接据会计数据计算：$\boldsymbol{是}$  
   - 可分析投资周转率和经营利润率：$\boldsymbol{是}$  
   - 具有一定主观性：$\boldsymbol{否}$  
   - 考虑不同部门资产风险特征：$\boldsymbol{否}$  
   - 可能导致错误决策：$\boldsymbol{是}$  
   - 考虑所得税影响：$\boldsymbol{否}$  
   - 考虑债权人影响：$\boldsymbol{否}$  


### （二）部门剩余收益  
1. 公式：$\boldsymbol{部门税前经营利润 - 部门平均净经营资产×要求的税前投资报酬率}$  
2. 特点判定（是/否 ）：  
   - 便于不同部门比较：$\boldsymbol{否}$  
   - 导致部门与企业目标相悖：$\boldsymbol{否}$  
   - 依赖会计数据质量：$\boldsymbol{是}$  
   - 可直接据会计数据计算：$\boldsymbol{否}$  
   - 可分析投资周转率和经营利润率：$\boldsymbol{否}$  
   - 具有一定主观性：$\boldsymbol{是}$  
   - 考虑不同部门资产风险特征：$\boldsymbol{是}$  
   - 可能导致错误决策：$\boldsymbol{否}$  
   - 考虑所得税影响：$\boldsymbol{否}$  
   - 考虑债权人影响：$\boldsymbol{否}$  


**核心逻辑**：围绕投资中心 “责任范围（收入、成本费用、资产 ）”，对比 “部门投资报酬率” 和 “部门剩余收益” 两类考核指标的公式、特点（是否便于比较、是否 align 企业目标等 ），体现投资中心业绩评价的不同逻辑与应用场景。

# 考点6 重要 三个中心（成本、利润、投资中心）对比梳理
| 对比维度       | 成本中心                | 利润中心                | 投资中心                |
|------------|---------------------|---------------------|---------------------|
| 是否有成本      | 是                   | 是                   | 是                   |
| 是否有收入      | 可能有                 | 是                   | 是                   |
| 是否有采购决策权   | 否                   | 是                   | 是                   |
| 是否有生产决策权   | 否                   | 是                   | 是                   |
| 是否有销售决策权   | 否                   | 是                   | 是                   |
| 是否有投资决策权   | 否                   | 否                   | 是                   |
| 是否有市场选择权   | 否                   | 是                   | 是                   |
| 是否有经营决策权   | 否                   | 是                   | 是                   |
| 是否有可控成本控制权 | 是                   | 是                   | 是                   |
| 是否对收入负责    | 否                   | 是                   | 是                   |
| 是否对利润负责    | 否                   | 是                   | 是                   |
| 是否对资产负责    | 否                   | 否                   | 是                   |
| 是否对折旧负责    | 否                   | 否                   | 是                   |
| 是否对分摊的管理费用负责 | 否                   | 否                   | 否                   |
| 评价指标       | 标准成本中心：既定产品质量和数量下的可控标准成本；<br>费用中心：可控费用预算 + 工作质量和服务水平 | 部门边际贡献；<br>部门可控边际贡献；<br>部门税前经营利润 | 部门投资报酬率；<br>部门剩余收益；<br>补充指标（现金回收率、剩余现金流量 ） |
| 范围判定       | 任何发生成本的责任领域，可确认为成本中心：$\boldsymbol{是}$ | 产生利润的组织可确认为利润中心：$\boldsymbol{否}$；<br>所有企业都有利润中心：$\boldsymbol{否}$ | 利润中心是否属成本中心：$\boldsymbol{否}$；<br>投资中心是否属利润中心：$\boldsymbol{是}$；<br>投资中心是否属成本中心：$\boldsymbol{否}$ |  

**核心逻辑**：从 “权责范围（成本、收入、决策权限等 ）、评价指标、适用范围” 维度，对比成本中心（聚焦成本控制 ）、利润中心（拓展至收入 - 利润责任 ）、投资中心（新增资产投资权责 ）的差异，构建责任会计中不同责任中心的区分框架 。
