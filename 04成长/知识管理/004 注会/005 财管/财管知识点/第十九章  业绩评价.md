# 考点1 重要 经济增加值与剩余收益对比（MD 表格版 ）
| 类别          | 部门剩余收益                     | 基本的经济增加值                  | 披露的经济增加值                  | 简化的经济增加值                  |
|-------------|-----------------------------|----------------------------|----------------------------|----------------------------|
| **公式**      | 部门税前经营利润 - 部门平均净经营资产×要求的税前投资报酬率 | 税后净营业利润 - 报表平均总资产×加权平均税后资本成本 | 调整后税后净营业利润 - 调整后平均资本占用×加权平均税后资本成本 | 调整后税后净营业利润 - 调整后平均资本×加权平均税后资本成本 |
| **数据是否调整** | 否                           | 否                          | 是                          | 是                          |
| **调整项目/方法** | -                           | -                          | ① 研究与开发费用；<br>② 战略性投资；<br>③ 为建立品牌、进入新市场或扩大市场份额发生的费用；<br>④ 折旧费用（沉没资金折旧法 ） | ① 调整后税后净营业利润 = 净利润 +（利息支出 + 研究开发费用调整项 ）×(1 - 25%)；<br>② 调整后平均资本 = 平均所有者权益 + 平均带息负债 - 平均在建工程 |
| **依据信息**    | 内部信息                        | 公开信息                       | 公开信息                       | -                          |
| **资本成本选择**  | 部门各自的资本成本（源于主观估计 ）        | 公司统一的资本成本（源于资本市场 ）      | 公司统一的资本成本（源于资本市场 ）      | 按企业类型：<br>① 充分竞争行业商业企业：6.5%；<br>② 国家使命类商业企业：5.5%；<br>③ 公益类企业：4.5%；<br>④ 特定垄断企业：下浮 0.5%  |
| **是否考虑所得税** | 否                           | 是                          | 是                          | 是                          |
| **其他特点**    | 资本成本主观估计，聚焦部门内部权责         | 基于公开报表，未调整会计扭曲          | 调整会计扭曲，更贴合价值创造          | 简化调整，适配国企等特定考核场景        |  

**核心逻辑**：从 “公式、数据调整、依据信息、资本成本、所得税考虑” 维度，对比部门剩余收益（内部考核，简化逻辑 ）与经济增加值（分基本、披露、简化类型，逐步调整会计扭曲，贴合价值创造 ），体现业绩评价指标的精准度差异与应用场景。

### 考点2 财务业绩评价 vs 非财务业绩评价梳理
| 特征                | 财务业绩评价       | 非财务业绩评价      |
|-------------------|-------------|-------------|
| 视角倾向              | 短视           | 体现长远业绩       |
| 导向类型              | 结果导向         | -           |
| 主观性                | 带有主观性（非财务 ） | -           |
| 数据获取难度            | 数据可靠（财务 ）    | 数据不易取得      |
| 业绩反映周期            | 反映短期业绩       | 体现长远业绩       |
| 会计准则影响            | 受会计准则影响      | -           |
| 经营成果反映            | 反映综合经营成果     | -           |
| 数据可靠性              | 数据可靠（财务 ）    | 数据可靠性难以保证   |
| 连线结果              | 短视、结果导向、反映短期业绩、受会计准则影响、反映综合经营成果、数据可靠 → 财务业绩评价<br>带有主观性、数据不易取得、体现长远业绩、数据可靠性难以保证 → 非财务业绩评价 |

### 考点3 关键绩效指标法梳理
| 层级   | 制定依据                 | 说明          |
|------|----------------------|-------------|
| 企业级  | 战略目标                | 基于企业整体战略定指标 |
| 单位（部门）级 | 企业级关键绩效指标          | 承接企业指标分解   |
| 岗位（员工）级 | 单位（部门）级关键绩效指标     | 落实到岗位个人    |
| 分类   | 结果类（反映企业绩效价值指标 ）、动因类（反映价值关键驱动因素 ） |
| 数量   | 每层一般不超过 10 个        | 控制指标数量，聚焦重点 |

### 平衡计分卡维度连线梳理
| 左侧指标/问题                | 对应平衡计分卡维度   |
|-------------------------|-------------|
| 市场份额、客户保持率、客户满意度、客户获利率、客户获得率、解决顾客如何看待我们 | 顾客维度        |
| 经济增加值、投资报酬率、解决股东如何看待我们        | 财务维度        |
| 交货及时率、生产负荷率、产品合格率、解决我们的优势是什么      | 内部业务流程维度   |
| 员工保持率、员工满意度、员工生产率、信息资本准备度、新产品开发周期、培训计划完成率、解决我们是否能继续提高并创造价值 | 学习与成长维度    | 

**说明**：平衡计分卡通过财务、顾客、内部业务流程、学习与成长四个维度，分别对应股东、客户、运营、员工发展视角，构建全面业绩评价体系。

### 绩效棱柱模型连线梳理
| 左侧内容                | 对应右侧要素   |
|---------------------|----------|
| 节能减排达标率（监管机构）、资本保值增值率（监管机构）、环保投入率（监管机构）、税收优惠程度（监管机构） | 利益相关者满意  |
| 采购折扣利率水平（供应商）、供应商关系质量（供应商）、逾期付款次数（供应商）、按时交货率（供应商） | 利益相关者满意  |
| 融资成本率（投资者）、流动比率（投资者）、总资产周转率（投资者）、净资产收益率（投资者） | 利益相关者满意  |
| 客户忠诚度（客户）、客户投诉率（客户）、准时交货率（客户）、品牌意识（客户） | 利益相关者满意  |
| 员工保持率（员工）、员工满意度（员工）、员工职业规划（员工）、培训费用支出率（员工） | 利益相关者满意  |
| 绩效棱柱模型的出发点、绩效棱柱模型的落脚点、目标是解决用什么战略来满足利益相关者需求、目标是解决需要哪些关键业务流程才能执行战略 | 企业战略    |
| -                   | 业务流程    |
| -                   | 组织能力    |
| -                   | 利益相关者贡献 | 

**说明**：绩效棱柱模型围绕 “利益相关者满意 - 企业战略 - 业务流程 - 组织能力 - 利益相关者贡献” 逻辑，通过多维度指标关联，构建从满足利益相关者需求到获取其贡献的闭环管理体系 。 实际完整连线需结合模型精准对应，这里先按已有关联梳理核心匹配，后续可细化补充业务流程、组织能力等维度与具体指标的连接 ，比如业务流程维度可关联按时交货率等体现流程效率的指标 ，组织能力维度关联员工培训、职业规划等反映组织支撑力的内容  。
