# 投资项目评价方法梳理表
| 方法分类                  | 具体方法        | 计算逻辑/步骤                                                                         | 决策规则                 | 特点/优缺点                                                                          | 特殊说明                                                                                                                     |
| --------------------- | ----------- | ------------------------------------------------------------------------------- | -------------------- | ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **折现现金流量法**（考虑时间价值）   | 净现值法（NPV）   | $NPV =$ 未来现金净流量现值 $-$ 原始投资额现值<br>= 现金流入总现值 $-$ 现金流出总现值                          | $NPV > 0$ 时，值得投资     | 特点：<br>- 绝对数指标，反映投资的**效益**；<br>- 对比期限、投资额不同项目有局限性                               | -                                                                                                                        |
|                       | 现值指数法（PI）   | $PI = \frac{未来现金净流量现值}{原始投资额现值}$                                                | $PI > 1$ 时，值得投资      | 特点：<br>- 相对数指标，反映投资的**效率**；<br>- 消除了投资额差异，仍未消除**项目期限**差异                        | -                                                                                                                        |
|                       | 内含报酬率法（IRR） | 用内插法求解：使项目未来现金净流量现值 = 原始投资额现值时的折现率                                              | $IRR >$ 项目资本成本时，值得投资 | 特点：<br>- 相对数指标，反映投资方案本身的报酬率；<br>- 可能出现**多解或无解**情况，导致结论无效                        | 修正内含报酬率：<br>Step1：以**项目资本成本**为折现率，分算现金流入、流出的现值（流入现值 + 流出终值 ）<br>Step2：用内插法计算该终值与现值的内含报酬率，克服 IRR 缺陷（假设流入再投资报酬率 = 项目资本成本 ） |
| **非折现现金流量法**（不考虑时间价值） | 静态回收期法      | 每年现金流入按原始额累加，到与投资额相等时所需时间（最后一年用 $\frac{尚未回收额}{最后一年现金净流量}$ 算分数 ）                 | 收回原始投资额时的时间越短越好      | 缺点：<br>- 忽视**时间价值**；<br>- 未考虑**回收期后现金流量**，无法衡量盈利性；<br>- 使公司偏爱**短期**项目，放弃长期有意义项目 | -                                                                                                                        |
|                       | 动态回收期法      | 每年现金流入按现值累加，到与投资额相等时所需时间（最后一年用 $\frac{尚未回收额}{最后一年现金净流量现值}$ 算分数 ）                | 收回原始投资额现值时的时间越短越好    | 较静态法考虑时间价值，但仍未解决回收期后现金流问题                                                       | -                                                                                                                        |
| **其他方法**              | 会计报酬率法      | 方法一：$\frac{寿命期内平均税后经营净利润}{原始投资额}$<br>方法二：$\frac{寿命期内平均税后经营净利润}{(原始投资额 + 残值)/2}$ | 会计报酬率越高越好            | 缺点：<br>- 忽视**折旧**对现金流的影响；<br>- 忽视**税后经营净利润的时间分布**对项目价值的影响                       | -                                                                                                                        |

# 可比公司法估计投资项目资本成本

### 一、用企业当前加权平均资本成本作为项目资本成本
- **条件**：  
    - 项目的 **经营风险** 与企业当前资产的 **平均经营风险** 相同  
    - 继续采用相同的 **资本结构** 为新项目筹资  
- **计算**：  
加权平均资本成本 = 净负债比重×税后债务资本成本 + 权益比重×权益资本成本  

### 二、用可比公司法估计投资项目资本成本
- **条件**：  
新项目的 **风险** 与现有资产的 **平均风险** 显著不同  
- **计算步骤**：  
1. 卸载可比公司财务杠杆：  
$\beta_{资产} = \frac{可比公司\beta_{权益}}{1 + \frac{可比公司净负债}{可比公司权益}×(1 - T)}$  
2. 加载目标项目财务杠杆：  
目标项目$\beta_{权益} = \beta_{资产}×\left[1 + \frac{目标项目净负债}{目标项目权益}×(1 - T)\right]$  
3. 算项目权益成本：  
项目权益成本 = $R_f +$ 目标项目$\beta_{权益}×(R_m - R_f)$（$R_f$为无风险利率，$R_m$为市场平均收益率 ）  
4. 算项目折现率：  
项目折现率 = 税后负债成本×净负债比重 + 权益成本×权益比重  


属于财务成本管理中“项目资本预算”模块，用于确定投资项目合适的折现率，辅助项目价值评估（如净现值计算 ）。

#  **设备更新决策**
现金流量梳理，按“无税三阶段”和“有税三阶段”，区分“使用旧设备”“购置新设备”在**期初、期中、期末**的现金流，实体信息整理：

| 阶段     | 场景    | 无税现金流     | 有税现金流                       |
| ------ | ----- | --------- | --------------------------- |
| **期初** | 使用旧设备 | - 旧设备变现价值 | - 旧设备变现价值 -（账面价值 - 变现价值）×税率 |
|        | 购置新设备 | - 新设备购置价格 | - 新设备购置价格                   |
| **期中** | 使用旧设备 | - 旧设备付现成本 | - 旧设备税后付现成本 + 旧设备折旧抵税       |
|        | 购置新设备 | - 新设备付现成本 | - 新设备税后付现成本 + 新设备折旧抵税       |
| **期末** | 使用旧设备 | + 旧设备残值变现 | + 旧设备残值变现 +（计税价值 - 变现价值）×税率 |
|        | 购置新设备 | + 新设备残值变现 | + 新设备残值变现 +（计税价值 - 变现价值）×税率 |

用于财务成本管理中设备更新决策的现金流分析，区分有无税环境下，不同阶段（购置、运营、处置 ）的现金流入/流出，辅助决策是否更新设备 。

# 敏感性分析


这是**投资项目敏感 敏感分析**知识点，涵盖最大最小法、敏感程度法的计算与局限性，实体信息整理：  

### 一、计算方法  
|方法|步骤|公式/关键逻辑|
| ---- | ---- | ---- |
|**最大最小法**|1. 算正常净现值（基准净现值）<br>2. 令**净现值 = 0**，其他变量不变，算选定变量临界值|通过找净现值为0时变量临界值，判断项目对变量的承受极限|
|**敏感程度法**|1. 算正常净现值（基准净现值）<br>2. 假设单一变量变、其他不变，重算净现值<br>3. 敏感系数 = $\frac{净现值变动百分比}{该变量变动百分比}$<br> - 系数为负→**变量与净现值反向变动**<br> - **敏感系数绝对值**越大→对净现值越敏感|通过系数衡量变量对净现值的影响程度|  


### 二、局限性  
1. 只允许 **一个变量变动**，假设 **其他变量不变**（现实中多变量可能联动 ）  
2. 没给出 **变量每个取值发生的可能性（概率）**（无法判断风险实际发生概率 ）  

用于投资项目评估，分析关键变量对净现值的影响，辅助判断项目风险与不确定性 。
