# 考点1 利率
| 类别               | 内容                                                                        |
| ---------------- | ------------------------------------------------------------------------- |
| **纯粹利率**         | 没有通货膨胀时，<u>短期政府债券</u>的利率可以视作纯粹利率。                                         |
| **无风险利率**（通胀补偿）  | 溢价：纯粹利率与<u>通货膨胀溢价</u>之和称为名义无风险利率。                                         |
| **风险溢价 - 违约风险**  | 溢价：<span class="cloze-span">债券因存在发行者到期时不能按约定足额支付本金或利息的风险而给予债权人的补偿。</span> |
| **风险溢价 - 流动性风险** | 溢价：<span class="cloze-span">债券因存在不能短期内以合理价格变现的风险而给予债权人的补偿。</span>         |
| **风险溢价 - 期限风险**  | 溢价：债<span class="cloze-span">券因面临存续期内市场利率上升导致价格下跌的风险而给予债权人的补偿。</span>     |

# 内插法求解利率\(i\)步骤梳理
| 步骤  | 具体内容                                                                                                                                                        |     |
| --- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| 第一步 | 根据**现金流出的现值 = 现金流入的现值**，列出等式                                                                                                                                |     |
| 第二步 | 令\(i = i_1\)，带入等式，得到\(w_1\)；令\(i = i_2\)，带入等式，得到\(w_2\)。<br>\(w_1\)和\(w_2\)需满足：等式中的\(w\)介于\(w_1\)和\(w_2\)之间 ；\(i_1\)和\(i_2\)需满足：\(i_1\)和\(i_2\)中间仅相差\(1\%\) |     |
| 第三步 | 根据等式\(\boldsymbol{\frac{i - i_1}{i_2 - i_1} = \frac{w - w_1}{w_2 - w_1}}\)，求解\(i\)                                                                          |     |

# 三种利率（报价利率、计息期利率、有效年利率）知识点梳理
| 类别                              | 内容                                                                                                                                                                   |     |
| ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| **关系式**（报价利率为\(r\)，一年计息\(m\)次 ） | 计息期利率\(=\boldsymbol{\underline{r÷m}}\)；有效年利率\(=\boldsymbol{\underline{(1 + r÷m)^m - 1}}\)                                                                            |     |
| **联系** - 每年计息一次时                | 有效年利率 \(\boldsymbol{\underline{=}}\) 报价利率                                                                                                                            |     |
| **联系** - 每年计息多次时                | 有效年利率 \(\boldsymbol{\underline{＞}}\) 报价利率                                                                                                                            |     |
| **联系** - 多年计息一次时                | 有效年利率 \(\boldsymbol{\underline{＜}}\) 报价利率                                                                                                                            |     |
| **联系** - 每年计息次数影响               | 每年计息次数越多，有效年利率越 \(\boldsymbol{\underline{大}}\)                                                                                                                       |     |
| **区别 - 归属**（连线题对应 ）             | 报价利率：名义利率、票面利率、年折现率、月折现率、金融机构提供的利率、一年计息多次时计算现值的折现率、一年计息多次时计息期利率×一年的计息次数<br>计息期利率：无对应单独名称（为报价利率÷计息次数 ）<br>有效年利率：资本成本、等价年利率、到期收益率、期望报酬率、必要报酬率、内含报酬率、比较各方案投资回报率时需要使用的利率 |     |
| **连续复利**                        | 复利次数无穷大时的有效年利率\(=\boldsymbol{\underline{e^{报价利率} - 1}}\)（\(e\)为自然常数 ）                                                                                                |     |

# 风险与报酬的计量核心公式梳理


|      | 风险（基础）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | 风险（样本/变异系数）                                                                                                                                                                                                                                                             |
| ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 单项资产 | **期望报酬率**<br>已测历史数据：$\bar{K}=\frac{\sum K_{i}}{n}$<br>概率已知数据：$\bar{K}=\sum_{i=1}^{n} K_{i} P_{i}$                                                                                                                                                                                                                                                                                                                                                                                                       | **样本标准差**<br>单类资产（已测历史数据）：$d=\sqrt{\frac{\sum\left(K_{i}-\bar{K}\right)^{2}}{n-1}}$<br>单类资产（概率已知数据）：$\sigma=\sqrt{\sum_{i=1}^{n}\left(K_{i}-\bar{K}\right)^{2} P_{i}}$<br><br>**变异系数**<br>单类资产（已测历史数据）：$Q=\frac{d}{\bar{K}}$<br>单类资产（概率已知数据）：$Q=\frac{\sigma}{\bar{K}}$ |
| 组合资产 | **期望报酬率**<br>单类资产$j$的收益：$K_{j}=\sum_{i=1}^{n} K_{i j} P_{i}$<br>按投资比重计算加权平均值：$\bar{K}=\sum_{j=1}^{m} W_{j} K_{j}$<br>**组合风险（协方差/相关系数）**<br>两项资产组合的方差：$\sigma_{p}^{2}=W_{1}^{2} \sigma_{1}^{2}+W_{2}^{2} \sigma_{2}^{2}+2 W_{1} W_{2} \rho_{1,2} \sigma_{1} \sigma_{2}$<br>两项资产组合的标准差：$C O V_{1,2}=\sigma_{1,2}=\rho_{1,2} \sigma_{1} \sigma_{2}$<br>其中：$j$ 类资产的标准差 $\sigma_{j}=$ 资产 $j$ 的<br>该期报酬率 - 该期平均报酬率的离差的平方与概率的乘积和的平方根<br>$\rho_{1,2}$ 为两项资产的相关系数，$T_{1,2}=1$ 时，组合不能抵消任何风险；$T_{1,2}=-1$ 时，抵消效应最大 | **组合资产**<br>充分投资组合的风险，只受证券之间协方差的影响，而与各证券本身的方差无关<br>组合资产的标准差：$\sigma_{p}=\sqrt{\sum_{j=1}^{m} \sum_{k=1}^{m} W_{j} W_{k} \sigma_{j k}}$（注：$\sigma_{j k}$ 为 $j$、$k$ 资产的协方差 ）                                                                                              |

# 资本资产定价模型
| 分类                   | 内容                                                                                                                                  |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| 基本公式                 | $R_s = R_f + \beta×(R_m - R_f)$                                                                                                     |
| $\beta$的理解 - 衡量风险    | 用以衡量**系统**风险的大小                                                                                                                     |
| $\beta$的理解 - 与市场风险关系 | $\beta \lt 1$：该资产的系统风险 **<** 市场风险；$\beta \gt 1$：该资产的系统风险 **>** 市场风险<br>$\beta = 1$：该资产的系统风险 **=** 市场风险；$\beta = 0$：该资产的系统风险 **= 0** |
| $\beta$的理解 - 收益变动关系  | $\beta \lt 0$：该资产收益率与市场平均收益率 **反向变动**                                                                                               |
| $\beta$的理解 - 计算      | 单项资产的$\beta$：$\beta_i = r_{im}×\frac{\sigma_i}{\sigma_m}$<br>资产组合的$\beta$：组合中各单项资产$\beta$系数的加权平均数                                   |

# 模型1： 付息頻率，按季付息
### 一、债券基本参数（题目给定）  
![Pasted image 20250604165135](https://raw.githubusercontent.com/huiyouguang/mutiply/main/obsidianPasted%20image%2020250604165231.png)
- 面值 \( F = 1000 \) 元  
- 票面年利率 \( r_{票面} = 12\% \)（按年付息，若分期付息需拆分，本题后续按季度付息场景有变化 ）  
- 必要报酬率（投资者要求收益率） \( R = 8.24\% \)  
- 发行日：2022年1月1日，期限6年；计算涉及不同时间节点（2022发行时、2025年1月1日等 ）  


### 二、核心计算步骤与公式  
#### （一）“计算债券发行时的价值”（按<span style="background:#d3f8b6">季度付息场景</span> ）  
1. **步骤1：换算计息期利率**  
   因题目“折现率为非整数百分比，无法成现值系数，可用 \((1 + r)^n\) 计算”，且“一年付息4次（\( m = 4 \) ）”，先拆分周期：  
   - 计息期利率（季度利率）：  
     \[ r_{计息期} = \frac{必要报酬率}{m} = \frac{8.24\%}{4} = 2.06\% \]  
     （注：题目示例中简化为 \( 2\% \) ，可能是近似处理 ）  
   - 总付息期数 \( n = 6年×4 = 24 \) 期  

2. **步骤2：债券价值公式（分期付息、到期还本）**  
   债券价值 = 利息现值（普通年金现值） + 面值现值（复利现值）  
   \[ V = C × (P/A, r_{计息期}, n) + F × (P/F, r_{计息期}, n) \]  
   其中：  
   - \( C \) 为每期利息 = 面值×票面季度利率 = \( 1000×\frac{12\%}{4} = 30 \) 元  
   - \( (P/A, r, n) \) 是年金现值系数（本题因折现率非整数，实际用公式 \( \frac{1 - (1 + r)^{-n}}{r} \) 计算 ）  
   - \( (P/F, r, n) \) 是复利现值系数（实际用公式 \( (1 + r)^{-n} \) 计算 ）  

   题目示例简化计算：  
   \[ V = 30×(P/A, 2\%, 24) + 1000×(P/F, 2\%, 24) = 1189.12 \text{元} \]  


#### （二）“计算2025年1月1日的价值”（场景：付息频率/折现周期变化 ）  
![Pasted image 20250604165210](https://raw.githubusercontent.com/huiyouguang/mutiply/main/obsidianPasted%20image%2020250604171143.png)
1. **逻辑：折现到下一个付息时点**  
   2025年1月1日距离发行日3年，剩余期限3年，但需先“折现到最近付息日”（如2025年2月1日 ），再调整。  

2. **公式：分阶段折现**  
   - 先算“第一次折现（到2025年2月1日，剩余10个季度付息期 ）”：  
     \[ V_{2025.2.1} = 30×(P/A, 2\%, 10) + 1000×(P/F, 2\%, 10) \]  
     示例结果：\( 1119.78 \) 元（再折现到2025年1月1日，需除以 \( (1 + 季度利率) \) ，即 \( 1119.78÷(1 + 8.24\%/4) = 1105.10 \) 元 ）  


#### （三）“判断债券是否有吸引力（到期收益率法）”  
![Pasted image 20250604165231](https://raw.githubusercontent.com/huiyouguang/mutiply/main/obsidianPasted%20image%2020250604171157.png)
1. **步骤1：列现金流等式**  
   假设2025年1月1日债券价格 \( P = 1180 \) 元（题目给定），未来现金流：  
   - 剩余利息：30元/季度，共8次（到到期日剩余2年，\( 2×4 = 8 \) 期 ）  
   - 面值：1000元（到期还本 ）  
   现金流公式：  
   \[ 1180 = 30×(P/A, i, 8) + 1000×(P/F, i, 8) \]  
   （\( i \) 为季度到期收益率，需试算求解 ）  

2. **步骤2：试算到期收益率 \( i \)**  
   题目通过试算 \( i = 1\% \) 和 \( i = 2\% \) ，用内插法近似：  
   - \( i = 1\% \) 时，现值 \( = 30×7.3255 + 1000×0.9235 = 1143.27 \) 元（小于1180 ）  
   - \( i = 2\% \) 时，现值 \( = 30×7.3255 + 1000×0.8507 = 1103.27 \) 元（小于1180 ）  
   （注：实际需调整试算利率，本质是解上述方程求 \( i \) ，再换算成年到期收益率 \( R_{到期} = i×4 \) ，与必要报酬率 \( 8.24\% \) 比较，判断是否投资 ）  


### 三、关键公式总结  
1. **分期付息债券价值基本公式**：  
   \[ V = \sum_{t=1}^{n} \frac{C}{(1 + r)^t} + \frac{F}{(1 + r)^n} \]  
   （\( C \) 每期利息，\( r \) 每期折现率，\( n \) 付息期数 ）  

2. **到期收益率（YTM）计算**：  
   令 “债券价格 = 未来现金流现值”，解方程求折现率 \( r \) ，再换算为年收益率（若分期付息，\( R_{年} = r×m \) ，\( m \) 为每年付息次数 ）  


核心逻辑围绕“**现金流折现**”，通过拆分付息周期、计算每期折现率，将未来利息和本金按投资者要求的报酬率折现为当前价值，对比价格判断投资价值。

# 各种利率对比表
| 利率类型         | 定义                                             |
| ------------ | ---------------------------------------------- |
| 必要报酬率        | 投资者要求的最低报酬率，是等风险投资的机会成本；当内含报酬率≥必要报酬率时，投资人才可能投资 |
| 期望报酬率（内含报酬率） | 购买股票的内含报酬率                                     |
| 到期收益率（内含报酬率） | 购买债券后一直持有至到期的内含报酬率                             |
| 票面利率         | 一年内向投资者支付的利息占票面金额的比率                           |
| 报价利率（名义利率）   | 金融机构提供的利率                                      |
| 计息期利率（周期利率）  | 每个计息期向投资者支付的利息占票面金额的比率；计息期利率 = 报价利率÷一年内的计息次数   |
| 有效年利率（实际利率）  | 按复利计算的一年期的利率；有效年利率 = （1 + 计息期利率）^一年内计息次数 - 1   |
| 年折现率         | 又叫贴现率，是等风险投资的市场利率，年折现率为按年计算的有效年利率              |
| 市场利率         | 由市场上供求关系决定的利率                                  |
