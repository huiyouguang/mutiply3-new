# 考点3 财务管理的基本目标（重要）
|               |                                                                                            |     |
| ------------- | ------------------------------------------------------------------------------------------ | --- |
| 目标类型          | 具体内容                                                                                       |     |
| 利润最大化         | 当**投入资本、利润取得的时间、获得利润承担的风险**相同时，该观点可以接受。                                                    |     |
| 每股收益最大化       | 当**每股收益的取得时间、承担风险、投入资本**相同时，该观点可以接受。                                                       |     |
| 股东财富最大化（最优观点） | 股东财富可以用**股东权益的市场价值**来衡量<br>股东财富的增加可以用**股东权益的市场增加值**来衡量<br>股东财富的增加计算公式：`股东权益的市场价值 - 股东投资资本` |     |
| 股价最大化         | 当**资本市场有效且股东投资资本**不变时，该观点可以接受。                                                             |     |
| 公司价值最大化       | 当**债务价值、股东投资资本**不变时，该观点可以接受。                                                               |     |
1
以下口诀从梳理各目标关键限定条件、核心衡量指标角度，帮助记忆：
1. **利润最大化**：利大接受条件：投时险同（投入资本、时间、风险相同  ） 
2. **每股收益最大化**：每益接受条件：时险投同（时间、风险、投入资本相同  ） 
3. **股东财富最大化**：股财衡量：权益市价；增加衡量：市价增值；计算：市价减投本 
4. **股价最大化**：价大接受条件：资市有效、投本不变（资本市场有效且股东投资资本不变  ） 
5. **公司价值最大化**：值大接受条件：债值、投本不变（债务价值、股东投资资本不变  ） 

可简化串联：“利大投时险同，每益时险投同；股财看权益市价、增值用市价减投本；价大资市效、投本稳，值大债值、投本定” ，口诀把各目标核心区分点提炼，方便快速回忆不同目标的适用条件与衡量逻辑 。

# 考点4 重要 利益相关者
| 利益相关者 | 与股东的冲突                                                                                                                                                   | 协调方式                                                                                                    |
| ----- | -------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| 经营者   | ①逆向选<span class="cloze-span">择：为自己的目标而背离股东的</span>目标；<br>②道德风险：<span class="cloze-span">不是尽最大努力去实现股东的目标</span><br>口诀：<span class="cloze-span">离心力</span> | ①监督：财务监督、信息披露、审计<br>②激励：奖金、股权激励<br>最佳解决办法：<span class="cloze-span">使得监督成本、激励成本、偏离股东目标的损失之和最小</span><br> |
| 债权人   | <span class="cloze-span">①未经债权人同意而投资于比债权人预期风险更高的新项</span>目；<br>②<span class="cloze-span">未经债权人同意而发行新债</span><br>口诀：<span class="cloze-span">高新</span>    | ①寻求立法保护：如破产时先行接管，先于股东分配剩余财产等<br>②加入限制性条款：限制用途、限制发新债等<br>③拒绝进一步合作：不提供新贷款、提前收回贷款等                         |


口诀：“经营者冲突逆选、道德，协调监督、激励，求和最小；债权人冲突高险、新债，协调立法、限条、拒合” ，提炼核心冲突与协调点，方便快速回忆 。
