# 一、成本性态分析
**成本性态**，又称成本习性，是指成本总额与业务量（如产品产量、销量等）之间的内在关系。成本按其性态分类，可分为固定成本、变动成本与混合成本三大类。

## 1 （一）固定成本
| 含义  | 是指在特定的业务量范围内不受业务量变动影响，一定期间的总额能保持相对稳定的成本  |
| --- | ---------------------------------------- |
| 特征  | 特定的业务量范围内固定成本总额具有稳定性，而单位固定成本随着业务量的增加反向变动 |
| 注意  | 一定期间的固定成本的稳定性是有条件的，即业务量的变动是在特定的相关范围之内    |

### 1.1 固定成本的分类
| <span style="background:#d3f8b6">约束性固定</span>成本                                                                                                                                                                     | <span style="background:#d3f8b6">酌量性固定</span>成本                                                   |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **含义和特点**：<span class="cloze-span">提供和维持生产经营所需设施、机构而发生的成本，不能通过当前的管理决策行动加以改变，属于企业“经营能力成本”，是企业为了维持一定的业务量所必须负担的最低成本<br></span><br><br>**降低措施**：只能合理利用经营能力、增加生产规模、进而降低单位固定成本<br><br>**举例**：固定资产折旧费、财产保险、管理人员工资、取暖费、照明费等 | **含义和特点**：可以通过管理决策行动而改变数额的固定成本，是由经理人员决定的，但对于期望保持长期稳定发展的企业来说，绝非可有可无<br><br>**举例**：科研开发费、广告费、职工培训费等 |

# 二、变动成本法
变动成本法下，产品成本只包括变动生产成本（直接材料、直接人工和变动制造费用）。固定制造费用和非生产成本全部作为制造边际贡献（销售额与变动生产成本的差额）的扣除项目，即全部作为期间成本。

## 1 （一）变动成本法与完全成本法的比较
| 项目    | 产品成本的构成            | 期间费用的构成        | 对存货计价的影响     |
| ----- | ------------------ | -------------- | ------------ |
| 变动成本法 | 变动生产成本（料、工和变动制造费用） | 固定制造费用和全部非生产成本 | 存货中不包括固定制造费用 |
| 完全成本法 | 全部生产成本             | 全部非生产成本        | 存货中包括固定制造费用  |

## 2 （二）变动成本法的优势
1. 消除了在完全成本法下，销售不变但可通过增加生产、调节库存来调节利润的问题，可以使企业内部管理者更加注重销售，更加注重市场，便于进行更为合理的内部业绩评价，为企业内部管理提供有用的管理信息，为企业预测前景、规划未来和作出正确决策服务；
2. 能够揭示利润和业务量之间的正常关系；
3. 便于分清各部门经济责任，有利于进行成本控制和业绩评价；
4. 可以简化成本计算，便于加强日常管理。

## 3 【例题•单选题】
如果企业采用变动成本法核算产品成本，产品成本的计算范围是（　　）。（2011年）
- A. 直接材料、直接人工
- B. 直接材料、直接人工、间接制造费用
- C. 直接材料、直接人工、变动制造费用
- D. 直接材料、直接人工、变动制造费用、变动管理及销售费用

**答案**：<span class="cloze-span">C  </span>
**解析**：<span class="cloze-span">变动成本法下，产品成本只包括变动生产成本（直接材料成本、直接人工成本和变动制造费用），固定制造费用和非生产成本全部作为制造边际贡献（销售额与变动生产成本的差额）的扣除项目，即全部作为期间成本，选项C正确。</span>

# 三、本量利分析基本模型的相关假设
| 假设类型     | 具体内容                                                                                                                              |
| -------- | --------------------------------------------------------------------------------------------------------------------------------- |
| 相关范围假设   | 1. **期间假设**：无论是固定成本还是变动成本，其固定性与变动性均体现在特定的期间内。<br>2. **业务量假设**：对成本按性态进行划分而得到的固定成本和变动成本，是在一定业务量范围内分析和计量的结果。                         |
| 模型线性假设   | 1. **固定成本不变假设**：在企业经营能力的相关范围内，固定成本是不变的。<br>2. **变动成本与业务量呈完全线性关系假设**。<br>3. **销售收入与销售数量呈完全线性关系**：通常假设销售价格为一个常数，销售收入与数量之间就呈现完全线性关系。 |
| 产销平衡假设   | 本量利分析中的“量”指的是销售数量而非生产数量，本量利分析的核心是分析收入与成本之间的对比关系，站在销售数量的角度进行本量利分析时，就必须假设产销关系是平衡的。                                                  |
| 品种结构不变假设 | 指在一个多品种生产和销售的企业中，各种产品的销售收入在总收入中所占的比重不会发生变化。                                                                                       |

# 四、本量利分析基本模型  
## 1 （一）损益方程式  
### 1.1 基本的损益方程式  
**利润** = 销售收入-总成本（S-VC-F）=<span class="cloze-span"> 单价×销量-单位变动成本×销量-固定成本（P×Q-V×Q-F）  </span>
**【教材例题】**  
某企业每月固定成本为1000元，仅生产一种产品，销售单价为10元，单位变动成本为6元，本月计划销量500件，问预期利润是多少？  
**答案**：利润 =<span class="cloze-span"> 10×500-6×500-1000 = 1000（元）。  </span>
### 1.2 包含期间成本的损益方程式  
税前利润 = <span class="cloze-span">销售收入-（变动生产成本+固定生产成本）-（变动销售和管理费用+固定销售和管理费用）= 单价×销量-（单位变动生产成本+单位变动销售和管理费用）×销量-（固定生产成本+固定销售和管理费用）  </span>
**提示**：该方程式假设税前利润的影响因素仅包含销售收入、生产成本、管理费用和销售费用，省略了税金及附加、财务费用等其他因素。  
### 1.3 计算税后利润的损益方程式  
**税后利润** =<span class="cloze-span"> 利润总额×（1-所得税税率）=（单价×销量-单位变动成本×销量-固定成本）×（1-所得税税率）  </span>


## 2 （二）边际贡献方程式  
### 2.1 核心概念  
| 项目 | 公式 |  
|------|------|  
| **单位边际贡献（m）** | 单价-单位变动成本（P-V） |  
| **边际贡献（M）** | ① 销售收入-变动成本（S-VC）<br>②（单价-单位变动成本）×销量（P-V）×Q<br>③ 单位边际贡献×销量（m×Q） |  
| **制造边际贡献** | 销售收入-产品变动成本 |  
| **产品边际贡献** | 制造边际贡献-变动销售和管理费用 |  

**提示**：未特别说明时，“边际贡献”通常指“产品边际贡献”。  


### 2.2 利润计算公式  
利润 = 边际贡献-固定成本（M-F） = （单价-单位变动成本）×销售量-固定成本［（P-V）×Q-F］ = 单位边际贡献×销售量-固定成本（m×Q–F） = 销售收入×边际贡献率-固定成本  


### 2.3 边际贡献率与变动成本率  
| 项目 | 公式 |  
|------|------|  
| **边际贡献率（cmR）** | 边际贡献/销售收入（M/S）= 单位边际贡献/单价（m/P） |  
| **变动成本率（bR）** | 变动成本/销售收入（VC/S）= 单位变动成本/单价（V/P） |  
| **关系公式** | 变动成本率+边际贡献率=1（bR + cmR = 1） |  

**手写板示意**：  
`变动成本率 + 边际贡献率 = （变动成本/销售收入） + （边际贡献/销售收入） = 1`

【例题】某企业生产甲产品，售价为60元/件，单位变动成本24元/件，固定成本总额100000元，当年产销量20000件。
【要求】计算单位边际贡献、边际贡献总额、边际贡献率及利润。
1. 【答案】
	1. 单位边际贡献={{c1::单价-单位变动成本=60-24=36（元/件）}}
	2. 边际贡献总额={{c1::单位边际贡献×产销量=36×20000=720000（元）}}
	3. 边际贡献率={{c1::36/60×100%=60%}}
	4. 或边际贡献率={{c1::720000/（60×20000）×100%=60%}}
	5. 利润={{c1::720000-100000=620000（元）}}。

## 3 （二）变动成本  
|项目|内容|  
|------|------|  
|**含义**|是指在特定的业务量范围内，其总额随业务量变动而正比例变动的成本。|  
|**特征**|1. 变动成本总额随业务量成**正**比例变动；<br>2. 单位变动成本**稳定**。|  

### 3.1 变动成本的分类  
|分类|技术性变动成本|酌量性变动成本|  
|------|------|------|  
|**含义和特点**|与产量有明确的生产技术或产品结构设计关系，是利用生产能力所必须发生的成本。|可以通过管理决策行动改变的变动成本。|  
|**举例**|直接材料、直接人工、外部加工费等。|按销售额一定比例支付的销售佣金、新产品研制费、技术转让费等。|  

### 3.2 【例题•单选题】  
下列各项中，属于酌量性变动成本的是（　　）。（2013年改编）  
- A. 直接人工成本  
- B. 直接材料成本  
- C. 职工培训费  
- D. 按销售额一定比例支付的销售代理费  

**答案**：<span class="cloze-span">D  </span>
**解析**：<span class="cloze-span">酌量性变动成本是可以通过管理决策行动改变的变动成本，例如按销售额一定比例开支的销售佣金、新产品研制费、技术转让费等，选项D正确；选项A和B属于技术性变动成本；选项C属于酌量性固定成本。</span>

### 4.1 （四）混合成本的分解

#### 4.1.1 回归直线法
- **含义**：根据一系列历史成本资料，运用数学上的最小平方法原理，计算能代表平均成本水平的直线截距和斜率，将其作为固定成本和单位变动成本的一种成本估计方法。
- **适用范围**：在采用传统成本计算方法时，可用回归直线法估计固定成本和单位变动成本数据，以用于成本计划和控制。

#### 4.1.2 工业工程法
- **含义**：运用工业工程的研究方法，对确定成本高低的每个因素进行逐项研究，在此基础上直接估算固定成本和单位变动成本的一种成本估计方法。
- **适用范围**：可在没有历史成本数据、历史成本数据不可靠，或者需要对历史成本分析结论进行验证的情况下使用。

### 4.2 【教材例题】
选择燃料成本作为研究对象。燃料用于铸造工段的熔炉，具体分为点火（耗用劈柴和焦炭）和熔化铁水（耗用焦炭）两项操作。对这两项操作进行观测和技术测定，寻找最佳的操作方法。按照最佳的操作方法，每次点火要耗用劈柴0.1吨、焦炭1.5吨，熔化1吨铁水耗用焦炭0.15吨；每个工作日点火一次，全月工作26天，点火燃料属固定成本；熔化铁水所用燃料与产量相联系，属于变动成本。劈柴每吨价格为500元，焦炭每吨价格为1800元。

根据上述资料计算燃料成本：
- 每日固定成本：0.1×500 + 1.5×1800 = 2750（元）
- 每月固定成本：2750×26 = 71500（元）
- 每吨铸件变动成本：0.15×1800 = 270（元）

设燃料总成本为y，产量为x吨铸件，则每月燃料总成本为：y = 71500 + 270x 。

### 4.3 一、保本分析与安全边际分析
| 表示方法    | 保本点                                                                                        | 安全边际                                                                   |
| ------- | ------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------- |
| **实物量** |                                                                                            |                                                                        |
| 指标      | **保本量**                                                                                    | **安全边际量**                                                              |
| 计算公式    | 保本量（$Q_0$） = 固定成本/（单价 - 单位变动成本） = $F/(P - V)$                                              | 安全边际量 = 正常销售量 - 盈亏临界点销售量 = $Q - Q_0$                                   |
| **金额**  |                                                                                            |                                                                        |
| 指标      | **保本额**                                                                                    | **安全边际（额）**                                                            |
| 计算公式    | 保本额（$S_0$） = 固定成本/边际贡献率 = $F/cmR$                                                          | 安全边际（额） = 正常销售额 - 盈亏临界点销售额 = $S - S_0$                                 |
| **相对数** |                                                                                            |                                                                        |
| 指标      | **盈亏临界点作业率**                                                                               | **安全边际率**                                                              |
| 计算公式    | 盈亏临界点作业率 = 盈亏临界点销售量（额）/实际或者预计销售量（额） = $Q_0/Q$（或$S_0/S$）                                    | 安全边际率 =（正常销售额（量） - 盈亏临界点销售额）/实际或者预计销售量（额）= $(Q - Q_0)/Q$或$(S - S_0)/S$ |
| 关系公式    | **盈亏临界点销售量 + 安全边际量 = 正常销售量**<br>**盈亏临界点销售额 + 安全边际（额） = 正常销售额**<br>**盈亏临界点作业率 + 安全边际率 = 1** |                                                                        |
|         |                                                                                            |                                                                        |

**解析**：
- **保本量**：是企业达到盈亏平衡时的销售数量。固定成本需由单位产品创造的边际贡献（单价 - 单位变动成本 ）来弥补，所以用固定成本除以单位边际贡献得到保本量 。
- **保本额**：是盈亏平衡时的销售金额。因为边际贡献率表示每一元销售收入中边际贡献所占比重，所以固定成本除以边际贡献率可得到保本额 。
- **安全边际量（额）**：反映了企业销售量（额）在超过盈亏临界点销售量（额）后，额外的销售规模，体现了企业盈利的缓冲空间。
- **盈亏临界点作业率**：表明企业在保本状态下，生产能力的利用程度。
- **安全边际率**：衡量企业经营安全程度，该比率越高，企业经营越安全。安全边际率与盈亏临界点作业率之和为1 ，二者此消彼长。
