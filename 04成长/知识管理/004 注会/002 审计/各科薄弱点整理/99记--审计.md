# 第一模块  审计基本概念
## 第一记 审计分类 1 分

| 要点          | 政府审计                                                               | 注册会计师审计                                |
| :---------- | :----------------------------------------------------------------- | :------------------------------------- |
| **执行主体**    | 政府审计机关（审计署和地方审计厅局）                                                 | 注册会计师                                  |
| **行为性质**    | 行政行为                                                               | 市场行为、有偿服务                              |
| **审计对象**    | 国务院各部门和地方各级人民政府及其各部门的财政收支、国有金融机构和企业事业组织的财务收支，以及其他按照该法规定应当接受审计的财政收支 | 除政府审计对象以外的事项（部分国有金融机构和企事业单位可能同时进行两种审计） |
| **审计目的**    | 监督上述财政收支或财务收支的真实、合法、效益                                             | 对被审计单位的财务报表或内部控制发表审计意见                 |
| **执业依据**    | ==《中华人民共和国审计法》和国家审计准则等==                                           | ==《中华人民共和国注册会计师法》和注册会计师审计准则等==         |
| **经费或收入来源** | 财政预算且由政府保证                                                         | 和审计客户协商                                |
| **取证权限**    | 具备更大的强制力，有关单位和个人应当支持、协助、反馈和提供材料                                    | 不具备行政强制力，很大程度上有赖于企业及相关单位配合和协助          |


| 要点          | 内部审计                            | 注册会计师审计                   |
| :---------- | :------------------------------ | :------------------------ |
| **执行主体**    | 企业的内部审计职能部门                     | 会计师事务所的注册会计师              |
| **业务性质**    | 企业内部管理活动                        | 外部独立鉴证活动                  |
| **审计独立性**   | 受到一定限制，独立性只是相对于本单位其他职能部门而言的     | 较强的独立性                    |
| **审计方式**    | ==根据自身经营管理的需要安排进行==             | ==接受委托进行==                |
| **审计程序**    | ==根据所执行业务的目的和需要选择并实施==          | 严格按照执业准则的规定进行             |
| **审计职责**    | 只对本单位负责                         | 不仅对被审计单位负责，而且对社会负责        |
| **审计质量的影响** | ==审计质量基本与外界无直接关系==              | 审计质量对广大财务信息使用者做出相关决策有直接影响 |
| **审计作用**    | ==只作为本单位改善工作的参考，对外不起鉴证作用，对外保密== | 起鉴证作用，阅读对象不限于被审计单位        |

## 第二记 注册会计师的业务类型和保证程度 1 分
| 业务类型     | 项目示例                                              | 相应的保证程度            |
| :------- | :------------------------------------------------ | :----------------- |
| **鉴证业务** |                                                   |                    |
| 审计       | （1）[[财务报表审计]]<br>（2）==企业内部控制审计==                  | 合理保证（高水平）          |
| 审阅       |                                                   | 有限保证（低于高水平）        |
| 其他鉴证业务   | 如 [[预测性财务信息审核]]                                   | ==合理保证和有限保证均可能涉及== |
| **相关服务** |                                                   |                    |
| -        | （1）代编财务信息<br>（2）对财务信息执行商定程序<br>（3）税务咨询<br>（4）管理咨询 | ==不提供任何程度的保证==     |

| 比较维度         | 财务报表审计                       | 财务报表审阅                              |
| :----------- | :--------------------------- | :---------------------------------- |
| **业务性质**     | 合理保证的鉴证业务                    | 有限保证的鉴证业务                           |
| **对独立性的要求**  | 必须具备形式上和实质上的独立性              | 必须具备形式上和实质上的独立性                     |
| **检查风险**     | 较低                           | 较高                                  |
| **证据数量**     | 较多                           | 较少                                  |
| **取证程序**     | 检查、观察、询问、函证、重新计算、重新执行、分析程序等  | 受到有意识的限制，==主要通过询问和分析程序，必要时才追加其他程序== |
|**结论方式（节选）**
结论方式（节选）| 积极方式：“我们认为按照规定编制了报表，公允反映了情况” | 消极方式：“我们没有注意到财务报表没有按照规定编制，未能公允反映情况” |

命题- **角度**：考查==审阅业务的概念要点==。
- **坑 1**：执行审阅业务时，==主要通过询问和分析程序来收集证据==，而非仅实施该等程序；与审计类似，审阅结论也需要==充分、适当的证据==予以支持。
- **坑 2**：在**集团审计**中，对于**重要组成部分**（因"**财务重大性**"或"**特别风险**"），注册会计师通常==无法仅实施审阅程序满足集团审计的目标==；对于**不重要组成部分**，注册会计师可能决定实施审阅程序。
- **坑 3**：执行审阅业务时，注册会计师需要具备<u>**独立性**</u>；需要确定<u>**重要性**</u>。

## 第三记 审计概念和要素
### 审计五要素
1. 预期使用者
	1. 主要利益相关者，通常包括股东、公司债权人、供应商、客户、证券监管机构等组织或人员
		1. 不包括 cpa
	2. 与 cpa 关系
	3. 与管理层关系
		1. （1）注册会计师的审计意见主要是向除管理层之外的预期使用者提供的，审计的目的是增强除管理层之外的预期使用者对财务报表信赖的程度。
		2. （2）管理层可能是预期使用者，但不能是唯一的预期使用者。
		3. （3）管理层和预期使用者可能来自同一企业，但不意味着两者是同一方
	
### 基本要求
### 总体目标
1. （1）对财务报表整体是否不存在舞弊或错误导致的重大错报获取合理保证，使注册会计师能够对财务报表是否在所有重大方面按照适用的财务报告编制基础编制发表审计意见。
2. （2）按照审计准则的规定，根据审计结果对财务报表出具审计报告，并与管理层和治理层沟通



> [!note]
> 坑1：审计不涉及为如何利用信息提供建议。
> 坑2：审计证据==大多数==是==说服性==而非结论性的，注意此处<u>不应绝对地表述为全部</u>。
> 坑3：审计的产品是**审计报告**，<u>不包括</u>后附的财务报表或审计工作底稿。
> 坑4：股东、债权人、证券监管机构、社会公众、公司员工等组织或个人均属于**预期使用者**。
> 坑5：管理层是预期使用者之一，但<u>不能是唯一的</u>预期使用者；注册会计师执行财务报表审计，<u>不属于</u>预期使用者。
> 坑6：对于上市公司而言，审计报告的收件人通常明确为上市公司的全体股东，但<u>不等同于</u>预期使用者是指上市公司的全体股东。



## 第四记 保持职业怀疑和运用职业怀疑
1. 保持职业怀疑的==影响因素==
	1. 与职业道德相互关联
	2. 对会计事务所业绩评价促进和削弱审计
	3. 取决于胜任能力
2. 保持职业怀疑的要求
3. 保持职业怀疑的作用
	1. 增加质量的关键因素
	2. 提高设计和执行的有效性，降低审计风险，对发现和防止舞弊失败只管只有



#### 职业判断的衡量标准

1. 准确性
2. 决策的一惯性（==“同一人对不同问题”==）
	1. 指同一注册会计师针对同一项目的不同判断问题，所作出的判断之间是否符合应有的内在逻辑（“同一人对不同问题”）
3. 意见的一致性==（“同一人对同一问题”）==
4. ==可辩护性==
	1. （1）对职业判断问题和目标的描述。
	2. （2）解决职业判断相关问题的思路。
	3. （3）收集到的相关信息。
	4. （4）得出的结论以及得出结论的理由。
	5. （5）就决策结论与被审计单位进行沟通的方式和时间
5. 稳定性

# 第五记  审计风险
| 项目              | 内容                                                                                                                                                        |
| :-------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **重大错报风险**      | **概念和特征**：指财务报表在审计前存在==重大错报==的可能性。与被审计单位的风险相关，==独立于财务报表审计而存在==，属于客观存在的风险。<br>**财务报表层次的重大错报风险**：指与财务报表整体存在==广泛联系==的重大错报风险，==难以界定于某类具体认定==，通常影响多项不同的认定。     |
| **认定层次的重大错报风险** | 指与==特定类别==的交易、账户余额或披露相关的重大错报风险。                                                                                                                           |
| **固有风险**        | **含义**：指在==不考虑控制==的情况下，某一认定易于发生错报的可能性。<br>**因素**：可以是定性的，也可以是定量的，包括：<br>1. 事项或情况的==复杂性、主观性、变化、不确定性==。<br>2. 管理层==偏向。==<br>3. 其他舞弊风险因素。<br>4. 产生经营风险的外部因素。  |
| **控制风险**        | **含义**：指某一认定发生错报，该错报单独或连同其他错报是重大的，但没有被内部控制==及时防止或发现并纠正==的可能性。<br>**理解要点**：<br>1. 取决于内部控制设计的合理性和运行的有效性。<br>2. 由于控制的固有局限性，控制风险==始终存在==。                     |
| **检查风险**        | **含义**：如果存在某一错报，该错报单独或连同其他错报可能是重大的，注册会计师为将审计风险降至==可接受的低水平==而实施程序后==没有发现==这种错报的风险。<br>**影响因素**：取决于审计程序设计的合理性和实施的有效性。                                       |
| **审计风险**        | **含义**：指财务报表存在重大错报时，注册会计师发表==不恰当审计意见==的可能性。<br>**构成**：取决于重大错报风险和检查风险。<br>**关系**：在审计风险既定的情况下，可接受的检查风险水平与评估的认定层次重大错报风险呈==反向关系==；评估的认定层次重大错报风险越高，可接受的检查风险越低。 |



#### 审计的固有限制
1.财报性质
2.审计程序的性质
3.财报及时性和成本效益的权衡


- **命题角度**：考查审计相关风险的概念要点。
- **坑 1**：重大错报风险包括财务报表层次和认定层次，两个层次的重大错报风险可能来源于错误或舞弊；不能绝对地表述为财务报表层次的重大错报风险均来源于舞弊，而认定层次的重大错报风险均来源于错误。
- **坑 2**：评估固有风险时==不考虑相关控制的影响。==
- **坑 3**：检查风险取决于==注册会计师设计和实施审计程序==的情况，抽样风险和非抽样风险均可能导致检查风险。
- **坑 4**：审计风险是一个专有名词，并不指注册会计师被起诉而发生赔款或名誉受损等风险。
- **坑 5**：重大错报风险不受到审计的影响，属于独立于审计而客观存在的风险；注册会计师只能识别、评估和应对重大错报风险，而不能控制、调节和消除重大错报风险。



# 第六记  职业责任


### 违反 1．《中华人民共和国注册会计师法》的规定如果违反法律法规、执业准则、规则等，可能会面临如下处罚：

| 主体         | 一般情形                              | 情节严重                     |
| :--------- | :-------------------------------- | :----------------------- |
| **会计师事务所** | 可能受到财政部门的行政处罚，包括警告、没收违法所得（可以并处罚款） | ==可以暂停其经营业务或者予以撤销==      |
| **注册会计师**  | 可能受到财政部门的行政处罚，包括警告                | ==可以暂停其执行业务或者吊销注册会计师证书== |

### 行政责任
2．《中华人民共和国证券法》的规定对于从事证券期货业务的会计师事务所如果违反本法相关规定，未勤勉尽责，所制作、出具的文件有虚假记载、误导性陈述或者重大遗漏的：

1. 1. 责令改正，==没收业务收入==，并处以业务收入 **1 倍以上 10 倍以下**的罚款；没有业务收入或者业务收入不足 <u>50 万元</u>的，处以 <u>50 万元以上 500 万元以下</u>的罚款。
2. ==情节严重的==，并处==暂停或者禁止从事证券服务业务==。
3. 对==直接负责的主管人员==和==其他直接责任人员==给予**警告**，并处以 **20 万元以上 200 万元以下**的罚款。



### 民事责任
1. **《中华人民共和国注册会计师法》** 规定：
   - 会计师事务所违反本法规定，给委托人、其他利害关系人造成损失的，应当依法承担 ==赔偿责任==。

2. **《中华人民共和国证券法》** 规定：
   - **证券服务机构**在证券业务活动中制作、出具文件时，应当 <u>勤勉尽责</u>，对文件资料的真实性、准确性、完整性进行核查和验证。
   - 若文件存在虚假记载、误导性陈述或者重大遗漏并给他人造成损失，应当与委托人承担 ==连带赔偿责任==（除非能证明自己没有过错）。


### 行使责任

1. **个人刑事犯罪**
    1. **提供虚假证明文件罪**
        - 主体：承担资产评估、验资、验证、会计、审计等职责的中介组织的人员。
        - 行为：==故意==提供虚假证明文件。
        - 处罚情形：
            - 情节严重的：处 `5 年以下`有期徒刑或者拘役，并处罚金。
            - 提供与==证券发行==相关的虚假证明文件，情节特别严重的：处 `5 年以上 10 年以下`有期徒刑，并处罚金。
            - 提供与==重大资产交易==相关的虚假证明文件，情节特别严重的：处 `5 年以上 10 年以下`有期徒刑，并处罚金。
            - 有上述行为，同时==索取他人财物==或者==非法收受他人财物==构成犯罪的：依照处罚较重的规定定罪处罚。
    2. **出具证明文件重大失实罪**
        - 主体：前款规定的人员。
        - 行为：==严重不负责任==，出具的证明文件有==重大失实==。
        - 后果：造成==严重后果==的。
        - 处罚：处 `3 年以下`有期徒刑或者拘役，并处或者单处罚金。
2. **单位刑事犯罪**
    - 法律依据：《中华人民共和国刑法》。
    - 犯罪主体：==会计师事务所==。
    - 涉嫌罪名：提供虚假证明文件和出具证明文件重大失实。
    - 处罚措施：
        - 对单位：判处罚金。
        - 对人员：对其==直接负责的主管人员==和==其他直接责任人员==，依照各条款的规定处罚。

# 第八记 错报
1. 事实错报
2. 判断错报
3. 分类错报

4. **错报的概念**具有==双重含义==：
   5. 将被审计单位作出的<u>金额、分类或列报</u>与**财务报告编制基础**作比较。
   6. 将被审计单位作出的<u>金额、分类或列报</u>与**注册会计师的判断**作比较。
7. **会计估计错报的评价**：
   8. **评价方法**：注册会计师可作出==点估计==或==区间估计==以评价管理层的点估计。
   9. **构成错报的情形**（当审计证据支持注册会计师结果时）：
      - 管理层的点估计与注册会计师的==点估计==之间的差异。
      - 管理层的点估计与注册会计师的==区间估计==之间的<u>最小差异</u>。
   10. **错报类型**：与会计估计有关的错报可能是==事实错报==、==判断错报==或==推断错报==。
11. **错报的累积与评价标准**：
   12. **是否累积并更正**：应以==明显微小错报临界值==作为衡量标准。
   13. **对审计意见或经济决策的影响**：应以==财务报表整体重要性==以及<u>特定类别的重要性</u>为衡量标准。
14. **特殊情形的恰当评价**：
   15. ==错报抵销==。
   16. ==分类错报==。
   17. 由==舞弊==或==系统性控制偏差==导致的错报。
   18. **样本错报对总体错报的影响**（如在<u>函证</u>、<u>监盘</u>或其他**审计抽样测试**中）。



# 第九记   审计证据的性质

1. **充分性**
    - **含义**：是对审计证据==数量==的衡量，与注册会计师确定的==样本量==有关。
    - **影响因素**：
        - **重大错报风险评估结果**：评估的被审计单位的重大错报风险==越高==，所需审计证据的数量==越多==。
        - **审计证据质量**：质量==越高==，可能需要的数量==越少==；质量存在==缺陷==，可能无法通过获取更多的审计证据予以弥补。
2. **适当性**
    - **含义**：是对审计证据==质量==的衡量，只有==相关且可靠==的审计证据才是高质量的。
        - **相关性**：指用作审计证据的信息与审计程序的目的、相关认定之间的==逻辑联系==。
        - **可靠性**：指证据的==可信程度==。
    - **相关性**：
        - **测试方向**：
            - **逆查**：由“账”到“证” → ==高估== → ==存在或发生==认定。
            - **顺查**：由“证”到“账” → ==低估== → ==完整性==认定。
        - **审计程序**：
            - 特定程序可能只与<u>某些认定</u>相关，而与其他认定无关。
            - 某一认定的审计证据，==不能替代==与其他认定相关的审计证据。
            - 不同来源或不同性质的审计证据可能与<u>同一认定</u>相关。
    - **可靠性**（判断原则）：
        1.  亲自获取 > 被审计单位提供。
        2.  来自==外部独立渠道== > 其他渠道（例如，被审计单位内部）。
        3.  ==内控有效== > 内控薄弱。
        4.  直接获取 > 间接获取或推论。
        5.  ==文件记录==（纸质或电子介质） > 口头形式。
        6.  ==原件== > 传真件或复印件。
        7.  多部门流转 > 财务部门内部流转。
3. **充分性和适当性的评价**
    - **文件记录可靠性的考虑**：审计通常<u>不涉及鉴定</u>文件记录的真伪，注册会计师不是鉴定文件记录真伪的专家。但如在审计过程中识别出的情况可能表明文件记录是==伪造==的，注册会计师应做出==进一步调查==。
    - **使用被审计单位生成信息时的考虑**：评价此类信息的==完整性==和==准确性==。
    - **证据相互矛盾时的考虑**：==相互印证==的审计证据具有更强的说服力；如不同来源或不同性质的证据==不一致==，表明某项审计证据可能==不可靠==，应当追加必要的审计程序。
    - **对获取证据的成本的考虑**：可以考虑取证成本与信息有用性之间的关系，但<u>不应</u>以取证的困难和成本为由减少==不可替代==的审计程序。

> [!note]
> 
> 
> 4. **命题核心**：辨析审计证据的**充分性**和**适当性**。
> 5. **充分性（数量）**
>    - 影响因素包括：
>      - **重大错报风险的评估结果**：风险越高，所需数量越多。
>      - **审计证据的质量**：质量越高，所需数量越少；质量存在缺陷时，无法通过增加数量弥补。
>    - 质量的间接影响：审计证据与审计目的的**相关程度**、**来源**及**获取环境**等因素，会间接影响证据数量。
> 6. **适当性（质量）与充分性的关系**
>    - **核心逻辑**：**适当性影响充分性**（即质量影响数量）。
>      - **相关性**和**可靠性**均影响充分性，反之不成立。
>      - **关键原则**：无论是充分性还是适当性，都**无法弥补对方的缺陷或不足**。
>    - **高质量标准**：只有**相关且可靠**的审计证据才是高质量的，二者缺一不可。
> 7. **证据矛盾的处理**
>    - 当不同来源和性质的审计证据相互矛盾时，注册会计师应：
>      - 加以**调查核实**。
>      - **追加必要的审计程序**。
>      - <u>不应</u>仅仅比较后直接采用其中更为可靠的一份。
> 8. **利用被审计单位内部生成信息的原则**
>    - **总体要求**：评价此类信息的**完整性**和**准确性**。
>    - **典型应用场景**：
>      1. **存货监盘**：需基于完整的**存货存放地点清单**。
>      2. **银行询证函**：需基于完整的**已开立银行结算账户清单**。
>      3. **金额验证**（如坏账准备、跌价准备）：需测试相关表单（计提表、账龄分析表等）的**准确性和完整性**。
>    - **测试目标的区分**：
>      - **完整性测试**：例如将观察到的残冷背次情况与计提表核对，确认是否所有项目均被纳入考虑。
>      - **准确性测试**：若表单由系统自动生成，除测试表单本身外，还需考虑**信息技术一般控制**和**信息处理控制**的有效性。
>

## （一）重要性的概念
1.  如果合理预期错报（包括漏报）**<u>单独或汇总</u>**起来可能影响财务报表使用者的**<u>经济决策</u>**，则通常认为错报是重大的；
2.  对重要性的判断是根据**<u>具体环境</u>**作出的，并受**<u>错报金额、性质</u>**或两者共同作用的影响；
3.  考虑财务报表使用者**<u>整体共同</u>**的财务信息需求，**不考虑**错报对**<u>个别</u>**财务报表使用者可能产生的影响。

---
## （二）确定重要性的考虑因素
### 1. 四类重要性概念对比
| 重要性概念                    | 含义和确定方法                                                                                                    | 考虑因素                                                                                                             |
| ------------------------ | ---------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **财务报表整体重要性**            | 1. 运用职业判断，不考虑与具体项目相关的固有不确定性<br>2. 选定“基准”×“百分比”确定                                                           | （1）选择基准：报表要素、使用者特别关注项目、被审计单位性质/生命周期/行业、所有权结构/融资方式、基准相对波动性<br>（2）确定百分比：是否为上市公司/公众利益实体、使用者范围、是否有大额对外融资、使用者对基准数据敏感度 |
| **特定交易类别/账户余额/披露的重要性水平** | 根据被审计单位特定情况予以确定                                                                                            | 法律法规/编制基础对特定项目计量或披露的预期、行业相关关键性披露、使用者是否特别关注特定业务披露                                                                 |
| **实际执行的重要性**             | 1. 旨在将未更正和未发现错报的汇总数超过整体重要性的可能性降至适当低水平<br>2. **低于**财务报表整体重∫∫要性<br>3. 也为特定类别/账户/披露确定                         | ==对被审计单位的了解、前期识别出的错报性质和范围、根据前期错报对本期错报的预期==                                                                       |
| **明显微小错报临界值**            | 1. 明显微小的错报，无论单独或汇总，从金额、性质或环境看均明显微不足道<br>2. 低于临界值的错报**可以不累积**<br>3. “明显微小”≠“不重大”<br>4. 不确定是否明显微小，则不能认为是明显微小 | ==以前年度识别出的错报数量和金额、重大错报风险评估结果、治理层/管理层对沟通错报的期望、财务指标是否勉强达监管/投资者期望==                                                 |

### 2. 财务报表整体重要性的常用基准示例
| 被审计单位的情况 | 可能选择的基准 |
|------------------|----------------|

### 3. 实际执行的重要性：百分比确定
- 通常为财务报表整体重要性的 **<u>50% ~ 75%</u>**
  - **较高比例（接近75%）**：连续审计且以前年度调整较少、项目总体风险**低到中等**、以前经验表明内部控制运行**有效**
  - **较低比例（接近50%）**：首次接受委托、连续审计且以前年度调整较多、项目总体风险**较高**、存在或预期存在值得关注的内部控制缺陷

### 4. 实务运用
注册会计师选取超过实际执行重要性的项目实施进一步程序，但**不能完全忽略**低于实际执行重要性的项目，需考虑：
1. ！ 单个金额低于实际执行重要性的项目**汇总后可能金额重大**；
2. ！  存在**低估风险**的项目，不能仅因金额低而不实施程序；
3. ！  识别出存在**舞弊风险**的项目，不能仅因金额低而不实施程序。

## （二）核心命题角度与易错点
### 命题角度1：固有风险的概念要点
- 坑1：固有风险因素**<u>既可能是定性的，也可能是定量的</u>**。
- 坑2：固有风险因素影响**<u>错报发生的可能性</u>**以及错报发生时**<u>可能的严重程度</u>**。
- 坑3：固有风险因素包括：**<u>复杂性、主观性、变化、不确定性、管理层偏向</u>**和其他因素。
- 坑4：固有风险因素之间的**<u>相互关系</u>**会影响风险评估。
- 坑5：注册会计师应依据固有风险识别**<u>认定层次的重大错报风险</u>**，并将其确定为**<u>相关认定</u>**。

### 命题角度2：风险评估程序的概念要点
- 坑1：穿行测试用于了解内部控制，获取的是**<u>控制设计合理性和是否得到执行</u>**的证据，**无法获取控制运行有效性**的证据；不能依据穿行测试结果直接缩小实质性程序范围。
- 坑2：项目合伙人应当参与**<u>项目组内部讨论</u>**；对未参与讨论的成员，合伙人应确定向其通报的事项。
- 坑3：
  - 了解**经营活动**：识别主要**<u>交易类别、重要账户余额和列报</u>**；
  - 了解**投资活动**：关注**<u>经营策略和方向的重大变化</u>**；
  - 了解**筹资活动**：评估**<u>融资压力和持续经营能力</u>**。
