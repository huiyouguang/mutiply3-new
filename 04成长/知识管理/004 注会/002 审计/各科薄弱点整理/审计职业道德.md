# 会计师事务所业务质量管理

# 【知识点116】质量管理必背制度
- [x] 1. 会计师事务所应当在全所范围内<u>统一设计、实施和运</u>行质量管理体系 ✅ 2026-04-06
- [x] ！ 2. 会计师事务所在全所范围内形成一种**质量至上**的文化，树立质量意识 ✅ 2026-04-06
- [x] 3. 会计师事务所**主要负责人（如首席合伙人、主任会计师或者同等职位的人员）**应当对**质量管理体系承担最终责任** ✅ 2026-04-06
- [x] 4. 应当建立以质量为导向的晋升机制，**不得以承接和执行业务的收入或利润作为晋升合伙人的首要指标**；应当针对合伙人的晋升建立和实施**质量一票否决制度** ✅ 2026-04-06
- [x] 5. 会计师事务所应当在全所范围内统一进行合伙人考核和收益分配。**不得以承接和执行业务的收入或利润作为首要指标**，**不应直接或变相以分所、部门、合伙人所在团队作为利润中心进行收益分配** ✅ 2026-04-06
- [x] 6. **至少每年一次**向所有需要按照相关职业道德要求保持独立性的人员获取其**已遵守独立性要求的书面确认** ✅ 2026-04-06
- [ ] 7. 应当对关键审计合伙人的轮换情况进行实时监控，通过建立关键审计合伙人服务年限清单等方式，管理关键审计合伙人相关信息，每年对轮换情况实施复核，并**在全所范围内统一进行轮换**
- [x] ！ 8.对承接上市实体审计业务的每个项目合伙人，**检查周期最长不得超过三年** ✅ 2026-04-06
- [ ] 9. 会计师事务所主要负责人应当代表会计师事务所对质量管理体系进行评价，这种评价应当以某一时点为基准，并且**应当至少每年一次**
- [x] ！ 10. 所执行的项目在意见分歧解决后、项目质量复核完成后才能出具业务报告 ✅ 2026-04-06
- [x] ！ 11. 会计师事务所领导层应当建立健全一体化管理制度体系并确保有效实施，在合伙协议中明确一体化管理要求。会计师事务所**主要负责人**应当对**一体化管理负主要责任**。 ✅ 2026-04-06
- [ ] 12. 会计师事务所应当实施**矩阵式管理**，即结合所服务客户的行业特点和业务性质，以及本会计师事务所（或分部）的地域分布，对业务团队进行专业化设置，以团队专业能力的匹配度为依据分派业务。

# 【知识点117】项目质量复核
- [x] 1. 项目质量复核人员的资质要求 ✅ 2026-04-06
项目质量复核人员应当**独立于**执行业务的项目组，因此项目合伙人和项目组其他成员不得成为本项目的项目质量复核人员。
- [x] 2. 质量复核人员的客观性： ✅ 2026-04-06
  - [x] ！ （1）项目之间**交叉实施**项目质量复核，可能对客观性造成影响； ✅ 2026-04-06
  - [ ] ！ （2）某一项目的前任项目合伙人被委任为该项目的项目质量复核人员。这之间应当规定一段冷却期，冷却期**至少应当2年**。
- [x] 3. 项目质量复核的实施 ✅ 2026-04-06
> ！如果项目质量复核人员怀疑项目组作出的重大判断或据此得出的结论不恰当，应当告知项目合伙人。




# 【知识点118】项目合伙人、项目组、项目质量复核相关概念辨析 
  
| 相关概念   | 具体内容                                                                                                                       |
| ------ | -------------------------------------------------------------------------------------------------------------------------- |
| 项目合伙人  | 会计师事务所中负责某项业务及其执行，并代表会计师事务所在出具的报告上签字的合伙人                                                                                   |
| 项目组    | 执行某项业务的所有合伙人和员工，以及为该项业务实施程序的所有其他人员，**但<span class="cloze-span">不包括外部专家和为项目组提供直接协助的内部审计人员</span>**                          |
| 项目质量复核 | 项目质量复核：**在报告日或报告日之前**，项目质量复核人员对项目组作出的重大判断及据此得出的结论作出的客观评价<br>项目质量复核人员：会计师事务所中实施项目质量复核的合伙人或其他类似职位的人员，或由会计师事务所委派实施项目质量复核的外部人员 |

# 【知识点126-127】经济利益的定义与禁止情形
- [x] ## 一、经济利益的定义（知识点126） ✅ 2026-04-06

| 类别         | 要点                                                                                                                                                    |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| **直接经济利益** | 1. 个人或实体直接拥有并控制的经济利益（包括授权他人管理的经济利益）<br>2. 通过集合投资工具、信托、实体或合伙组织、或第三方而实质拥有的经济利益，并且**有能力控制这些投资工具，或影响其投资决策**<br>3. 常见类型：证券或其他参与权，如股票、债券、认沽权、认购权、期权、权证和卖空权等 |
| **间接经济利益** | 个人或实体通过集合投资工具、信托、实体或合伙组织、或第三方而实质拥有的经济利益，但**没有能力控制这些投资工具，或影响其投资决策**                                                                                    |
| **举例**     | 投资经理投资了共同基金，而这些共同基金投资了一揽子基础金融产品：<br>→ 共同基金属于**直接经济利益**<br>→ 基础金融产品属于**间接经济利益**                                                                        |

---
- [x] ## 二、不得拥有的经济利益（知识点127） ✅ 2026-04-06
### 1. 无防范措施可降低不利影响的情形
| 情形                    | 禁止拥有直接或重大间接经济利益的主体                                                                                    |
| --------------------- | ----------------------------------------------------------------------------------------------------- |
| **在审计客户中拥有经济利益**      | 1. 会计师事务所、审计项目团队成员及其主要近亲属<br>2. 与执行审计业务的项目合伙人同处一个分部的其他合伙人及其主要近亲属<br>3. 为审计客户提供非审计服务的其他合伙人、管理人员及其主要近亲属 |
| **在控制审计客户的实体中拥有经济利益** | 当一个实体在审计客户中拥有控制性的权益，并且审计客户对该实体重要时：<br>会计师事务所、审计项目团队成员或其主要近亲属在该实体中拥有                                   |

### 2. 例外提示
上述规定存在例外情况：如果与执行审计业务的项目合伙人同处一个分部的其他合伙人的主要近亲属，或者为审计客户提供非审计服务的其他合伙人或管理人员的主要近亲属同时满足法定条件，则该主要近亲属可以在审计客户中拥有直接经济利益或重大间接经济利益。

### 💡 核心考点速记
- [x] **直接/间接经济利益核心区分**：**是否能控制/影响投资决策** ✅ 2026-04-06
- [x] **禁止持有红线**：审计相关主体（事务所、项目组、主要近亲属、同分部合伙人、非审计服务人员）**绝对禁止**在审计客户中拥有直接/重大间接经济利益，无防范措施可豁免（除法定例外） ✅ 2026-04-06
- [x] **间接利益判定**：通过无控制权的集合工具持有的底层资产，属于间接利益，仅重大时禁止 ✅ 2026-04-06

## 四、知识点137：为审计客户提供非鉴证服务

### 1. 承担管理层职责
- [ ] ！ 核心要求：会计师事务所承担审计客户的管理层职责，将因自身利益、自我评价、密切关系、过度推介对独立性产生**非常严重的不利影响**，**导致没有防范措施能够将其降低至可接受的水平**
- [ ] 例外：仅向审计客户提供意见和建议以协助其管理层履行职责，通常不视为承担管理层职责
- [ ] 管理层职责示例：
  1. 制定政策和战略方针
  2. 招聘或解雇员工
  3. 指导员工与工作有关的行动并对其行动负责
  4. 对交易进行授权
  5. 控制或管理银行账户或投资
  6. 确定采纳会计师事务所或其他第三方提出的建议
  7. 代表管理层向治理层报告
  8. 负责按照适用的财务报告编制基础编制财务报表
  9. 负责设计、执行、监督和维护内部控制

### 2. 各类非鉴证服务的具体要求
- [ ] 会计和记账服务
  - [ ] 要求：提供以下会计和记账服务活动，不对独立性产生不利影响：
    1. 沟通审计相关的事项
    2. 提供会计咨询服务（如为了遵守集团会计政策从企业会计准则转换为国际财务报告准则提供咨询服务）
    3. 日常性或机械性的会计和记账服务（如根据客户批准的试算平衡表中的信息编制财务报表，根据客户批准的记录编制相关财务报表附注）
- [ ] 评估服务
  - [ ] 要求：会计师事务所不得向审计客户提供下列评估服务：**评估结果涉及高度的主观性，且评估服务对被审计财务报表具有重大影响**
- [ ] 内部审计服务
  - [ ] 要求：在审计客户属于公众利益实体的情况下，会计师事务所不得提供与下列方面有关的内部审计服务：
    1. 财务报告内部控制的组成部分
    2. 财务会计系统
    3. 单独或累积起来对被审计财务报表具有重大影响的金额或披露
- [ ] 信息技术系统服务
  - [ ] 要求：如果存在下列情形之一时，会计师事务所不得向属于公众利益实体的审计客户提供或设计与实施信息技术系统相关的服务：
    1. 信息技术系统构成财务报告内部控制的重要组成部分
    2. 信息技术系统生成的信息对会计记录或被审计财务报表影响重大
- [ ] 法律服务
  - [ ] 要求：提供下列法律服务，将产生非常严重的不利影响，导致没有防范措施能够将其降低至可接受的水平：
    1. 在审计客户解决纠纷或进行法律诉讼时，如果会计师事务所人员担任辩护人，并且纠纷或法律诉讼所涉金额对被审计财务报表有重大影响
    2. 会计师事务所人员不得担任审计客户的首席法律顾问
- [ ] 招聘服务
  - [ ] 要求：在向审计客户提供招聘服务时，会计师事务所不得代表客户与应聘者进行谈判；如果属于审计客户拟招聘董事、高级管理人员，或所处职位能够对客户会计记录或被审计财务报表的编制施加重大影响的高级管理人员，会计师事务所不得提供：寻找或筛选候选人；对候选人实施背景调查
- [ ] 财务服务
