## 内部研发的无形资产
1. 1.科目；研发支出-费用化支出/资本化支出
2. 结转科目：管理费用/无形资产
3. 无法确认规则的参造一下

4. 序号	业务场景	处理原则 / 科目
5. ①	无形资产：无法区分研究阶段与开发阶段的支出	==费用化（计入当期损益）==
6. ②	无形资产：外购房屋价款中土地使用权与建筑物价值无法分配	==全部作为固定资产核算==
7. ③	无形资产：无法合理估计使用寿命	==作为使用寿命不确定的无形资产（不计摊销，至少每年减值测试）==
8. ④	无形资产：无法可靠确定预期消耗方式	采用==直线法==摊销
9. ！ ⑤	可收回金额：无法估计公允价值减处置费用后的净额	以==预计未来现金流量的现值==作为可收回金额
10. ！ ⑥	会计政策 / 估计变更：难以区分两者	作为==会计估计变更==处理
11. ⑦	收入：无法在未履行与已履行履约义务间区分支出	计入==当期损益==
12. ⑧	收入：质量保证同时包含保证类与服务类且无法区分	一起作为==单项履约==义务
13. ！ ⑨	政府补助：同时包含与资产相关和与收益相关且无法区分	作为与==收益相关的==政府补助
14. ！ ⑩	租赁：难以确定供应方是否拥有实质性替换权	视为==没有实质性替换权==（认定为租赁）
15. ! ⑪	租赁：无法确定租赁内含利率	使用承租人==增量借款利率==计算租赁付款额现值

	1. 投资性成本转公允，按==会计政策变更==处理

```
    投资性房地产累计折旧
    投资性房地产累计摊销投资性房地产减值准备以前年度损益调整
    
    
    
    借：以前年度损益调整
    贷：盈余公积利润分配——未分配利润
```

## 三、长期股权投资后续计量（成本法 vs 权益法）

| 被投资方变动       | 成本法                      | 权益法                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ------------ | ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **宣告发放现金股利** | 借：应收股利<br>贷：==投资收益==     | 借：应收股利<br>贷：==长期股权投资——损益调整==                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **实现利润**     | 无                        | 借：长期股权投资——损益调整（调整后净利润×持股比例）<br>贷：投资收益<br><br>**净利润调整**：<br>① 投资当日：<br>  - 存货增值：调整后利润 = 调整前利润 -（投资日公允 - 账面）×当年出售比例<br>  - 固定资产增值：调整后利润 = 调整前利润 -（投资日公允 - 账面）/尚可使用年限 × 当年折旧月数/12<br>② 内部交易（含以非货币性资产设立立司）：<br>  - 存货：<br>    - 交易当年实现：调整后利润 = 调整前利润 -（售价 - 账面） +（售价 - 账面）×==当年出售比例<br>    - 以后年度实现：调整后利润 = 调整前利润 +==（售价 - 账面）×当年出售比例<br>  - 固定资产：<br>    - 交易当年实现：调整后利润 = 调整前利润 -（售价 - 账面） +（售价 - 账面）/交易后剩余使用年限 × 当年折旧月数/12<br>    - 以后年度实现：调整后利润 = 调整前利润 +（售价 - 账面）/交易后剩余使用年限 × 当年折旧月数/12 |
| **发生减值**     | 借：资产减值损失<br>贷：长期股权投资减值准备 | 借：资产减值损失<br>贷：长期股权投资减值准备                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **发生超额亏损**   | 无                        | 按顺序处理：<br>① 冲减 长期股权投资的账面价值，直到减记为0；<br>② 冲减 长期应收款；<br>借：投资收益<br>贷：长期应收款<br>③ 确认 预计负债；<br>借：投资收益<br>贷：预计负债<br>④ 账外备查登记<br><br>**若后续被投资方净资产增加**：<br>① 按当期享有被投资方净资产增加的份额确认长期股权投资；<br>② 确认备查登记的未确认亏损；<br>借：投资收益<br>贷：长期股权投资——损益调整<br>③ 冲减预计负债<br>借：预计负债<br>贷：长期股权投资——损益调整<br>④ 恢复长期应收款<br>借：长期应收款<br>贷：长期股权投资——损益调整                                                                                                                                                                          |
| **其他综合收益变动** | 无                        | 借：长期股权投资——其他综合收益<br>贷：其他综合收益                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **其他权益变动**   | 无                        | 借：长期股权投资——其他权益变动<br>贷：资本公积——其他资本公积                                                                                                                                                                                                                                                                                                                                                                                                                                                      |

### 备注
- 涉及冲减资本公积时，假设资本公积金额足够冲减，用“付出对价”代替投资方付出的资产对应的科目。
- 同一控制下企业合并强调“最终控制方合并报表中的账面价值份额”，非同一控制下强调“公允价值”。
- 权益法核心是**随被投资方净资产变动而调整**，成本法核心是**宣告股利时确认收益**。

# 股份支付


权益工具无法确定公允价值
1. 内在价值=股份公允价孩子-股份支付应支付的行权价格


加速可行权的形式


## 修改股份支付条件
1. 市场条件和非市场条件
	1. （1）市场条件和非可行权条件是否得到满足，不影响企业对预计可行权情况的估计。
如果最终未满足市场条件导致不能行权，对已经认定的费用或成本不作转回处理
（2）对于可行权条件为业绩条件的股份支付，企业在授予日确定权益工具的公允价值时，应考虑市场条件的影响。只要职工满足了其他所有非市场条件，企业就应当确认已取得的服务。如果最终未满足非市场条件导致不能行权，需要对已经认定的费用或成本作转回处理。

> # （三）取消、结算和作废
> ## 1．取消或结算
> ### （1）加速可行权处理
> 将取消或结算**作为加速可行权处理**，立即确认原本应在剩余等待期内确认的金额。
> 
> 会计分录：
> ```
> 借：管理费用等
> 　贷：资本公积——其他资本公积（剩余等待期应确认的金额）
> ```
> 
> ### （2）支付款项回购处理
> 取消或结算时支付给职工的所有款项，**均作为权益回购处理**；
> 回购支付金额＞该权益工具回购日公允价值的部分，计入当期费用。
> 
> 会计分录：
> ```
> 借：资本公积——其他资本公积（等待期内累计确认金额）
> 　　管理费用等（差额）
> 　贷：银行存款（实际支付价款）
> ```
>

> ## 补充说明
> 1. 职工因激励计划行权价较高**自愿退出股权激励计划**，属于股权激励计划取消；企业应按**加速行权**处理，剩余等待期应确认金额立即计入当期损益，同时确认资本公积。
> 2. 职工/其他方有权选择满足非可行权条件，但等待期内**未满足非可行权条件**的，企业将其作为**授予权益工具的取消**处理。
