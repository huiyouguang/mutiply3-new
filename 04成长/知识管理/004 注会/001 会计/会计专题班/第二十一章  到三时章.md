
# 债务重组




##  债务重组仅为金融资产

1. 债权人
	1. 借：收到资产
		1. 坏账准备
	2. 贷：应收账款
	3. 银行存款
	4. 投资收益（放弃债债权公允-账面）

债务人
借：应付账款
		贷：付出对价
		投资收益

## 债务资产为非金融资产
1. 债权人计算投资收益
2. 债务人计算**其他收益**


债务人偿还是非金融资产+金融资产
1. 债权人计算投资收益
2. 债务人计算**其他收益**

# 外币折算



##  考点 1 重要 汇兑差额
填写下列以外币购入或偿还的交易事项相关的外币折算业务

| 项目        | 货币性或非货币性 | 期末是否产生汇兑差额 | 期末汇兑差额计入什么科目                           |
| --------- | -------- | ---------- | -------------------------------------- |
| 存货        | 非货币性     | 是          | ==资产减值损失==                             |
| 无形资产      | 非货币性     | 否          | 无                                      |
| 应收账款      | 货币性      | 是          | 财务费用                                   |
| 预收账款      | 非货币性     | 否          | 无                                      |
| 应付账款      | 货币性      | 是          | 财务费用                                   |
| 预付账款      | 非货币性     | 否          | 无                                      |
| 合同资产      | 非货币性     | 否          | 无                                      |
| 合同负债      | 非货币性     | 否          | 无                                      |
| 长期借款      | 货币性      | 是          | 财务费用<br>专门借款(用于建造固定资产)资本化期间的汇兑差额计入在建工程 |
| 应付债券      | 货币性      | 是          | 财务费用                                   |
| 债权投资      | 货币性      | 是          | 财务费用                                   |
| 其他债权投资    | 货币性      | 是          | 财务费用                                   |
| 其他权益工具投资  | 非货币性     | 是          | ==其他综合收益==                             |
| 交易性金融资产   | 非货币性     | 是          | 公允价值变动损益                               |
| 长期股权投资    | 非货币性     | 否          | 无                                      |
| 接受外币资本性投资 | 货币性      | 否          | 无                                      |

---

### 核心规律总结
1.  **货币性项目**（如应收/应付账款、长短期借款、债权投资等）：期末**必然产生汇兑差额**，一般计入**财务费用**（长期借款资本化期间除外）。
2.  **非货币性项目**：
    -   仅**存货、其他权益工具投资、交易性金融资产**会产生汇兑差额，分别计入**资产减值损失、其他综合收益、公允价值变动损益**；
    -   其余非货币性项目（如无形资产、预收/预付账款、合同资产/负债、长期股权投资等）**不产生汇兑差额**。
3.  **特殊例外**：
    -   专门借款（外币）在固定资产资本化期间的汇兑差额，计入**在建工程**而非财务费用；
    -   接受外币资本性投资（货币性但属于权益性交易），不产生汇兑差额。



##  考点 4 重要：外币报表折算
### 一、外币报表折算方法
| 项目分类    | 折算规则                                             |
| ------- | ------------------------------------------------ |
| 收入费用项目  | 按**交易发生日的即期汇率或近似汇率**折算                           |
| 资产项目    | 按**资产负债表日即期汇率**折算                                |
| 负债项目    | 按**资产负债表日即期汇率**折算                                |
| 所有者权益项目 | 除**未分配利润**项目外，其他项目按**发生时的即期汇率**折算                |
| 折算差额    | 外币财务报表折算差额在**其他综合收益**中列示，处置境外经营时转入**当期损益（投资收益）** |

---

### 二、外币报表折算差额的列示方法
| 承担主体     | 列示科目       |
| -------- | ---------- |
| 母公司承担部分  |其他综合收益|
| 少数股东承担部分 | 少数股东权益     |

---

### 三、实质上构成对境外经营净投资的外币货币性项目
| 报表类型   |   | 汇兑差额计入科目 |
|------------|---|------------------|
| 个别报表   |   | 财务费用         |
| 合并报表   |   | 其他综合收益     |

---

### 核心记忆点💡
1.  **资产负债类**：统一使用**资产负债表日即期汇率**折算。
2.  **收入费用类**：使用**交易发生日汇率或近似汇率**折算。
3.  **所有者权益**：实收资本、资本公积等项目按**发生时的即期汇率**折算，未分配利润为倒挤数。
4.  **折算差额归属**：母公司承担的部分计入**其他综合收益**，少数股东承担的部分计入**少数股东权益**；处置境外经营时，其他综合收益需转入**当期损益（投资收益）**。
5.  **净投资外币项目**：个别报表层面汇兑差额计入**财务费用**，合并报表层面计入**其他综合收益**。
