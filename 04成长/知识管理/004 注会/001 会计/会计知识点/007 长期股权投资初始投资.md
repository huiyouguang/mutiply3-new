# 长期股权投资的确认与初始计量

---

## 一、长期股权投资的确认
### （一）基本确认原则
1. **确认时点**：在被投资单位股权取得[[控制权]]时确认
2. **核心概念**：
   - 非同控下：[[购买方]]、[[|购买日]]≈
   - 同控下：[[合并方]]、[[合并日]]
3. **特殊情形处理**：
   - 多次交易合并：每个交易日确认<span class="cloze-span">单项投资</span>
   - 分步合并：以<span class="cloze-span">控制权转移日</span>作为购买日
   - 联营/合营企业投资参照<span class="cloze-span">子公司确认条件</span>

---

### （二）控制权判断标准（[[../inbox/合并日|合并日]]/购买日）#
需同时满足：
- 企业合并协议获<span class="cloze-span">股东大会</span>批准
- 完成相关<span class="cloze-span">审批程序</span>
- 财产权<span class="cloze-span">转移手续</span>办理完毕º
- 支付对价符合<span class="cloze-span">协议约定</span>
- 实际取得被投资单位<span class="cloze-span">财务/经营控制权
   
   
   </span>
   
   对于投资的**初始确认**，若合同**明确**约定认缴出资的**时间和金额**，且投资方按**认缴比例享有**相应的股东权益，则投资方应确认一项**金融负债**及相应的**资产**；
   若合同没有明确约定，则属于一项未来的出资承诺，**不确认**<span class="cloze-span"><font color="#00b050">金融负债及相应的资产</font>。</span>
	 ### 3.投资者投入的长期股权投资
    应当按照投资合同或协议**约定的价值**作为**初始投资成本**，但合同或协议约定价值**不公允**的除外

## 二、初始计量原则
### （一）基本分类

| 投资类型        | 核算方法 | 适用情形                                             |
| ----------- | ---- | ------------------------------------------------ |
| 对[[子公司]]投资  | 成本法  | 控制（持股＞<span class="cloze-span">50%</span>）       |
| 对[[合营企业]]投资 | 权益法  | 共同控制                                             |
| 对[[联营企业]]投资 | 权益法  | 重大影响（<span class="cloze-span">20%≤持股≤50%</span>） |
### （二）对联营企业、合营企业投资的初始计量
- -企业在支付对价取得长期股权投资时，对于实际支付  的价款中包含的对方**已经宣告**但**尚未发放**的**现金股利或利润**，应作为<span class="cloze-span">应收股利</span><处理。
- 2.以发行权益性证券方式取得的长期股权投资
	- 以**发行权益性证券**取得长期股权投资的，应当按照所发行证券的**公允价值**作为**初始投资成本**。
	- 为发行**权益性工具**支付给有关证券承销机构等的手续费、佣金等与权益性工具发行直接相关的**费用**，**不构成**取得长期股权投资的**成本**。该部分费用应自所发行证券的**溢价发行收入**中**扣除**，溢价收入不足冲减的，应依次冲减盈余公积和未分配利润。如
- 3.投资者投入的长期股权投资
		- 应当按照投资合同或协议约定的价值作为初始投资成本，但合同或协议约定价值不公允的除外

## 三、同一控制下企业合并处理
### （一）一次交易处理规则
1. **初始成本确认**：
   - =被合并方在[[最终控制方]]合并报表中的<span class="cloze-span">净资产账面价值份额</span>
   - 包含最终控制方收购被合并方时形成的<span class="cloze-span">商誉</span>
2. **费用处理**：
   - 中介费用：<span class="cloze-span">管理费用</span>
   - 权益证券发行费：冲减<span class="cloze-span">溢价</span>或计入负债
   - 债务证券工具：
     计入**债务性工具**的**初始确认金额**。  
		<span style="background:#fff88f">	借：<span class="cloze-span">应付债券——利息调整</span>  </span>
| <span style="background:#fff88f">　　贷：<span class="cloze-span">银行存款</span></span> |

2. **改制企业处理**：
   - 按改制后<span class="cloze-span">评估值</span>调整账面价值确认
3. **多层级合并**：
   - 以[[最终控制方合并报表]]<span class="cloze-span">账面价值</span>为基准

## 四、非同一控制下企业合并处理
### （一）核心计量原则
1. **初始成本确认**：
   =支付对价的<span class="cloze-span">公允价值之和</span>
   （含资产、负债、权益性证券的公允价值）
2. **损益确认**：
   - 存货：确认<span class="cloze-span">销售收入</span>
   - 固定资产：确认<span class="cloze-span">处置损益</span>
   - 长期股权投资：确认<span class="cloze-span">处置损益</span>
  - 营业外收入：合并成本小于公允价值比例
---

### （二）费用处理
1. 中介费用：计入<span class="cloze-span">当期管理费用</span>
2. 证券发行费：
   - 债务工具：计入<span class="cloze-span">初始负债金额</span>
   - 权益工具：冲减<span class="cloze-span">股本溢价</span>

---

## 五、分步实现企业合并
### （一）同一控制下分步合并
1. **个别报表处理**：
   - 初始成本=合并日被合并方在[[最终控制方合并报表]]中的<span class="cloze-span">净资产份额</span>
   - 差额调整<span class="cloze-span">资本公积/留存收益</span>
2. **原持有股权**：
   - 其他综合收益<span class="cloze-span">暂不处理</span>，待处置时结转

---

### （二）非同一控制下分步合并
按[[CAS 20]]处理：
1. 原权益法核算股权按<span class="cloze-span">公允价值</span>重新计量
2. 原金融资产按<span class="cloze-span">公允价值</span>重新计量
3. 合并成本=原持有股权<span class="cloze-span">公允价值</span>+新增投资成本

---

## 六、不形成控股合并的特殊处理
| 取得方式           | 初始计量规则                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| 现金支付           | 实际支付价款+<span class="cloze-span">直接相关费用</span>（不含已宣告股利） |
| 发行权益性证券      | 证券<span class="cloze-span">公允价值</span>确认成本，发行费用冲减股本溢价  |
| 债务重组取得        | 按[[CAS 12]]以<span class="cloze-span">公允价值</span>确认                 |
| 非货币性资产交换    | 按[[CAS 7]]以<span class="cloze-span">换出资产公允价值/账面价值</span>确认  |

#### **案例解析（2×14-2×15年）**  

| 时间       | 事件概要                                                                 |  
|------------|--------------------------------------------------------------------------|  
| 2×14.1.1   | 取得20%股权（[[权益法]]核算），支付<span class="cloze-span">3800万</span> |  
| 2×14年度   | 乙公司净利润<span class="cloze-span">2000万</span>                        |  
| 2×15.1.1   | 追加40%股权（[[成本法]]核算），支付<span class="cloze-span">6000万</span> |  

**📝 分步处理**  

- **2×14年初始投资**  
- 借：长期股权投资——投资成本  38,000,000  
- 贷：银行存款                          38,000,000  
- ▶ 初始成本 > 份额（16000万×20%=<span class="cloze-span">3200万</span>），差额<span class="cloze-span">600万</span>不调整  

- **2×14年确认投资收益**  
- 借：长期股权投资——损益调整  4,000,000（2000万×20%）  
- 贷：投资收益                        4,000,000  
- ▶ 2×14年末账面价值 = <span class="cloze-span">3800万+400万=4200万</span>  

- **2×15年合并日处理**  
   - 合并成本 = 18000万×60% = <span class="cloze-span">10,800万</span>  
   - 合并对价总和 = 4200万 + 6000万 = <span class="cloze-span">10,200万</span>  
   - 差额 = 10,800万 - 10,200万 = <span class="cloze-span">600万</span>  

- 借：长期股权投资             108,000,000  
- 贷：长期股权投资——投资成本     38,000,000  
    - 长期股权投资——损益调整      4,000,000  
    - 银行存款                  60,000,000  
    - 资本公积——资本溢价           6,000,000

#### **[[核心要点]]总结**  
✅ 必须穿透至[[最终控制方合并报表]]数据  
✅ 原权益法核算的<span class="cloze-span">账面价值</span>需追溯调整  
✅ 差额仅调整<span class="cloze-span">资本公积/留存收益</span>（不确认损益）  
❌ 与[[非同一控制下企业合并]]的本质差异：  

| 对比项       |   | 同一控制下                  | 非同一控制下              |
|--------------|---|----------------------------|--------------------------|
| 初始成本基准 |   | 账面价值                   | 公允价值                 |
| 合并前收益   |   | 追溯调整                   | 重新计量                 |
| 差额处理     |   | 权益调整                   | 确认商誉/损益           |

- 【例题·计算分析题】甲公司<span style="background:#fff88f">2×24年7月1日</span>以银行存款2 700万元<font color="#00b050">自母公司（A公司）</font>取得乙公司<font color="#00b050">60％股权。</font>该股权系A公司于2×21年6月自外部公开市场购入，为进行该项交易，<font color="#00b050">甲公司另支付有关审计等中介机构费用120万元。不考虑其他因素。  </font>
- 假定一：A公司在2×21年6月自外部公开市场购入乙公司60％股权时在合并报表中未确认商誉。<span style="background:#fff88f">2×24年7月1日</span>，乙<font color="#00b050">公司个别财务报表中净资产账面价值为4 600万元（等于乙公司所有者权益在最终控制方A公司合并财务报表中的账面价值）。  </font>

- 假定二：A公司在2×21年6月自外部公开市场购入乙公司<span style="background:#fff88f">60％股权时在合并报表中确认商誉800万</span>元。2×24年7月1日，乙公司个别财务报表中净资产账面价值为4 600万元。以A公司取得该股权时乙公司可辨认净资产公允价值为基础持续计算的可辨认净资产价值为4 800万元。  
- 要求<span style="background:#fff88f">：分别计算甲公司应确认对乙公司股权投资的初始投资成</span>本，并编制相关的会计分录。  
- **『正确答案』**  

- **假定一：**资料中已知合并日乙公司个别报表中的净资产账面价值4 600万元（等于乙公司所有者权益在最终控制方A公司合并财务报表中的账面价值），则甲公司应确认对乙公司股权投资的初始投资成本＝4 600×60％＝2 760（万元）。  
	- 借：长期股权投资　2 760  
	- 贷：银行存款　　  　2 700  
	- 资本公积　     　60  
	- 借：管理费用　　　  120  
	- 贷：银行存款　　　   120  
- **假定二：**  
- 资料中既已知合并日乙公司个别报表中的净资产账面价值为4 600万元，又已知以A公司取得该股权时乙公司可辨认净资产公允价值为基础持续计算的乙公司可辨认净资产价值为4 800万元，则甲公司应确认对乙公司股权投资的初始投资成本＝4 800×60％＋800＝3 680（万元）。  
	- 借：长期股权投资　3 680  
	- 贷：银行存款　　　 2 700  
	- 资本公积　　　　 980  
	- 借：管理费用　　　 120  
	- 贷：银行存款　　　　 120

- 【例题·计算分析题】A公司为母公司，其子公司包括甲公司、乙公司。有关投资业务如下：
