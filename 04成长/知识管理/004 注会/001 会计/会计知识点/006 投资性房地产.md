# 投资性房地产

## 1 一、定义
**投资性房地产**是指为赚取租金或资本增值，或两者兼有而持有的房地产。

---

## 2 二、范围

### 2.1 属于投资性房地产的项目

| 类别               | 内容                                |
| ---------------- | --------------------------------- |
| 已出租的土地使用权        | 经营租赁方式出租                          |
| 持有并准备增值后转让的土地使用权 | 较少见                               |
| 已出租的建筑物          | 企业拥有产权并以经营租赁方式出租的建筑物（含在建或拟建用于出租的） |

### 2.2 不属于投资性房地产的项目

1. <span class="cloze-span">自用房地产</span>  
   - 企业自用的厂房、办公楼等（作为固定资产/无形资产）
2. <span class="cloze-span">作为存货的房地产</span>  
   - 房地产开发企业销售的商品房或为销售而开发的土地

## 3 九、税务处理要点
| 税种    | 处理规则                              |
| ----- | --------------------------------- |
| 房产税   | 按租金收入从租计征（12%税率），空置期间需按原值计征（1.2%） |
| 土地增值税 | 转让增值额超扣除项目20%部分需缴纳（30%-60%累进税率）   |
| 所得税   | 公允价值变动损益不计入应纳税所得额（成本模式下已提折旧可税前扣除） |
