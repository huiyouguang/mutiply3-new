1. **函证对象**
    - **银行**
        - 涵盖银行存款、借款和其他重要信息，包括零余额、注销情况。
        - 豁免情况：对财务报表不重要且重大错报风险很低，<span style="background:#fdbfff">需同时满足。</span>
        - 通常情况应当函证。
    - **应收账款**
        - 豁免情况：对财务报表不重要<span class="cloze-span">或</span>函证很可能无效，<span style="background:#fdbfff">非同时满足。</span>
        - 通常情况应当函证。
    1. **函证截止日**
        - 资产负债表日
            - 资产负债表日后适当时间，前提是重大错报风险评估为低水平。
        - 资产负债表日前
            - 日前进行函证。
            - 截止日至资产负债表日执行实质性程序。
2. **函证实施 - 发出前控制**
    - 直接发出、直接收回。
    - 核对信息，包括函证内容、被询证者、地址。
3. **函证实施 - 发出和评价**
    - **邮寄**
        - 独立寄发，验证评价，涉及原件、地址、邮戳、印章。
    - **跟函**
        - 亲自控制，需了解流程、确认人员、观察处理流程、创造安全环境。
    - **电子**
        - 风险较高，需电话确认，必要时回复原件。
    1. **函证不符处理 - 积极函证**
        - 再积极一次，采用替代审计程序。
        - 若错误，可直接替代程序或改发消极函证。
    2. **函证不符处理结果**
        - 调查不一致后，判断不存在错报或存在错报。 


##例题
(2019年真题
】ABC会计师事务所的A注册会计师负责审计甲公司2018 年度财务报表。审计工作底稿中与函证相关的部分内容摘录如意:
对第(1)至(5)项,逐项指出A注册会计师的做法是否恰当。如不恰当,简要说明理由。
```
(1)A注册会计师对甲公司年内已注销的某人民币银行账户实施函证,银行表示无法就已注销账户回函。A注册会计师检查了该账户的注销证明原件,核对了亲自从中国人民银行获取的《已开立银行结算账户清单》中的相关信息,结果满意。
```
- [x] π✅2025-05-21/22:34 
### 考点总结
1. **函证的对象**
    - 注册会计师需对银行存款、借款及与金融机构往来的其他重要信息实施函证。函证对象包含零余额账户和本期内注销的账户 。
    - 银行对账单、银行存款余额调节表等原始凭证不能替代函证程序 。
    - 除关注银行存款信息，还应留意银行借款及其他重要信息（如抵押资产等 ）。
2. **函证的实施 - 发出和评价**：若未得到被询证者回应，注册会计师要实施替代审计程序 。

### 答题思路
1. 对于已注销的银行账户，注册会计师应实施函证程序。 
	1. 其次,在银行无法回函的情况下,注册会计师应当实施替代程序,那么我们需要分析“检查了该账户的注销证明原件,核对了亲自从中国人民银行获取的《已开立银行结算账户清单》中的相关信息”的做法是否充分。检查注销证明原件,这个程序其实已经能够说明账户确实被注销了;而题干中注册会计师又加了第二道保险:获取其他证明信息,也就是从中国人民银行获取《已开立银行结算账户清单》,来确保银行账户真的被注销了。
2. 因此,题干中注册会计师对已注销账户实施函证的做法以及银行不回函时实施的替代程序都是恰当的。

```
(2)在实施应收账款函证程序时,A注册会计师将财务人员在发函信封上填写的客户地址与销售部门提供的客户清单中的地址进行核对后,亲自将询证函交予快递公司发出。
```
## 重点
### 考点总结
1. **函证实施 - 发出前控制 - 核对信息 - 地址**
    - **函证内容**：询证函中确认信息通常为被审计单位账簿记录信息（如银行存款对应银行对账单信息 ）。
    - **被询证者恰当性**：需具备知情、回函资格且保持客观性 。
    - **确定被询证者名称和地址的方式**：可通过拨打电话、查询网站信息、查阅合同信息、发票信息等途径 。回函地址是被询证者直接向注册会计师回函的地址 。
2. **答题思路**：当出现被审计单位内部提供信息（如销售部门提供的客户清单 ）相关表述时，注册会计师一般不应依赖，应寻求可靠性更高的外部信息，如合同信息、网站信息等。 

### 答案分析
认为“客户清单属于内部信息”，<span style="background:#fdbfff">不是用以验证发函地址准确性的适当证据，应当通过合同、公开网站等来源核对地址，所以答案不恰当 。 </span>

```
(3) 甲公司根据销售合同在发出商品时确认收入。客户乙公司回函确认金额小于函证金额,甲公司管理层解释系期未发出商品在途所致。A注册会计师检查了合同、出库单以及签收单等支持性文件,并与乙公司财务人员电话确认了相关信息,结果满意。
```

### 考点总结
1. **函证的实施 - 函证的发出和评价 - 不符处理**
    - 注册会计师有责任调查函证中的不符事项，以此判断是否存在错报 。
    - **不表明存在错报的情况**：<span style="background:#fdbfff">函证程序时间安排、计量或书写错误等 。</span>
    - **表明存在错报或潜在错报的情况**：需评价是否存在舞弊，以及内部控制是否存在缺陷 ，同时这些情况也为判断类似被询证者回函质量及类似账户回函质量提供依据 。
2. **答题思路**：当出现如“客户乙公司回函确认金额小于函证金额”这类函证回函不符事项时，<span style="background:#fdbfff">注册会计师要调查不符事项，思考所实施程序能否获取充分、适当的审计证据 。</span>
3. “A注册会计师检查了合同、出库单以及签收单等支持性文件,并与乙公司财务人员电话确认了相关信息”,通过这些审计程序,注册会计师可以充分验证甲公司对不符事项的解释“差异系期末发出商品在途所致”是真实无误的。

### 答案分析
由于出现函证回函不符事项时，<span style="background:#fdbfff">注册会计师调查不符事项是符合准则要求的</span>，所以答案恰当 。 

```
(4)A 注册会计师对应收账款余额实施了函证程序,有15家客户未回函。A注册会计师对其中14 家实施了替代程序,结果满意;对剩余一家的应收账款余额,因其小于明显微小错报的临界值,A注册会计师不再实施替代程序。
对所有的进行函证
```
### 考点总结
1. **函证的实施 - 函证的发出和评价 - 积极函证 - 替代审计程序**
    - 对于积极式函证，若在合理时间内未收到询证函回函，注册会计师应考虑必要时再次向被询证者寄发询证函。
    - 如果最终仍未能得到被询证者的回应，注册会计师应当实施替代审计程序。
    - 常见的不当情景包括注册会计师没收到回函就直接实施替代程序，或者没有收到积极函证回函就改发消极函证且未收到回函等情况。
2. **答题思路**
    - 一般情况下，题干未特别说明时，函证通常指积极函证。
    - 在审计答题中要揣摩出题人意图，如本题结合题干中“A注册会计师对小于明显微小错报临界值的未回函余额不再实施替代程序”这一关键信息，<span style="background:#fdbfff">可得出答案不恰当的结论，</span>因为对所有未回函的余额都应当实施替代程序，不应有例外。

### 答案分析
本题答案不恰当，<span style="background:#fdbfff">原因是注册会计师未对所有未回函的余额实施替代程序，不符合积极函证未收到回函时应采取的审计程序要求。</span>

```
(5)甲公司未对货到票未到的原材料进行暂估。A注册会计师从应付账款明细账中选取 90%的供应商实施函证程序,要求供应商在询证函中填列余额信息
```
不恰当。应从供应商清单中选取函证对象/从应付账款明细账中选取函证对象不足以应对低估风, 险。
### 考点总结
- **函证的对象**：结合采购与付款循环中应付账款的实质性程序，由于应付账款存在低估风险（“完整性”认定），注册会计师对应付账款实施函证程序时，需从非财务部门（如采购部门）获取适当的供应商清单，如本期采购清单、所有现存供应商名录等，从中选取样本进行测试。

### 答案分析
- **错误做法及原因**：甲公司未对货到票未到的原材料进行暂估入账，导致应付账款明细账中的供应商不完整。注册会计师仅在应付账款明细账中选取供应商实施函证程序，无法应对低估风险，因为未入账的原材料所对应的供应商可能不在明细账中。
- **正确做法**：<span style="background:#fdbfff">应从采购部门获取的供应商清单中选取函证对象，这样才能更全面地覆盖可能存在应付账款的供应商，有效应对应付账款的低估风险，以判断是否需要对财务报告作出审计调整。</span>

# 分析程序
[[_resources/主观题突破/c510ed6ef3f420d6edb26626150afa61_MD5.jpeg|Open: Pasted image 20250514212450.png]]
![[_resources/主观题突破/c510ed6ef3f420d6edb26626150afa61_MD5.jpeg]]
### 知识点总结
1. **分析程序分类及要点**
    - **风险评估程序（必做 ）**
        - **目的**：了解被审计单位及其环境，评估重大错报风险，关注异常变化 。
        - **程序**：包含“3 + 1” ，涉及合理预期值并与记录金额比较 。
        - **特点**：数据汇总性强，但证明力较弱，且不用于了解内控 。
    - **实质性分析程序（选做 ）**
        - **目的**：提高审计效率和效果 。
        - **程序**：可单独或结合细节测试使用 。
        - **考虑因素**
            - **认定的适用性**：适用于存在、发生、完整性（高估或低估 ）等认定 。
            - **数据的可靠性**：应当测试数据可靠性 。
            - **预期值的准确程度** 。
            - **可接受的差异额**：小于等于实际执行的重要性水平 ，需对差异额的全额进行调查 。 
    - **总体复核（必做 ）**
        - **目的**：检查审计结论与了解的情况是否一致 。
        - **时间**：在审计结束或临近结束时进行 。
        - **特点**：集中在报表层面开展 。 
## 例题
```
【2015年真题】甲公司是 ABC 会计师事务所的常年审计客户。A注册会计师负责审计甲公司 2014年度财务报表。审计工作底稿中与分析程序相关的部分内容摘录如下: 要求: 针对第(1)至第(5)项,逐项指出 A注册会计师的做法是否恰当。如不恰当,提出改进建议。
(1)甲公司所处行业2014年度市场需求显著下降。A注册会计师在实施风险评估分析程序时,以2013年财务报表已审数为预期值,将2014年财务报表中波动较大的项目评估为存在重大错报风险的领域。
```
### 考点总结
- **风险评估程序 - 合理预期值**：当分析程序用作风险评估程序时，注册会计师需要对关键的账户余额、趋势和财务比率关系等形成合理预期。
### 答案分析
- **不恰当原因**：2014年与2013年相比市场需求显著下降，这种情况下仍沿用之前预期值不合理，应依据2014年度的变化情况重新设定预期值，才能准确识别“变化”，实现风险评估程序中分析程序的目的。 
- 题目中强调了“甲公司所处行业2014年度市场需求显著下降”,说明2014年有变化,那么“2013年财务报表已审数”就不再是合理的预期值,应该以变化后的2014年度的情况作为预期值,作为比较的基准。
```
(2)A 注册会计师对营业收入实施实质性分析程序,将实际执行的重要性作为已记录金额与预期值之间可接受的差异额。
恰当
```
```
(3)甲公司的产量与生产工人工资之间存在稳定的预期关系。A注册会计师认为产量信息来自非财务部门,具有可靠性,在实施实质性分析程序时据以测算直接人工成本。

```
### 考点总结
1. **实质性分析程序 - 考虑因素 - 数据的可靠性**
    - **数据来源**：
        - **内部数据**：包括前期数据（可调整）、当期数据、预算或预测、非财务数据等 。
        - **外部数据**：如政府发布的信息、行业监管协会发布的信息、经济预测组织发布的行业业绩指标、公开出版的财务信息、证券交易所发布的信息等 。
    - **考量数据可靠性的方面**：
        - **可获得信息的来源**：外部独立来源数据越可靠，预期准确性越高，分析越有效 。
        - **可获得信息的可比性**：对于特殊产品销售，可能需要宽泛的行业数据进行补充 。
        - **可获得信息的性质和相关性**：例如预算作为预期的结果，而非将要达到的目标 。
        - **与信息编制相关的控制**：用以确保信息完整、准确和有效 。
        - **信息在本期或前期是否经过审计**：审计后的信息更可靠 。
2. **答题思路**：当将实质性程序用于分析程序（即实质性分析程序）时，需考虑数据的可靠性。<span style="background:#fdbfff">题目中数据来源是“非财务部门”，属于企业内部数据。</span>通常外部独立来源数据更可靠，<span style="background:#fdbfff">对于内部来源数据的可靠性，注册会计师不能直接采用，而应测试其可靠性。</span>

### 答案分析
本题强调了在实质性分析程序中，对于来自企业内部非财务部门的数据，注册会计师不能简单地直接使用，而要对其可靠性进行测试，这是符合实质性分析程序中对数据可靠性要求的考点内容的。所以答案围绕此考点展开是恰当的。

# **必备考点三:存货的监盘**
### 知识点总结
1. **概述**
    - **盘点和监盘**：被审计单位制定盘点计划，确定存货的数量和状况；CPA制定监盘计划，获取存货数量和状况的充分、适当审计证据 。
2. **目标**
    - **主要目标**：确保存货的存在、完整性、准确性、计价和分摊 。
    - **次要目标**：涉及权利和义务，需通过其他实质性审计程序实现 。
3. **监盘计划 - 考虑因素**
    - 考虑与存货相关的重大错报风险、与存货相关的内部控制的性质 。
    - 关注对存货盘点是否制定了适当程序并下达正确指令，若存在缺陷需提请调整，必要时实施其他审计程序 。
    - **存货盘点的时间安排**：若在资产负债表日以外盘点，需关注存货盘点日与财务报表日之间<span style="background:#fdbfff">存货变动是否得到恰当记录 。</span>
    - **考虑存货存放清单的完整性**
        - **分析程序**：关注仓库变动情况 。
        - **业务**：询问仓储和销售人员、检查入库单和出库单 。
        - **账簿**：查看租赁明细账和合同、固定资产明细账 。
    - **确定适当的监盘地点**
        - **选择监盘地点**：针对不预先通知、舞弊情况，可同一天对所有存货监盘或变更监盘地点 。
    - 考虑是否采用永续盘存制、是否需要专家协助 。 
### 知识点总结
1. **监盘程序**
    - **程序**
        - **观察盘点执行**：评价管理层记录和控制存货盘点结果的指令和程序，关注存货移动（设独立过渡区域，防止重盘或漏盘 ）、存货截止 。
        - **检查存货** 。
        - **执行抽盘**：采用局部事后不告知、双向抽盘，保证不可预见性 ；出现差异时，查明原因并更正，扩大检查范围，必要时重新盘点（限定范围 ）。
    - **结束时工作**
        - 再次观察盘点现场 。

### 知识点详解
对于第三方保管或控制的存货，相关审计程序包括：
1. 向持有被审计单位存货的第三方函证存货的数量和状况。
2. 实施检查或其他适合具体情况的审计程序：
    - 实施或安排其他注册会计师实施对第三方的存货监盘（如可行）。
    - 获取其他注册会计师或服务机构注册会计师针对用以保证存货得到恰当盘点和保管的内部控制的适当性而出具的报告。
    - 检查与第三方持有的存货相关的文件记录，如仓储单。
    - 当存货被作为抵押品时，要求其他机构或人员进行确认。

### 答题思路
存货期末按成本和可变现净值孰低原则计量，有迹象表明存货发生减值时，企业应于期末计算存货的可变现净值，计提存货跌价准备。所以对于存货的审计，除确定数量和单价外，还需对存货的状况获取审计证据，以识别需要计提跌价准备的存货项目。因此，仅关注存货的数量和单价是不全面的，对于第三方保管的原材料，还应当就其状况获取审计证据。



# 必备考点四:货币资金的审计
这是关于库存现金监盘的思维导图，以下是详细解读：
### 人员
 - **组成**：库存现金监盘参与人员包括出纳、会计主管和注册会计师（CPA ） 。
 - 出纳负责现金<span class="cloze-span">保管，是现金盘点的直接执行者；</span>
 - 会计主管对<span class="cloze-span">财务工作负有管理责任；</span>
 - CPA 执行审计<span class="cloze-span">监督职能，保证监盘过程合规、结果可靠。 </span>
### 盘点范围
 - **涵盖内容**：需盘点各部门所有的现金，包括<span class="cloze-span">零用金、找换金 ，以及已收到但未存入银行的现金。确保不遗漏任何现金存放点和现金类型，全面核实企业现金状况</span>。 
### 监盘时间
 - **时机选择**：最好实施突击性检查，一般选择<span class="cloze-span">上午上班前或下午下班</span>时 。此时现金业务活动相对较少，现金收支处于相对静止状态，能更准确反映现金实际结存情况，防止被审计单位提前准备、掩盖问题。 
 - **多处盘点**：当存在两处或两处以上现金存放点时，可以考虑同时<span class="cloze-span">进行盘点 。这样能有效防止现金在不同存放点之间转移作弊</span>，保证盘点结果真实可靠。 
### 监盘程序
 - **查账**：核对库存现金日记账与现金收付凭证 ，检查记账凭证与日记账记录是否相符，凭证是否真实、合法、完整，有无涂改、伪造等情况，确保现金收支记录准确。 
 - **查实**：对比现金实存数与库存现金日记账余额 ，通过实际清点现金，<span style="background:#fdbfff">确定账实是否一致。若存在差异，需进一步追查原因，如是否存在未入账的收支、白条抵库等问题。 </span>
 - **非资产负债表日监盘**：<span style="background:#fdbfff">若在非资产负债表日进行监盘，需对监盘日至资产负债表日之间的现金变动情况实施程序 。</span>比如通过检查这段期间的现金日记账、收付凭证等，将监盘日现金余额调节至资产负债表日余额，保证资产负债表日现金余额的准确性 。
 - 这是银行存款实质性程序的思维导图，具体解读如下：
### 验证银行账户完整性
 - **方法**：CPA（注册会计师）<span style="background:#fdbfff">需亲自观察并检查《已开立银行结算账户清单》 </span>，并结合其他相关细节测试。通过这种方式，确保被审计单位所有银行账户都已纳入审计范围，防止存在账外账户等情况。 
### 取得并检查
 - **对账单**
    - **获取要求**：<span style="background:#fdbfff">强调加盖印章、亲自取得并全过程控制 ，同时要观察登录、核对页面、检查信息一致性 。这能保证对账单来源可靠，防止被篡改。</span>
    - **核对内容**：需将对账单余额与银行日记账余额核对 ，以及银行对账单与银行询证函回函核对 ，检查双方记录是否一致，查找差异原因。 
 - **银行存款余额调节表**
    - **核对项目**：<span style="background:#fdbfff">检查调节后日记账余额与对账单余额是否一致 。</span>
    - **关注重点**：检查调节事项是否合理，关注长期未达账项有无异常，以及是否存在异常的领款事项 ，判断企业银行存款收支的真实性和准确性。 
### 函证
 - **范围**
    - **涵盖对象**：包括银行存款（含零余额和注销的账户 ）、借款及其他相关信息 。
    - **豁免情况**：当对财务报表不重要且与之相关的重大错报风险很低时可豁免函证 。 
 - **函证方式**：采用积极式函证 ，即要求银行无论函证内容是否相符都要回函，以获取更可靠审计证据。 
 - **函证控制**：对函证全过程进行控制 ，并调查不符事项 ，保证函证结果真实有效，对不符情况查明原因。 
## 例题

本题主要围绕银行存款实质性程序中对银行存款余额调节表的审查展开，旨在通过对调节表的检查以及对调节事项的核实，确保银行存款余额的准确性，发现可能存在的错报或舞弊情况。

### 知识点详解
1. **调节表加计数及余额核对**：注册会计师需要检查银行存款余额调节表中加计数的正确性，这是保证调节表计算准确的基础。同时，要核对调节后银行存款日记账余额与银行对账单余额是否一致。若两者不一致，可能存在记账错误或未达账项等问题，需进一步审查。
2. **调节事项检查**
    - **企付银未付款项**：对于企业已付款但银行尚未付款的事项，要检查被审计单位付款的原始凭证，确保付款业务的真实性和合规性。同时，检查该项付款是否在期后银行对账单上得以反映。在检查期后银行对账单时，要将对账单上记载的支票编号、金额等信息与被审计单位的支票存根进行核对，判断款项是否真正支付，***防止企业虚构付款业务或存在未及时入账的情况。通过这种方式，核实未达账项的真实性和准确性，保障银行存款余额的可靠性。***
	   ### 2. 企收银未收款项
		- 对于企业已收款但银行尚未收款的事项，需要检查被审计单位收款入账的原始凭证，以此确认企业记录的收款业务是否真实发生。同时，***检查该笔款项是否已在期后银行对账单上得以反映。如果在期后银行对账单中出现了该笔收款记录，说明只是正常的时间差异导致的未达账项***；若期后银行对账单中一直未出现该笔记录，则可能存在问题，如企业虚构收款业务或银行方面存在差错等，需要进一步调查核实。
	### 3. 银收企未收款项/银付企未付款项
	- 对于银行已收款但企业尚未收款以及银行已付款但企业尚未付款的事项，要检查收、付款项的内容及金额，确定是否为截止错报。截止错报是指将本应记录在本期的交易记录到了下期，或者将下期的交易提前记录到了本期。通过仔细核对款项的内容、金额以及相关的日期等信息，判断企业是否存在因截止错误而导致的银行存款余额不准确的情况。例如，如果银行在资产负债表日之前已经收到款项，但企业在资产负债表日之后才入账，就可能影响到当期银行存款余额的准确性和财务报表的真实性。

### 4. 大额或长期未达账项
- 如果企业的银行存款余额调节表存在大额或长期未达账项，注册会计师应追查原因并检查相应的支持文件。大额未达账项可能对财务报表产生重大影响，而长期未达账项则可能暗示存在潜在的问题，如资金被挪用、舞弊或账务处理错误等。通过追查原因和检查支持文件，判断是否为错报事项。如果确定是错报事项，应根据其对财务报表的影响程度，确定是否需要提请被审计单位进行调整。例如，如果发现大额未达账项是由于企业将款项错误地记入了其他账户，或者是因为银行方面的失误导致款项未及时到账，就需要要求被审计单位进行相应的账务调整或与银行沟通解决。

### 5. 关注长期未达账项与异常支付事项
- 关注长期未达账项，查看是否存在挪用资金等事项。长期未达账项可能是资金被挪用的一个信号，因为如果资金被非法挪用，可能会导致款项无法按时到账或支付，从而形成长期未达账项。同时，特别关注银付企未付、企付银未付中支付异常的领款事项，包括没有载明收款人、签字不全等支付事项。这些异常情况可能表明存在舞弊行为，例如，可能有人通过虚构支付事项来套取企业资金，或者是在支付过程中存在违规操作。注册会计师需要对这些异常事项保持高度警惕，进一步调查核实，以确定是否存在舞弊行为，并根据调查结果采取相应的审计措施。

### 总结
对银行存款余额调节表的审查是银行存款实质性程序的重要环节，注册会计师应严格按照上述要点进行检查，不放过任何可能影响银行存款余额准确性的因素，以获取充分、适当的审计证据，降低审计风险。

### 关于证据的可靠性
- **甲公司解释的局限性**：甲公司的解释属于口头证据。口头证据通常具有较高的主观性和易变性，其可靠性相对较低。被审计单位可能出于各种原因，如掩盖错误、粉饰财务状况等，而提供不真实或不准确的解释。因此，仅依靠甲公司的解释，无法为审计结论提供充分、适当的审计证据。

- **保持职业怀疑**：职业怀疑是注册会计师执行审计工作的重要态度。在面对甲公司的解释和提供的收款回单时，注册会计师不能轻易相信，而应保持谨慎和质疑的态度，充分考虑可能存在的风险和问题。
- **实施进一步审计程序**：为了获取充分、适当的审计证据，注册会计师需要实施进一步的审计程序。比如亲自到银行进行核实，这是一种较为直接和有效的方法。通过与银行直接沟通，查阅银行的原始记录和相关凭证，可以确认收款回单的真实性，以及款项的实际到账情况。此外，还可以向银行函证，要求银行提供关于该笔收款的详细信息，包括收款日期、金额、付款人等，以核实甲公司提供的信息是否准确。同时，检查与该笔收款相关的其他文件和记录，如销售合同、发货凭证等，以验证交易的真实性和完整性。通过实施这些进一步的审计程序，注册会计师可以更全面地了解情况，降低审计风险，确保审计结论的准确性和可靠性。、

# 集团审计

### 重要组成部分
- **具有财务重大性**：以集团指标为选定基准 ，通常达到集团指标15%上下的组成部分，具有财务重大性。针对此类组成部分，必须实施审计，这是唯一的应对措施 。
- **存在特别风险**：特定性质或情况可能使组成部分存在导致集团财务报表发生重大错报的特别风险，比如从事不同交易、新收购、存在特别风险、出现重大差异或重大变化等情况。对于这类组成部分，有多种可选审计应对方式，包括实施审计、对账户余额、交易或披露实施审计、实施特定的审计程序。 
### 不重要组成部分
- **常规程序**：一般情况下，对不重要组成部分在集团层面实施分析程序 。
- **补充程序**：若已执行的工作不能提供充分适当审计证据，就需要进一步采取措施，可选择实施审计、对账户余额、交易或披露实施审计、实施特定程序、实施审阅等 。 
这是关于集团财务报表审计中重要性相关概念的思维导图，解读如下：
### 集团的重要性
 - **负责主体**：集团项目组制定集团总体审计策略 。
 - **内容**：包括集团财务报表整体的重要性 ，以及集团特定类别的交易、账户余额或披露的重要性 ，且涵盖计划和实际执行层面。 
### 组成部分的重要性
 - **计划的重要性**
    - **负责主体**：集团项目组制定组成部分总体审计策略 ，前提是对组成部分实施审计或审阅 。
    - **执行标准**：单个组成部分重要性必须低于集团财务报表整体重要性 ，无需均分，无固定比例要求 ，且各组成部分重要性汇总可大于集团财务报表整体重要性 。 
 - **实际执行的重要性**：遵循“谁执行谁制定”原则，由组成部分注册会计师或集团项目组制定，集团项目组进行评价 。 
### 明显微小错报临界值
 - **负责主体**：由集团项目组确定 。
 - **处理方式**：超过临界值的错报需通报给集团项目组 。

### 解析
1. **答案解读**
    - A注册会计师仅依据资产总额、营业收入或利润总额超过设定金额来识别重要组成部分，做法不恰当。在集团审计中，识别重要组成部分不能只看财务重大性 。除了具有财务重大性的组成部分（比如资产、收入等规模大的部分 ），还需关注那些因自身特定性质或情况，可能使集团财务报表出现重大错报的组成部分。例如新收购的公司，业务整合可能存在风险；从事外汇业务的组成部分，汇率波动会带来不确定性等。
2. **考点 - 重要组成部分**
    - **财务重大性判断**：
      - **判断方法**：采用选定的基准×百分比的方式。比如以集团营业收入为基准，若某组成部分营业收入占集团营业收入一定比例（假设按常见的15% ，但实际是职业判断 ），就可能认定为重要组成部分。
      - **基准选择**：可从集团资产、负债、现金流量、利润总额、营业收入等指标里选。不同行业、集团特点下，合适的基准不同，像零售行业可能更关注营业收入，制造业可能更关注资产等。
      - **百分比确定**：是职业判断过程，15%只是示例，不同集团根据自身情况，这个比例可高可低。
    - **特定性质或情况导致的特别风险**：
      - 若组成部分从事外汇业务，因汇率波动频繁且难以准确预测，会给集团财务报表带来重大错报风险。
      - 新收购的组成部分，在业务、财务、内控等融合过程中，可能存在诸多不确定，像业务衔接不顺、财务数据整合有误等，导致重大错报。
      - 以前年度审计发现过特别风险的组成部分，说明其内部可能存在长期未解决的问题，后续仍可能影响财务报表准确性。
      - 内控、信息系统有重大差异或变化时，原有的风险控制机制可能失效，容易产生重大错报。 
3. **答题思路**
本题考查对重要组成部分概念的理解。解题关键在于牢记重要组成部分有财务重大性和特定性质或情况这两个判断维度 。看到题目中仅提及金额方面的判断标准，就要想到遗漏了特定性质或情况这一维度，从而判断做法不恰当并能准确阐述理由。 
```

(2)乙公司为重要组成部分，各项主要财务指标均占集团财务报表相关财务指标的50%以上。A注册会计师亲自担任组成部分注册会计师，选取乙公司财务报表中所有金额超过组成部分重要性的项目执行了审计工作，结果满意。 
```
### 答案解析
 - **结论**：答案判定原做法“不恰当” 。原因是乙公司作为各项主要财务指标占集团财务报表相关指标50%以上的组成部分，属于具有财务重大性的重要组成部分<span style="background:#d3f8b6">。按照审计准则要求，对于此类组成部分，应当对其财务信息执行审计程序。</span>而题目中仅表明选取金额超过组成部分重要性的项目执行审计工作，未全面涵盖对乙公司财务信息的审计，不符合规定。
### 考点及知识点详解
 - **考点**：重要组成部分——具有财务重大性。当单个组成部分的财务指标在集团中占比较大（如本题乙公司占比50%以上 ），就属于具有财务重大性的组成部分。
 - **知识点**：对于具有财务重大性的单个组成部分，集团项目组或代表集团项目组的组成部分注册会计师，必须运用该组成部分的重要性对其实施审计。这是因为此类组成部分对集团财务报表影响重大，全面审计其财务信息才能合理保证集团财务报表整体的准确性和可靠性。
### 答题思路解析
 - 在审计重要组成部分时，关键要区分组成部分类型。如果是具有财务重大性的组成部分，只能且必须实施审计程序，不能仅采取部分审计手段或其他程序。如果是特定性质可能导致特别风险的组成部分，审计程序相对灵活，可以实施审计、特定项目审计、特定审计程序中的一项或多项。本题乙公司属于具有财务重大性的组成部分，所以必须对其财务信息完整执行审计程序，原做法不符合要求。 


### 答案解析
 - **结论**：答案判定原做法“不恰当”。原因是组成部分重要性的确定主体应为集团项目组，而不是由组成部分注册会计师确定后经集团项目组复核认可。即便A注册会计师评估认为丙公司的组成部分注册会计师具备专业胜任能力，也不能改变这一确定主体的规定。 
### 考点及知识点详解
 - **考点**：重要性。在集团审计中，组成部分重要性的确定是关键内容。
 - **知识点**：
    - **制定阶段**：在制定集团审计总体审计策略时，需确定组成部分重要性。
    - **制定者**：集团项目组基于集团审计目的，为组成部分确定重要性。这是因为集团项目组从整体把控审计风险，能综合考虑集团财务报表整体以及各组成部分对其的影响。
    - **制定前提**：组成部分注册会计师要对组成部分财务信息实施审计或审阅工作，需依据集团项目组确定的组成部分重要性开展后续工作。
    - **执行标准**：
        - 集团项目组设定的组成部分重要性低于集团财务报表整体重要性，确保各组成部分的错报汇总后不会影响集团财务报表整体的准确性。
        - 不同组成部分重要性可能不同，并非平均分配，要根据各组成部分的性质、规模等因素灵活确定。
        - 不采用将集团财务报表整体重要性按比例分配的方式确定组成部分重要性。
        - 不同组成部分重要性汇总数可能高于集团财务报表整体重要性，这是合理的，因为实际执行中各组成部分不会同时达到最大错报水平。
        - 组成部分注册会计师使用集团项目组确定的组成部分重要性，来评价识别出的未更正错报是否重大。 
### 答题思路解析
 - 在集团审计里，组成部分注册会计师负责对组成部分财务信息执行审计工作，但组成部分计划的重要性应由集团项目组确定，不能由组成部分注册会计师确定实际执行的重要性来替代计划重要性的确定。集团项目组还需对组成部分层面确定的实际执行重要性的适当性进行评价。本题中A注册会计师认可组成部分注册会计师确定的重要性，违背了上述准则要求，所以做法不恰当。 

(5)A注册会计师要求所有组成部分注册会计师汇报组成部分的控制缺陷和超过组成部分实际执行重要性的未更正错报，将其与集团层面的控制缺陷和未更正错报汇总评估后认为:甲集团公司不存在值得关注的内部控制缺陷;集团财务报表不存在重大错报。
### 答案解析
 - **结论**：答案判定原做法“不恰当”。在集团财务报表审计中，A注册会计师要求组成部分注册会计师汇报的内容不完整。仅汇报组成部分的控制缺陷和超过组成部分实际执行重要性的未更正错报是不够的，还应当要求汇报超过集团层面明显微小错报临界值的错报 。
### 考点及知识点详解
 - **考点**：明显微小错报。这是集团审计中关于错报评估的重要概念。
 - **知识点**：
    - 注册会计师（集团项目组）需要设定明显微小错报临界值。其意义在于区分哪些错报是极其微小，不需要累积和进一步考虑的。超过该临界值的错报，就不能被视为对集团财务报表明显微小的错报，需要进行累积和评估 。
    - 组成部分注册会计师在对组成部分财务信息审计过程中，一旦识别出超过集团层面设定的明显微小错报临界值的错报，就需要通报给集团项目组。这是因为这些错报汇总起来可能对集团财务报表整体产生影响，集团项目组需要综合考虑这些错报来评估集团财务报表是否存在重大错报 。
### 答题思路解析
 - 集团财务报表审计类似于扩展后的单体财务报表审计，都需要关注累积超过一定临界值的错报。在集团审计情境下，这个临界值是以集团层面为准的明显微小错报临界值 。组成部分注册会计师应汇报的是超过集团层面明显微小错报临界值的错报，而不是仅依据组成部分实际执行重要性来汇报，A注册会计师的要求不符合审计准则规定，所以做法不恰当。 


# 会计估计
以下是根据思维导图整理的有序列表：
1. **会计估计审计**
    - **风险评估程序**
        - **了解**：被审计单位及其环境、适用的财务报告编制基础和内部控制体系各要素
        - **复核**：以前期间会计估计的结果（重新估计）
            - **目的**：有助于识别和评估本期重大错报风险
            - **具体要求**
                - 固有风险较高：更详细复核
                - 常规或重复交易：分析程序
                - 关注数据和假设
            - 差异并不必然表明上期财务报表存在错报
        - **确定**：是否需要专门技能或知识
    - **识别和评估**
        - 估计不确定性的程度
        - 复杂性、主观性和其他固有风险因素影响程度
        - **涉及方面**
            - 方法、假设、数据
            - 管理层点估计和披露
        - 评估是否为特别风险
    - **应对**
        - **测试管理层会计估计**
            - 从截至审计报告日发生的事项获取审计证据
            - 评价计量方法
            - 评价重大假设
            - 评价使用的数据
            - 评价管理层作出的会计估计
        - **作出注册会计师的点估计或区间估计**
            - **点估计**：差异构成错报
            - **区间估计**
                - 区间估计可能不包括管理层的点估计
                - 错报为管理层的点估计与注册会计师的区间估计之间的最小差异
                - 区间可能较大：重新考虑区间估计范围内的金额的合理性
                - 评价与会计估计相关的披露 

## 例题
### 题目
ABC会计师事务所的A注册会计师负责审计甲公司2017年度财务报表。与会计估计审计相关的部分事项如下：针对第（1）至（5）项，逐项指出A注册会计师的做法是否恰当。如不恰当，简要说明理由。
1. A注册会计师就管理层确认的某项预计负债作出了区间估计，该区间包括了甲公司所有可能承担的赔偿金额。管理层确认的预计负债处于该区间内，A注册会计师据此认可了管理层确认的金额。 
2. 2016年末，管理层对某项应收款项全额计提了坏账准备，因2017年全额收回该款项，管理层转回了相应的坏账准备。A注册会计师据此认为2016年度财务报表存在重大错报，要求管理层更正2017年度财务报表的对应数据。 
3. 管理层编制盈利预测以评价递延所得税资产的可回收性。A注册会计师向管理层询问了盈利预测中使用的假设的依据，并对盈利预测实施了重新计算，结果满意，据此认可了管理层的评价。 
4. 2017年末，甲公司确认与产品保修义务相关的预计负债400万元，A注册会计师作出的点估计为600万元。管理层将预计负债调增至550万元。A注册会计师将未调整的50万元作为错报累积。 
5. A注册会计师认为应收账款坏账准备的计提存在特别风险，在了解相关内部控制后，对应收账款坏账准备实施了实质性分析程序，结果满意，据此认可了管理层计提的金额。

## 持续经营假设
1. **适当地**
    - **不涵盖**：持续经营假设 - 否定意见
    - **涵盖**
        - **是否存在重大不确定性**
            - **不存在**：无保留意见
            - **存在**
                - **充分披露**：无保留意见 + 与持续经营相关的重大不确定性
                - **未充分披露**：保留意见或否定意见 ，无法表示意见 
2. **替代基础（非持续基础）**：无保留意见 + 强调事项段 


### 题目
ABC会计师事务所的A注册会计师负责审计多家上市公司2018年度财务报表，与审计报告相关的事项如下：针对第（1）至（4）项，逐项指出A注册会计师的做法是否恰当。如不恰当，简要说明理由。
1. 甲公司的某重要子公司将于2019年清算，其2018年度财务报表以非持续经营为基础编制，甲公司管理层在合并财务报表附注中披露了该情况，A注册会计师拟在对甲公司合并财务报表出具的审计报告中增加强调事项段，提醒财务报表使用者关注该事项。 
2. A注册会计师实施审计程序后，认为乙公司管理层运用持续经营假设编制财务报表不适当，拟在审计报告中增加与持续经营相关的重大不确定性部分，提醒财务报表使用者关注。 
3. 因丙公司名义违规对外提供多项担保，导致丙公司2018年发生多起重大诉讼，多个银行账户被冻结，业务停止，主要客户和员工流失。管理层在2018年度财务报表中确认了大额预计负债，并披露了持续经营存在的重大不确定性，A注册会计师认为存在多项对财务报表整体具有重要影响的重大不确定性，拟对丙公司财务报表发表无法表示意见。 
4. 丁公司的某重要子公司2018年末处于停产状态，其核心技术人员已离职成立新公司，与丁公司竞争且占据主要市场份额。管理层拟在三年内自主研发替代性技术，基于该假设编制的预计未来现金流量现值显示，收购该子公司形成的大额商誉不存在减值。A注册会计师认为技术研发成功的可能性存在重大不确定性，在无保留意见的审计报告中增加了强调事项段提醒报表使用者关注。 

### 解析
| 序号  | 做法                                            | 是否恰当 | 理由                                                                                 | 考点                   |     |
| --- | --------------------------------------------- | ---- | ---------------------------------------------------------------------------------- | -------------------- | --- |

### 题目
ABC会计师事务所的A注册会计师负责审计甲公司2019年度财务报表。与关联方审计相关的部分事项如下：针对下述第（1）至（5）项，逐项指出A注册会计师的做法是否恰当。如不恰当，简要说明理由。
1. A注册会计师通过询问关联方名称、关联方自上期以来发生的变化、是否与关联方发生交易以及交易的类型、定价政策和目的，向管理层了解关联方关系及其交易，并在审计工作底稿中记录了询问情况。 
2. 甲公司与关联方乙公司签订协议，向其转让一幢办公楼并售后回租。A注册会计师认为该项交易影响重大，查阅了相关协议，评价了交易的商业合理性和交易价格的公允性，向管理层询问核实了交易条款，检查了收款记录和过户文件，结果满意，据此认可了该交易的会计处理和披露。 
3. 甲公司管理层在财务报表附注中披露，其向控股股东控制的集团财务公司的借款为公平交易。A注册会计师将该借款的利率与同期银行借款利率进行了比较，未发现差异，据此认可了管理层的披露。 
4. 因会计人员疏忽，甲公司将与关联方丙公司的交易披露为与关联方丁公司的交易。A注册会计师要求管理层作出调整，并检查了其他关联方交易的披露是否存在类似情况，结果满意，因而未与治理层沟通该项事项。 
5. A注册会计师怀疑甲公司2019年新增加的大客户戊公司是甲公司的关联方。管理层解释戊公司是甲公司为开拓某市场而签约的经销商，并非关联方。A注册会计师查阅了相关的经销合同，向戊公司函证了销售金额和应收账款余额，检查了出库物流单据和签收记录，结果满意，认可了管理层的解释。

### 题目（1）解析
 - **做法判断**：A注册会计师仅询问关联方名称、变化、交易相关情况，做法不恰当。
 - **理由**：关联方审计时，询问内容应全面。除已询问的方面外，关联方的特征（如经营性质、规模等 ）和关联方关系的性质（如控制、共同控制或重大影响等 ）对于准确评估关联方交易对财务报表的影响十分关键。了解这些信息有助于判断交易是否公允、是否存在潜在风险等，A注册会计师遗漏这些内容，无法全面掌握关联方情况。
 - **考点关联**：本题考查关联方审计的范围和方法，强调审计过程中获取信息应全面，不能遗漏关键要素。

### 题目（2）解析
 - **做法判断**：A注册会计师对甲公司与乙公司售后回租交易的审计做法不恰当。
 - **理由**：在关联方交易审计中，除了对交易本身的商业合理性、价格公允性等进行审查外，交易的授权和批准情况是重要的审计内容。未经恰当授权和批准的关联方交易，可能因缺乏有效监督而存在欺诈风险，或者不符合公司既定的内部控制政策和程序，A注册会计师未获取此方面审计证据，审计程序不完整。 
 - **考点关联**：本题涉及关联方交易的审计程序和评估，突出在审计关联方交易时需全面涵盖各类审计程序，确保交易合规性。

### 题目（3）解析
 - **做法判断**：A注册会计师仅比较借款利率就认可管理层关于公平交易的披露，做法不恰当。
 - **理由**：评估关联方交易的公允性不能仅依赖利率这一因素。借款合同中的还款期限、担保情况、违约条款等其他条款和条件，都会对借款交易的实际成本和风险产生影响，进而影响交易的公允性。A注册会计师忽略这些因素，无法准确判断交易是否真正公平。 
 - **考点关联**：本题考查关联方交易的公允性评估，说明在审计关联方交易公允性时，需综合考虑多方面合同条款，不能片面判断。
 
### 题目（4）解析
 - **做法判断**：A注册会计师发现披露错误并处理后未与治理层沟通，做法恰当。
 - **理由**：注册会计师在发现关联方交易披露错误后，要求管理层作出调整并检查其他关联方交易披露情况，确保了错报得到纠正且不存在类似问题。由于该错误是疏忽导致且已及时处理，从重要性和必要性角度，未与治理层沟通在这种情况下是合理的，符合审计中关于关联方交易披露及与治理层沟通的相关要求。 
 - **考点关联**：本题围绕关联方交易的披露以及注册会计师的沟通职责，体现注册会计师需根据具体情况判断是否与治理层沟通关联方交易相关事项。

### 题目（5）解析
 - **做法判断**：A注册会计师仅通过查阅合同、函证和检查物流单据等就认可管理层解释，做法不恰当。
 - **理由**：确认关联方关系较为复杂，需要对双方的所有权结构、管理层关系、主要业务活动和交易行为等进行深入调查分析。比如双方管理层是否存在亲属关系、是否存在间接控制关系等。仅通过目前实施的程序，只能确认交易相关事实，无法深入核实双方本质关联，审计程序不足以证实戊公司与甲公司是否存在关联方关系。

## 审计报告
### 非无保留意见
1. **存在重大错报**
    - 重大且具有广泛性：否定意见
    - 重大但不具有广泛性：保留意见
2. **无法获取充分、适当审计证据**
    - 重大且具有广泛性：无法表示意见
    - 重大但不具有广泛性：保留意见
3. **注册会计师认为必要的其他事项**
    - 导致非无保留意见的其他事项
    - 与持续经营相关的重大不确定性
    - 对应数据和比较财务报表相关事项
### 强调事项
1. **审计基础**
    - 导致保留意见的事项段
    - 导致否定意见的事项段
    - 导致无法表示意见的事项段
2. **不确定事项**
    - 与持续经营相关的重大不确定性
    - 会计估计的不确定性
    - 未决诉讼或仲裁
### 其他事项
1. **对应数据**
    - 上期导致非无保留意见的事项本期影响重大：非无保留意见
    - 上期导致非无保留意见的事项本期影响不重大：无保留意见
    - 上期存在重大错报，未更正也未重述：保留或否定意见
    - 上期财务报表未经审计：在其他事项段说明
    - 上期财务报表已由前任注册会计师审计：在其他事项段提及前任审计情况
2. **持续经营假设对审计报告影响**
    - **适当性**
        - 不适当：持续经营假设 - 否定意见
        - 适当：替代基础（非持续基础） - 无保留意见 + 强调事项段
    - **是否存在重大不确定性**
        - 不存在：无保留意见
        - 存在
            - 充分披露：无保留意见 + 与持续经营相关的重大不确定性
            - 未充分披露：保留或否定意见 ，无法表示意见

### 解析总结
1. **其他信息错报处理**：关注审计师对其他信息重大错报的沟通及后续措施，明确不影响审计报告本身时无需重新出具报告。
2. **审计受限与持续经营**：理解无法执行审计工作出具无法表示意见报告的情形，以及持续经营不确定性披露与报告内容的关系。
3. **报表错报意见类型**：掌握财务报表存在不同程度错报时，应准确选择否定意见等合适的审计意见类型。 
4. **关键审计事项**：把握关键审计事项的性质，避免对财务报表单一要素单独发表意见。 
5. **强调事项段运用**：清楚强调事项段针对已在财务报告中披露事项，防止错误使用。

    - 客观性影响：应当尽量避免在同一渠道交叉实施项目质量复核（特殊情况除外）

- **提供咨询的人员** ：排除项目合伙人和项目组其他成员

### 题目
ABC会计师事务所的质量管理制度部分内容摘录如下，针对第（1）至（5）项，逐项指出ABC会计师事务所的质量管理制度的内容是否违反《会计师事务所质量管理准则第5101号—业务质量管理》和《会计师事务所质量管理准则第5102号—项目质量复核》的相关规定。如违反，简要说明理由。
1. 合伙人的收益以各业务部门为单位进行分配，具体分配方案由各业务部门制定，原则上以执业质量为首要考核指标。 
2. 事务所质量管理部对上市实体审计业务的关键审计合伙人轮换进行实时监控，并每年对轮换情况实施复核。其他审计业务的关键审计合伙人轮换由各业务部门自行监控及复核。 
3. 项目合伙人对项目管理和项目质量承担总体责任，项目质量复核人员对项目质量复核的实施承担总体责任。

### 题目（1）解析
 - **判断**：违反规定。
 - **理由**：根据相关准则，会计师事务所应在全所范围内统一进行合伙人的考核和收益分配，不能以各业务部门为单位各自为政。而且在考核和分配时，应综合考虑执业质量、管理能力、经营业绩、社会声誉等多项指标，不能将承接和执行业务的收入或利润作为首要指标，也不应以部门等作为利润中心进行收益分配 。本题中ABC会计师事务所的做法不符合这些要求。
 - **考点关联**：本题主要涉及质量管理体系中治理层和领导层关于合伙人考核与收益分配的规定，强调全所统一分配以及综合考量多因素的重要性。

### 题目（2）解析
 - **判断**：违反规定。
 - **理由**：会计师事务所不仅要对上市实体审计业务的关键审计合伙人轮换进行监控和复核，对于公众利益实体审计业务都应如此操作。公众利益实体范围广泛，除上市公司外，还包括法律法规规定按上市公司审计独立性要求接受审计的实体等。ABC会计师事务所仅对上市实体进行监控，而让各业务部门自行处理其他审计业务关键审计合伙人轮换的监控复核，不符合准则要求 。
 - **考点关联**：本题考查质量管理体系中相关职业道德要求，突出对公众利益实体审计业务关键审计合伙人轮换监控的全面性规定。

### 题目（3）解析
 - **判断**：不违反规定。
 - **理由**：在审计项目中，项目合伙人本来就应当对管理和实现审计项目的高质量承担总体责任，虽然可将部分任务分配给项目组其他成员，但仍需指导、监督并复核其工作；项目质量复核人员也应对项目质量复核的实施承担总体责任，这种职责划分符合准则规定 。
 - **考点关联**：本题涉及对财务报表审计实施的质量管理中，审计项目合伙人管理和实现审计质量的领导责任以及项目质量复核的责任规定，明确不同角色在质量管理中的责任分工。

### 题目（4）解析
 - **判断**：不违反规定。
 - **理由**：如果项目组成员或项目质量复核人员担任内部质量检查人员，就相当于自己检查自己的工作，难以保证检查的客观性和公正性。所以规定他们不得担任检查人员，是合理且符合质量管理体系要求的 。
 - **考点关联**：本题围绕质量管理体系中内部质量检查人员的独立性要求，体现保证质量检查客观公正的原则。

### 题目（5）解析
 - **判断**：违反规定。
 - **理由**：为保证项目质量复核工作的客观性，项目合伙人和项目组其他成员既不能担任本项目的项目质量复核人员，也不能为本项目的项目质量复核提供协助。因为他们参与项目执行工作，提供协助可能会影响复核的客观性，ABC会计师事务所的规定不符合准则要求 。
 - **考点关联**：本题考查项目质量复核中项目质量复核人员的委派和资质要求，重点在于保证复核过程的客观性。
