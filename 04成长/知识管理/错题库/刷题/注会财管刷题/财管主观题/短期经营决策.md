# 租赁决策与短期经营决策结合题解题指引
## 一、知识关联
```
本题融合租赁决策与短期经营决策 。需先算关键设备现有利用情况，再借租赁或购买增设备扩产能（租赁决策），扩产后决定是否接追加订单（短期经营决策，看成本收益） 。租赁与购买决策类似投资项目资本预算，分期初、期间、期末现金流，复杂程度稍低，需关注租赁相关处理。
```

## 二、解题步骤
### （一）约束资源利用（针对设备产能等约束）
1. 算各产品单位约束资源边际贡献 ，公式一般为（单位边际贡献÷单位产品消耗约束资源量 ）。
2. 按单位约束资源边际贡献从大到小排产品，优先产贡献大的 。
3. 检查有无剩余约束资源，若有，按上述排序接着生产 。

### （二）租赁决策
1. 判断租赁性质，如短期低价值租赁或其他类型 。
2. 算现金流：
    - 短期低价值租赁：租金可抵税，考虑税后租金现金流 。
    - 其他租赁：租金不能抵税，但每年计提折旧可抵税；期末若取得所有权，变现收益要纳税；未取得所有权，残值损失可抵税 。

### 一、约束资源利用（基于材料一，生产线为约束资源 ）
1. 计算单位约束资源边际贡献 
    - 甲：单位边际贡献 = 9 - 6 = 3（万元），单位约束资源边际贡献 = 3÷1 = 3（万元/小时） 
    - 乙：单位边际贡献 = 15 - 11 = 4（万元），单位约束资源边际贡献 = 4÷2 = 2（万元/小时） 
    - 丙：单位边际贡献 = 12 - 9 = 3（万元），单位约束资源边际贡献 = 3÷3 = 1（万元/小时） 
2. 排序：甲（3）＞乙（2）＞丙（1），优先安排甲生产，再乙，最后丙 。
3. 算各产品最大产量及消耗工时 
    - 甲：市场销量1200台，消耗工时1200×1 = 1200小时 
    - 乙：市场销量800台，消耗工时800×2 = 1600小时 
    - 前两者共消耗1200 + 1600 = 2800小时，剩余3100 - 2800 = 300小时，可生产丙：300÷3 = 100台 ，即丙最大按约束资源能产100台（结合材料三，后续新增设备后有变化 ）

### 二、租赁决策（材料二，方案1自行购买、方案2租赁 ）
1. 方案1（自行购买） 
    - 判断：不属于短期低价值租赁，按常规租赁决策处理 
    - 算现金流： 
        - 年初：借款6000万（流入），购设备（流出6000万，抵消后净额0 ）；支付维护费30万（流出，考虑税后：30×(1 - 25%) = 22.5万 ） 
        - 每年：折旧 = 6000÷6 = 1000万，折旧抵税 = 1000×25% = 250万（流入 ）；利息支出 = 6000×8% = 480万，利息抵税 = 480×25% = 120万（流入 ） 
        - 期末：变现价值1200万，账面价值 = 6000 - 1000×5 = 1000万，变现收益纳税 = (1200 - 1000)×25% = 50万，实际流入1200 - 50 = 1150万 
2. 方案2（租赁） 
    - 判断：租赁期5年，所有权不转让，看是否短期低价值（题目未明确，按常规处理，租金不能抵税，考虑手续费等 ） 
    - 算现金流： 
        - 年初：支付首付1100万 + 手续费380万 = 1480万（流出，税后：因租金等抵税情况，需看后续 ）；每年年初支付租金1100万（除第一年已付），共4次 。 
        - 每年：租金不能直接抵税，无折旧（设备由租赁公司维护 ），但手续费等可能需分摊？或按题目，主要考虑租金、手续费的税后影响，以及期末无所有权，无残值相关税 。 税前有担保利率8%，作为折现率，需算各年现金流现值，比较方案1和方案2净现值，选优 。

### 三、追加订单决策（材料三，丙空调追加订单 ）
1. 情形1：剩余生产力无法转移，追加100台，专属成本40万 
    - 相关收入 = 100×10 = 1000万 
    - 相关成本 = 100×9（单位变动成本） + 40 = 940万 
    - 边际贡献 = 1000 - 940 = 60万，若为正，可接受 
2. 情形2：剩余生产能力可外租（租金60万），追加150台，原销量减50台 
    - 相关收入 = 150×10 = 1500万 
    - 相关成本： 
        - 变动成本 = 150×9 + 50×9（原销量减少的变动成本损失） = 1800万 
        - 机会成本（外租租金） = 60万 
    - 边际贡献 = 1500 - (1800 + 60) = -360万，若为负，不接受 （具体需结合约束资源利用后实际剩余产能，这里假设按材料三新增设备后丙可产1100台，原按材料一约束只能产100台，新增后有剩余，需精准计算剩余产能再分析，此处为思路 ）
