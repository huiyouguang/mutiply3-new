# 期权价值评估解题思路与要点
## 一、题目特征与考查内容
本题围绕**期权价值评估**展开，需按流程完成：  
利用**套期保值原理算看涨期权价值**→借**平价定理算看跌期权价值**→算**期权投资策略到期净损益**，是典型期权投资评价流程。  
新增的**实物期权**内容，虽难度大，但首年考查大概率常规，建议以掌握教材例题为目标 。  

## 二、解题套路（步骤）
1. **算股价**：计算上行股价 \( S_u \)、下行股价 \( S_d \)  
2. **定期权价值**：确定到期日股价上行/下行时的期权价值 \( C_u \)（看涨）、\( C_d \)（看涨）  
3. **套期保值/风险中性相关计算**  
    - 套期保值原理：算套期保值比率（买股数量）\( H \)、借款金额 \( D \)  
    - 风险中性原理：算上行概率、下行概率  
4. **算期权价值**：结合上述结果，算出看涨期权价值  
5. **算看跌期权价值**：用平价定理，公式为 \( \text{看涨期权价格 } C - \text{看跌期权价格 } P = \text{标的资产价格 } S - \text{执行价格现值 } PV(X) \)，变形可求看跌期权价值 。

# 「考点 4」可转换公司债券


# 典型例题 - 2021年真题计算题解题思路 
1. **做题步骤**：做大题先读题目要求，带着问题看题干。  
2. **考点说明**：考查可转债发行相关内容，虽考查频率不高但有难度，需重视；可转债条款常出主观题。  
3. **解题套路**：  
    - **投资角度**：  
        ①计算可转债底线价值（转股前：纯债券价值；转股后：纯债券价值或转换价值 ）  
        ②判断投资者是否购买可转债  
        ③判断投资者是否转股及转股时间  
        ④计算可转债期望收益  
    - **筹资角度**：  
        ①计算可转债底线价值  
        ②计算税前资本成本（转股前利息现值 + 转股后底线价值现值 = 发行价格，内插法算资本成本 ）  
        ③判断筹资方案是否可行（资本成本介于债、股成本之间 ）  
        ④若不可行，思考修改方案使其可行
