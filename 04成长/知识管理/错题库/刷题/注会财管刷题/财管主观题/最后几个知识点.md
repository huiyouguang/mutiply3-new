### 全面预算之现金预算考点梳理
本题聚焦全面预算里的现金预算内容，近年考查频率较高，核心围绕判断下期现金是否充足及资金调配展开，解题逻辑与步骤如下：  

#### 一、现金预算核心目的  
企业编制现金预算，关键是明确下期现金是否满足经营需求（是否≥最低现金余额），据此决策：现金不足则借款，有结余可短期投资。  


#### 二、关键业务逻辑与规则  
1. **借款与利息**：  
    - 借款通常发生在**期初**（如第一季度借款，即季初借入 ），当期会产生利息费用；若按季付息，借款当季需计提并支付利息。  
    - 现金结余时，优先偿还已有借款，剩余再考虑短期投资。  

2. **编制前置步骤**：  
    需依次完成系列营业预算，为现金预算提供数据支撑：  
    ```mermaid
    graph LR
    A[销售预算] --> B[生产预算]
    B --> C[直接材料预算]
    B --> D[直接人工预算]
    B --> E[制造费用预算]
    C & D & E --> F[产品成本预算]
    A --> G[销售和管理费用预算]
    ```  


#### 三、解题套路（步骤）  
1. **营业预算编制**：依次完成销售、生产、直接材料、直接人工、制造费用、产品成本、销售和管理费用预算，明确各项经营活动的成本与收入框架。  
2. **营业现金流确认**：结合赊销、赊购比例，计算经营活动实际现金流入（收入到账）与流出（采购付款等 ）。  
3. **资本支出识别**：确认是否有设备购置等资本性支出（题目一般明确给出 ）。  
4. **所得税预估**：按题目给定的所得税金额或计算规则，确定现金流出。  
5. **借款利息计算**：依据借款时间（期初/期末）、利率及付息周期，算出当期需支付的利息金额。  
6. **现金余缺测算**：通过“现金收入 - 现金支出（含经营、资本、所得税、利息等）”，结合期末最低现金余额要求，判断现金多余或不足。  
7. **资金调配决策**：  
    - 现金不足：计算需新增借款（考虑期初借款规则 ）；  
    - 现金多余：优先偿还已有借款（冲减负债），剩余资金可短期投资。  
8. **期末现金余额计算**：完成资金调配后，算出最终期末现金余额，验证是否符合企业留存要求。  


通过以上步骤，可系统完成现金预算编制，精准规划企业资金流，保障经营与投融资活动有序开展。

# 「考点 10」股利分配

# 2019年真题计算题  
甲公司在外流通的普通股加权平均股数2000万股，2018年净利润为5000万元，其董事会正在讨论相关分配方案：  
- **方案一**：每10股发放现金股利6元  
- **方案二**：每10股发放股票股利10股。  

预计股权登记日：2019年10月20日；现金红利到账日：2019年10月21日；除权（息）日：2019年10月21日。假设甲公司股票2019年10月20日收盘价为30元。  

**要求**：  
(1) 如果使用方案一，计算甲公司每股收益、每股股利，如果通过股票回购将等额现金支付给股东，回购价格每股30元，设计股票回购方案，并简述现金股利和股票回购的相同点和不同点。  
(2) 若采用方案二，计算发放股票股利后甲公司每股收益，每股除权参考价。如果通过股票分割方式达到同样的每股收益，设计股票分割方案，并简述股票股利与股票分割的相同点和不同点。
这是**品职教育CPA财管主观题突破**中的2019年真题计算题，涉及**甲公司股利分配方案**（现金股利、股票股利 ），需计算每股收益、每股股利、除权参考价等，还对比现金股利与股票回购、股票股利与股票分割的异同 。核心实体是甲公司股利分配相关财务计算与概念对比，用于CPA财管科目学习，帮助理解股利分配形式及对财务指标、股东权益影响。

### （1）方案一相关解答
- 每股收益 = 5000÷2000 = 2.5元/股 
- 每股股利 = 6÷10 = 0.6元 
- 股票回购方案：现金股利总额 = 0.6×2000 = 1200万元，回购股数 = 1200÷30 = 40万股，即按30元/股回购40万股 。 
- 相同点：公司现金减少，股东权益减少 。 
- 不同点：
    - 税负：现金股利缴股利收益税，回购缴资本利得税 。 
    - 性质：现金股利是股利支付方式，回购不是 。 
    - 影响：现金股利收益稳定，回购影响不确定；现金股利不减少股数，回购减少股数、提股价、降被收购风险；回购传递股价低估信号 。 

### （2）方案二相关解答
- 每股收益 = 5000÷（2000 + 2000×10÷10） = 1.25元/股 
- 每股除权参考价 = 30÷（1 + 10÷10） = 15元/股 
- 股票分割方案：一股分割成2股 。 
- 相同点：
    - 股东权益内部变动，不影响资产和负债 。 
    - 增加普通股股数，股东权益及每股市场价值不变 。 
    - 盈利总额和市盈率不变时，每股收益、每股市价降低 。 
- 不同点：
    - 结构：股票分割不改变股东权益结构，股票股利会改变 。 
    - 面值：股票分割使面值变小，股票股利不影响 。 
    - 适用：股票分割用于股价暴涨难下跌时，股票股利用于股价涨幅不大时 。 
    - 性质：股票分割不是股利支付方式，股票股利是 。

# 考点 11 杠杆系数

# 题目内容
甲公司乙部门只生产一种产品，投资额25000万元，2019年销售500万件。该产品单价25元，单位变动成本资料如下：

|项目|单位变动成本（元）|
| ---- | ---- |
|直接材料|3|
|直接人工|4|
|变动制造费用|2|
|变动销售费用|1|
|合计|10|

该产品目前盈亏临界点作业率20%，现有产能已满负荷运转。因产品供不应求，为提高销量，公司经可行性研究，2020年拟增加50000万元投资。新产能投入运营后，每年增加2700万元固定成本。假设公司产销平衡，不考虑企业所得税。

## 要求：
（1）计算乙部门2019年税前投资报酬率；假设产能扩张不影响产品单位边际贡献，为达到2019年税前投资报酬率水平，计算2020年应实现的销量。
（2）计算乙部门2019年经营杠杆系数；假设产能扩张不影响产品单位边际贡献，为达到2019年经营杠杆水平，计算2020年应实现的销量。


### （1）2019年税前投资报酬率及2020年目标销量计算
- **2019年税前投资报酬率**：  
    - 先算固定成本：盈亏临界点作业率20%，即盈亏临界点销量 = \(500×20\% = 100\)（万件），单位边际贡献 = \(25 - 10 = 15\)（元/件），所以固定成本 = \(100×15 = 1500\)（万元） 。  
    - 税前投资报酬率 = \(\frac{500×15 - 1500}{25000} = \frac{7500 - 1500}{25000} = \frac{6000}{25000} = 24\%\) （或用边际贡献×（1 - 盈亏临界点作业率）计算：\(\frac{500×15×(1 - 20\%)}{25000} = 24\%\) ）。  
 - **2020年目标销量**：  
    设2020年销量为\(X\)，2020年投资额变为\(25000 + 50000 = 75000\)万元，固定成本变为\(1500 + 2700 = 4200\)万元 。  
    根据税前投资报酬率公式：\(\frac{X×15 - 4200}{75000} = 24\%\)，解得\(X = 1480\)（万件） 。  


### （2）2019年经营杠杆系数及2020年目标销量计算
- **2019年经营杠杆系数**：  
    经营杠杆系数（\(DOL\)） = \(\frac{边际贡献}{边际贡献 - 固定成本} = \frac{500×15}{500×15 - 1500} = \frac{7500}{6000} = 1.25\) （或用\(\frac{1}{1 - 盈亏临界点作业率} = \frac{1}{1 - 20\%} = 1.25\) ）。  
 - **2020年目标销量**：  
    设2020年销量为\(Y\)，2020年固定成本\(4200\)万元 。  
    根据经营杠杆系数公式：\(\frac{Y×15}{Y×15 - 4200} = 1.25\)，解得\(Y = 1400\)（万件） 。  

综上，（1）2019年税前投资报酬率为\(\boldsymbol{24\%}\)，2020年应实现销量\(\boldsymbol{1480}\)万件；（2）2019年经营杠杆系数为\(\boldsymbol{1.25}\)，2020年应实现销量\(\boldsymbol{1400}\)万件。

# 2022年真题计算题 变动成本法
甲公司是一家小家电制造企业，只产销X产品。公司采用完全成本法核算产品成本，每年按实际产量分配固定制造费用，采用先进先出法进行存货计价。  

2022年末，总经理收到会计部门转来的利润表后，比较了2022年和2021年的税前经营利润，发现了一个奇怪的现象：2022年销量比2021年增加24%，但税前经营利润却大幅下降。公司管理层就此问题进行了讨论，财务经理建议采用变动成本法重新核算产品成本然后再做分析。相关资料如下表所示：  


### 2021年和2022年X产品实际数量  
|项目|2021年|2022年|  
| ---- | ---- | ---- |  
|年初存货数量|0|12000|  
|本年产量|62000|50000|  
|本年销量|50000|62000|  
|年末存货数量|12000|0|  


### 2021年和2022年X产品生产费用实际数据  
|项目|金额|  
| ---- | ---- |  
|单价|58元/件|  
|单位直接材料|6元/件|  
|单位直接人工|7元/件|  
|单位变动制造费用|2元/件|  
|固定制造费用总额|1860000元|  
|单位变动销售和管理费用|1元/件|  
|固定销售和管理费用总额|220000元|  


### 要求：  
(1) 分别计算2021年、2022年X产品在完全成本法下和变动成本法下的单位产品成本。  
(2) 分别计算2021年、2022年X产品在完全成本法下和变动成本法下的税前经营利润及其差异，并解释差异产生的原因。  
(3) 简要说明财务经理建议采用变动成本法的理由。
### （1）单位产品成本计算  
- **2021年**：  
    - 完全成本法：单位产品成本 = 直接材料 + 直接人工 + 变动制造费用 + 固定制造费用分配额 = \(6 + 7 + 2 + 1860000÷62000 = 45\)（元/件）  
    - 变动成本法：单位产品成本 = 直接材料 + 直接人工 + 变动制造费用 = \(6 + 7 + 2 = 15\)（元/件）  
- **2022年**：  
    - 完全成本法：单位产品成本 = \(6 + 7 + 2 + 1860000÷50000 = 52.2\)（元/件）  
    - 变动成本法：单位产品成本 = \(6 + 7 + 2 = 15\)（元/件）  


### （2）税前经营利润及差异分析  
#### 2021年  
- 完全成本法：税前经营利润 = \((58 - 45)×50000 - 50000×1 - 220000 = 380000\)（元）  
- 变动成本法：税前经营利润 = \((58 - 15 - 1)×50000 - 1860000 - 220000 = 20000\)（元）  
- 差异：\(380000 - 20000 = 360000\)（元）  
- 原因：完全成本法下，年末存货（12000件）吸收固定制造费用 \(12000×(1860000÷62000) = 360000\) 元，未计入当期损益；变动成本法下固定制造费用全额计入当期，导致利润差异。  


#### 2022年  
- 完全成本法：税前经营利润 = \(58×62000 - 12000×45 - 50000×52.2 - 62000×1 - 220000 = 164000\)（元）  
- 变动成本法：税前经营利润 = \((58 - 15 - 1)×62000 - 1860000 - 220000 = 524000\)（元）  
- 差异：\(164000 - 524000 = -360000\)（元）  
- 原因：完全成本法下，年初存货（12000件）释放固定制造费用 \(12000×(1860000÷62000) = 360000\) 元，使当期成本增加、利润减少；变动成本法下无此影响，导致利润差异。  


### （3）财务经理建议采用变动成本法的理由  
1. 消除完全成本法下通过调节生产、库存操纵利润的问题，利润更真实反映经营成果。  
2. 引导管理者重视销售，避免盲目生产，契合市场导向。  
3. 清晰划分变动与固定成本责任，便于部门业绩考核。  
4. 揭示利润与业务量的线性关系，支持经营决策（如盈亏平衡分析、销量 - 利润测算 ）。  
5. 聚焦变动成本控制，简化成本管理，适配日常运营管控。  
6. 减少固定制造费用分配环节，成本计算更简洁高效。

# 经济增加值

### 题目  
甲公司是家保健品代工企业（后述业务显示实际为充分竞争行业商业类企业，以最终业务描述为准 ），乙部门负责国内市场生产销售（本题核心考核为甲公司整体业绩，围绕经济增加值 ）。甲公司为充分竞争行业和领域的商业类企业，采用经济增加值作为业绩评价指标，控股股东对其2021年度经营业绩评价相关资料如下：  
（1）2021年，甲公司净利润为50亿元，费用化利息支出为10亿元，资本化利息支出为20亿元；研发费用为18亿元，当期确认为无形资产的开发支出为2亿元。  
（2）2021年甲公司年末无息负债为500亿元，年初无息负债为300亿元；年末带息负债为900亿元，年初带息负债为500亿元；年末所有者权益为400亿元，年初所有者权益为300亿元；年末在建工程为160亿元，年初在建工程为200亿元。  

**要求**：  
（1）计算2021年甲公司调整后的税后净营业利润。  
（2）计算2021年甲公司调整后资本和经济增加值。  
（3）回答经济增加值作为业绩评价指标的优点和缺点。  


### 答案  
#### （1）调整后的税后净营业利润  
调整公式：调整后的税后净营业利润 = 净利润 +（利息支出 + 研发费用调整项）×（1 - 所得税税率）  
其中，利息支出含费用化与资本化利息（本题未提及所得税税率，默认按25% ，结合解题过程逻辑 ），研发费用调整项为研发费用 + 确认为无形资产的开发支出 = \(18 + 2 = 20\) 亿元；利息支出总额 = \(10 + 20 = 30\) 亿元。  

代入计算：  
调整后的税后净营业利润 = \(50 + (30 + 20)×(1 - 25\%) = 50 + 50×0.75 = 50 + 37.5 = 87.5\)（亿元）  
（注：原解题过程利息支出计算可能简化，按题目资料准确应为费用化10 + 资本化20，研发调整项18 + 2，若解题过程“(10 + 18 + 2)” ，推测是将资本化利息视为已计入相关资产成本，在计算税后净营业利润调整时，利息支出取费用化部分10，结合研发调整项18 + 2 ，则：  
调整后的税后净营业利润 = \(50 + (10 + 18 + 2)×(1 - 25\%) = 50 + 30×0.75 = 50 + 22.5 = 72.5\)（亿元）—— 与原解题过程一致，以原逻辑为准，即利息支出取费用化10，研发调整含研发费用18和开发支出2 ）  


#### （2）调整后资本和经济增加值  
- **调整后资本**：  
调整后资本 = 平均所有者权益 + 平均带息负债 - 平均在建工程  
    - 平均所有者权益 = \((300 + 400)÷2 = 350\)（亿元）  
    - 平均带息负债 = \((900 + 500)÷2 = 700\)（亿元）  
    - 平均在建工程 = \((160 + 200)÷2 = 180\)（亿元）  
    调整后资本 = \(350 + 700 - 180 = 870\)（亿元）  

- **平均资本成本率**：  
甲公司为充分竞争行业商业类企业，股权资本成本率取6.5%；债务资本成本率 = 利息支出总额÷平均带息负债 = \((10 + 20)÷700 ≈ 4.3\%\)（原解题过程简化为费用化利息10，若按题目资料含资本化20，债务利息支出总额30 ）。  
按原解题过程逻辑（利息支出取费用化10 ，可能题目隐含简化 ）：  
债务资本成本率 = \(10÷700 ≈ 1.43\%\)（但原解题过程为 \((10 + 20)÷700 = 4.3\%\) ，再考虑税后：\(4.3\%×(1 - 25\%)\) ）  
平均资本成本率 = 债务资本成本率×（平均带息负债÷调整后资本）×（1 - 所得税税率） + 股权资本成本率×（平均所有者权益÷调整后资本）  
= \(4.3\%×(700÷1050)×(1 - 25\%) + 6.5\%×(350÷1050) ≈ 4.32\%\)  
因甲公司为非工业企业，2021年资产负债率（77.8% ）在75%-80% ，平均资本成本率上浮0.2% ，即 \(4.32\% + 0.2\% = 4.52\%\)  

- **经济增加值**：  
经济增加值 = 调整后的税后净营业利润 - 调整后资本×平均资本成本率  
= \(72.5 - 870×4.52\% ≈ 72.5 - 39.324 = 33.176\)（亿元）  


#### （3）经济增加值优缺点  
- **优点**：  
    - 直接关联股东财富创造，追求经济增加值即追求股东价值最大化。  
    - 不只是业绩评价指标，更是全面财务管理与薪酬激励体系框架，推动企业关注长期价值。  
- **缺点**：  
    - 无法直接比较不同规模企业业绩。  
    - 对处于不同成长阶段企业的经营业绩反映有偏差，成长期与成熟期企业经济增加值不具直接可比性。  
    - 计算中对净利润调整（如研发、利息等处理 ）及资本成本确定（不同行业、企业资本成本估算争议大 ）存在较多主观判断与争议 。  


（注：解题过程中利息支出、研发费用调整的具体口径，需严格结合题目隐含逻辑与教材规范，本题按原解题过程还原，实际考试需精准把握经济增加值调整规则。 ）

# 题目
甲公司是一家制造业企业，主业处于关系国家安全、国民经济命脉的重要行业和关键领域，资产通用性较好。上级单位参考《中央企业负责人经营业绩考核办法》采用经济增加值对其业绩进行计量和评价。相关资料如下：  
（1）甲公司2022年净利润150亿元；利息支出总额45.98亿元，其中资本化利息支出31.98亿元；研发费用20亿元，当期确认为无形资产的开发支出10亿元。  
（2）甲公司2022年资产负债表相关数据如下（单位：亿元 ）：  

|项目|2022年初|2022年末|  
| ---- | ---- | ---- |  
|在建工程|260|340|  
|带息负债|1000|1200|  
|负债总额|1200|1425|  
|所有者权益|800|1000|  
（3）甲公司2021年经济增加值106亿元，税后净营业利润176.4亿元，调整后资本1600亿元，平均资本成本为4.4%。  
（4）甲公司2022年股权资本成本为5.5%，企业所得税税率为25%。假设不考虑企业所得税加计扣除及其他税费的影响。  

**要求**：  
（1）计算甲公司2022年税后净营业利润、调整后资本、平均资本成本和经济增加值。  
（2）计算甲公司2022年与2021年经济增加值的差异额，并使用连环替代法，按照税后净营业利润、调整后资本、平均资本成本的顺序，分别计算三个因素变动对2022年经济增加值的影响额。  
（3）简要说明采用经济增加值进行业绩评价的优点和缺点。  


# 答案  
### （1）2022年核心指标计算  
- **税后净营业利润**：  
  税后净营业利润 = 净利润 +（费用化利息支出 + 研发费用调整项）×（1 - 所得税税率）  
  费用化利息支出 = 利息支出总额 - 资本化利息支出 = \(45.98 - 31.98 = 14\)（亿元）  
  研发费用调整项 = 研发费用 + 确认为无形资产的开发支出 = \(20 + 10 = 30\)（亿元）  
  代入公式：  
  税后净营业利润 = \(150 + (14 + 30)×(1 - 25\%) = 150 + 44×0.75 = 150 + 33 = 183\)（亿元）  

- **调整后资本**：  
  调整后资本 = 平均带息负债 + 平均所有者权益 - 平均在建工程  
  平均带息负债 = \((1000 + 1200)÷2 = 1100\)（亿元）  
  平均所有者权益 = \((800 + 1000)÷2 = 900\)（亿元）  
  平均在建工程 = \((260 + 340)÷2 = 300\)（亿元）  
  调整后资本 = \(1100 + 900 - 300 = 1700\)（亿元）  

- **平均资本成本**：  
  平均资本成本 = 债务资本成本×（平均带息负债÷调整后资本）×（1 - 所得税税率） + 股权资本成本×（平均所有者权益÷调整后资本）  
  债务资本成本 = 费用化利息支出÷平均带息负债 = \(14÷1100 ≈ 1.27\%\)（注：原答案简化逻辑，按利息支出总额对应平均带息负债计算债务资本成本，即 \(45.98÷1100\) ，但结合经济增加值调整规则，实际应为费用化利息影响，此处按原答案公式 ）  
  代入原答案逻辑公式：  
  平均资本成本 = \(\frac{45.98}{1100}×\frac{1100}{1100 + 900}×(1 - 25\%) + 5.5\%×\frac{900}{1100 + 900} = 4.2\%\)  

- **经济增加值**：  
  经济增加值 = 税后净营业利润 - 调整后资本×平均资本成本  
  = \(183 - 1700×4.2\% = 183 - 71.4 = 111.6\)（亿元）  


### （2）经济增加值差异及连环替代分析  
- **差异额**：2022年与2021年经济增加值差异 = \(111.6 - 106 = 5.6\)（亿元）  

- **连环替代法**：  
  - 基期（2021年）经济增加值：\(176.4 - 1600×4.4\% = 106\)（亿元） —— ①  
  - 替代税后净营业利润：\(183 - 1600×4.4\% = 183 - 70.4 = 112.6\)（亿元） —— ②  
  - 替代调整后资本：\(183 - 1700×4.4\% = 183 - 74.8 = 108.2\)（亿元） —— ③  
  - 替代平均资本成本：\(183 - 1700×4.2\% = 111.6\)（亿元） —— ④  

  - 税后净营业利润变动影响：② - ① = \(112.6 - 106 = 6.6\)（亿元）  
  - 调整后资本变动影响：③ - ② = \(108.2 - 112.6 = -4.4\)（亿元）  
  - 平均资本成本变动影响：④ - ③ = \(111.6 - 108.2 = 3.4\)（亿元）  


### （3）经济增加值评价优缺点  
- **优点**：  
  1. 考虑全部资本成本，真实反映企业价值创造能力。  
  2. 统一企业、经营者、员工利益，激励创造价值。  
  3. 遏制盲目扩张，引导关注长期价值。  
  4. 融合资本预算、业绩评价、薪酬激励，构建全面管理框架。  
  5. 便于企业向投资人传递目标，助力投资人筛选企业，也是股票分析工具。  

- **缺点**：  
  1. 仅衡量当期或短期（1 - 3年 ）价值，难反映长远战略价值。  
  2. 依赖财务指标，无法综合评价运营效率与效果。  
  3. 不同行业、阶段、规模企业，调整项与资本成本差异大，计算复杂，影响可比性。  
  4. 绝对数指标，不便于不同规模企业业绩对比。  
  5. 不同成长阶段企业对比难，计算争议多，难建立统一规范，多用于企业内部历史分析与评价。
