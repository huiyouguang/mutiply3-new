#农产品增值税进项税额核定扣除

2.15  某公司属于以购进农产品为原料生产销售液体乳的增值税一般纳税人,其农产品实行增值税进项税额核定扣除办法,2023 年 12 月期间购进<font color="#2DC26B"> 2 000 吨</font>农产品,用于直接对外销售,当<span style="background:#d3f8b6">期损耗 100 吨,</span><span style="background:#d3f8b6">对外销售 1 500</span> 吨。已知农产品平均购买单价为 3 000 元 / 吨。
当期允许抵扣的进项税额为(    )。
A.391 115.40 元 
B.392 188.76 元
C.521 487.20 元 
D.540 000 元

  A  本题考查[[农产品增值税进项税额核定扣除]]的计算。选项 A 当选,具体计算过程如下: (1)以<font color="#2DC26B">购进农产品为原料生产销售液体乳及乳制品、酒及酒精、植物油的增值税一般纳税人,纳入农产品增值税进项税额核定扣除试点范围。</font>
(2)**当试点纳税人购进农产品直接销售时,农产品增值税进项税额按照以下方法核定扣除:** 损耗率 = 损耗数量 ÷ 购进数量 ×100%。
当期允许抵扣农产品增值税进项税额 =<span style="background:#d3f8b6"> 当期销售农产品数量 ÷(1- 损耗率)× 农产品平均购买单价 ÷(1+9%)×9% </span>

综上,损耗率 =100÷2 000×100%=5%,当期允许抵扣农产品增值税进项税额 =1 500÷(1-5%)×3 000÷(1+9%)×9%=391 115.4(元)。
选项 B 不当选,损耗率计算有误。选项 C 不当选,计算公式错用当期购进的农产品数量计算。
选项 D 不当选,未按核定扣除计算。

2.17  某餐饮服务企业为增值税一般纳税人,2023 年 5 月从农户处购进一批农产品用于餐饮服务,取得农产品销售发票上注明买价 300 万元,已知该企业的销售额符合加计抵减的条件且上期计提的加计抵减额仍余 0.5 万元未抵减完。假定不考虑其他因素,该企业当期可抵减的加计抵减额为(    )。
A.3.2 万元 
B.3.5 万元
C.4.05 万元 
D.5 万元


  A  本题考查农产品进项税额的计算抵扣和增值税的加计抵减政策。选项 A 当选,具体计算过程如下:
  (1)购进农产品用于餐饮服务,按照农产品销售发票上注明的买价和 9% 的扣除率计算抵扣进项税额,当期可抵扣的进项税额 =300×9%=27(万元); #农产品增值税抵扣
  (2)餐饮服务企业属于生活性服务业,2023 年适用加计抵减比例 10%,当期计提的加计抵减额 = 当期可抵扣进项税额 ×10%=27×10%=2.7(万元)。当期可抵减加计抵减额 = 上期末加计抵减额余额 + 当期计提加计抵减额 - 当期调减加计抵减额 =2.7+0.5=3.2(万元)。
  选项 B 不当选,误按照 10% 的扣除率计算购进农产品的进项税额。
  选项 C 不当选,误按照 5% 的加计抵减比例计算加计抵减额。选项 D 不当选,误按照 10% 的扣除率计算购进农产品的进项税额且误按照 15% 的加计抵减比例计算加计抵减额

“免抵”和“免抵退”是出口货物退（免）税的相关概念，主要应用于增值税一般纳税人的生产企业出口货物时的税收处理，以下为你详细解释： ### 1. 基本概念 - **免**：指对生产企业出口的自产货物，免征本企业生产销售环节增值税。也就是出口货物不需要缴纳销项税额。例如，一家生产企业将生产的产品出口到国外，对于出口这一销售行为，不需要计算缴纳增值税销项税。 - **抵**：指生产企业出口自产货物所耗用的原材料、零部件、燃料、动力等所含应予退还的进项税额，抵顶内销货物的应纳税额 。简单来说，企业在国内采购原材料等取得的进项税额，原本可以用于抵扣内销货物产生的应纳税额，同时也可以用于抵扣出口货物对应的进项税额。当出口货物有进项税额时，就可以用这部分进项税额去抵减内销货物的应纳税额。 - **退**：当生产企业出口自产货物在当期内应抵顶的进项税额大于内销应纳税额时，对未抵顶完的部分予以退税。即如果企业出口货物对应的进项税额在抵减内销货物应纳税额后还有剩余，这部分剩余的进项税额就可以申请退税。 ### 2. “免抵退”计算步骤 以一般贸易出口且生产企业有内销和外销业务的情况为例： 1. **计算当期不得免征和抵扣税额**： 当期不得免征和抵扣税额 = 当期出口货物离岸价×外汇人民币折合率×（出口货物适用税率 - 出口货物退税率） 这部分税额是由于出口货物适用的税率和退税率不一致，不能享受退税和抵扣的部分，需要做进项税额转出处理，增加出口货物的成本。 2. **计算当期应纳税额**： 当期应纳税额 = 当期内销货物的销项税额 - （当期进项税额 - 当期不得免征和抵扣税额）-上期留抵税额 如果计算结果为正数，说明企业当期内销货物的应纳税额大于可以抵扣的进项税额，需要缴纳增值税；如果为负数，说明企业有留抵税额。 3. **计算当期免抵退税额**： 当期免抵退税额 = 当期出口货物离岸价×外汇人民币折合率×出口货物退税率 这是企业按照规定可以享受的最大退税额度。 4. **确定当期应退税额和免抵税额**： - 若当期期末留抵税额（当期应纳税额计算结果的绝对值）≤当期免抵退税额，则当期应退税额 = 当期期末留抵税额，当期免抵税额 = 当期免抵退税额 - 当期应退税额。 - 若当期期末留抵税额＞当期免抵退税额，则当期应退税额 = 当期免抵退税额，当期免抵税额 = 0，结转下期继续抵扣的税额 = 当期期末留抵税额 - 当期应退税额 。 “免抵”实际上是在计算过程中，出口货物进项税额对内销货物应纳税额的抵减过程；“免抵退”则是一个完整的出口退税计算体系，综合考虑了出口货物的进项、内销的销项以及出口退税政策，确定企业最终可以获得的退税金额和免抵金额。

    - **题目**：某生产企业为增值税一般纳税人，2025年1月外购原材料取得增值税专用发票，注明增值税税额100万元并通过认证。当月内销货物取得不含税销售额150万元，外销货物取得收入115万美元（美元与人民币的比价为1：7)，该企业适用增值税税率为13%，出口退税率为9%。该企业1月免抵的增值税为（　）万元。
    - **错误原因**：对增值税免抵税额计算方法不熟悉，未正确计算各步骤税额。
    - **正确解法**：
        - 计算当期不得免征和抵扣税额：\(115×7×(13\% - 9\%) = 32.2\)（万元）
        - 计算当期应纳税额：当期内销货物销项税额\(150×13\% = 19.5\)万元，当期应纳税额\(= 19.5 - (100 - 32.2)= - 48.3\)万元
        - 计算当期免抵退税额：\(115×7×9\% = 72.45\)万元
        - 确定当期应退税额和免抵税额：因为当期期末留抵税额\(48.3\)万元小于当期免抵退税额\(72.45\)万元，所以当期应退税额\( = 48.3\)万元，当期免抵税额\( = 72.45 - 48.3 = 24.15\)万元。
2. **题目3**
    - **题目**：某服装厂为增值税一般纳税人，出口货物的征税税率为13%，退税税率为9%。2025年2月外购棉布一批，取得的增值税专用发票注明价款200万元，增值税26万元，货已入库。当月进口料件一批，海关核定的计税价格25万美元，已按购进法向税务机关办理了《生产企业进料加工贸易免税证明》。当月出口服装的离岸价格75万美元，内销服装不含税销售额80万元。该服装厂上期期末留抵税额5万元。假设美元比人民币的汇率为1：6.4，服装厂进料加工复出口符合相关规定。该服装厂当期应退税额（　　）万元。
    - **错误原因**：计算失误，在计算各环节税额时出现偏差。
    - **正确解法**：
        - 计算当期“免、抵、退”税不得免征和抵扣税额：\((75 - 25)×6.4×(13\% - 9\%) = 12.8\)万元
        - 计算当期应纳税额：内销货物销项税额\(80×13\% = 10.4\)万元，当期应纳税额\( = 10.4 - (26 - 12.8) - 5 = - 7.8\)万元
        - 计算当期“免、抵、退”税额：\((75 - 25)×6.4×9\% = 28.8\)万元
        - 确定当期应退税额：因为当期期末留抵税额\(7.8\)万元≤当期“免、抵、退”税额\(28.8\)万元，所以当期应退税额\( =\)当期期末留抵税额\( = 7.8\)万元。

### 二、知识点总结
1. **“免、抵、退”税计算关键步骤**
    - 计算“免、抵、退”税不得免征和抵扣税额抵减额（涉及免税进口料件时）：免税进口料件的组成计税价格×（出口货物征税税率 - 出口货物退税税率）
    - 计算“免、抵、退”税不得免征和抵扣税额：当期出口货物离岸价×外汇人民币折合率×（出口货物征税税率 - 出口货物退税税率） - “免、抵、退”税不得免征和抵扣税额抵减额
    - 计算当期应纳税额：当期内销货物的销项税额 - （当期进项税额 - “免、抵、退”税不得免征和抵扣税额） - 上期留抵税额
    - 计算“免、抵、退”税额抵减额（涉及免税进口料件时）：免税进口料件的组成计税价格×出口货物退税率
    - 计算出口货物“免、抵、退”税额：当期出口货物离岸价×外汇人民币折合率×出口货物退税率 - “免、抵、退”税额抵减额
    - 确定当期应退税额：比较当期期末留抵税额和当期“免、抵、退”税额，若当期期末留抵税额≤当期“免、抵、退”税额，则当期应退税额 = 当期期末留抵税额；若当期期末留抵税额＞当期“免、抵、退”税额，则当期应退税额 = 当期“免、抵、退”税额。
2. **易错点提醒**
    - 注意各公式中数据的准确代入，特别是涉及外币换算时，汇率使用要正确。
    - 区分清楚各个税额的含义和计算顺序，如不得免征和抵扣税额与应纳税额的计算关系。
    - 在有免税进口料件的情况下，相关抵减额的计算和应用不能遗漏或错误。

### 题目
某外贸公司2025年2月份购进及出口电饭煲情况如下：(1)第一次购进电饭煲500台，不含税单价150元/台；第二次购进电饭煲200台，不含税单价148元/台（均取得增值税专用发票)；(2)将两次外购的电饭煲全部报关出口，离岸单价25美元/台，此笔出口已收汇并作销售处理。（美元与人民币比价为1：7，电饭煲出口退税率为13%)。该笔出口业务应退增值税为（　）元。

### 答案及解析
- **答案**：D
- **解析**：外贸企业出口货物退还增值税应依据购进出口货物的增值税专用发票上所注明的金额和出口货物所适用的退税率计算。先计算购进货物的总金额，即第一次购进金额\(500×150 = 75000\)元，第二次购进金额\(200×148 = 29600\)元，两者相加得到\(75000 + 29600 = 104600\)元。然后用总金额乘以退税率，可得应退增值税\(104600×13\% = 13598\)元。

### 知识点总结
- 外贸企业出口退税计算方法：与生产企业的“免、抵、退”税计算不同，外贸企业出口货物退还增值税是依据购进出口货物的增值税专用发票上注明的金额和出口货物所适用的退税率来计算应退税额，公式为：应退税额 = 购进出口货物的不含税金额×出口货物退税率。
- 注意事项：
    - 准确获取购进货物的金额信息，包括不同批次购进的数量和单价，分别计算金额后汇总。
    - 明确出口货物所适用的退税率，不同商品的退税率可能不同，要正确选用。
    - 对于涉及外币的情况，要注意汇率的使用，将外币金额换算为人民币时要确保汇率使用正确，但本题中离岸单价在计算退税时未用到，属于干扰信息，要注意区分题目中的相关和无关信息。

### 一、错题题目
生产企业出口的视同自产货物，可以实行“免、抵、退”税管理办法。下列货物视同自产货物的有（　　）。
A. 外购的与本企业所生产的货物名称、性能相同，使用本企业注册商标，并出口给进口本企业自产货物的境外单位或个人的货物
B. 外购的与本企业所生产的货物配套出口，不经过本企业加工或组装，出口后能直接与本企业自产货物组合成成套设备，并出口给进口本企业自产货物的境外单位或个人的货物
C. 外购的与本企业所生产的货物配套出口，出口给进口本企业自产货物的境外单位或个人的，用于维修本企业出口的自产货物的零部件
D. 委托加工收回的与本企业所生产的货物名称、性能不同，出口给进口本企业自产货物的境外单位或个人的货物

### 二、错误原因分析
对生产企业视同自产货物的认定标准掌握不准确。误选了选项D，没有正确理解委托加工收回货物视同自产货物的条件，即货物名称、性能需与本企业所生产货物相同（或用本企业生产货物再委托深加工收回），而选项D中明确说明货物名称、性能不同，不符合视同自产货物的规定。

### 三、知识点总结
1. **视同自产货物的认定标准**
    - **外购货物视同自产**：
        - 外购的与本企业所生产的货物名称、性能相同，使用本企业注册商标，并出口给进口本企业自产货物的境外单位或个人的货物（选项A）。
        - 外购的与本企业所生产的货物配套出口，不经过本企业加工或组装，出口后能直接与本企业自产货物组合成成套设备，并出口给进口本企业自产货物的境外单位或个人的货物（选项B）。
        - 外购的与本企业所生产的货物配套出口，出口给进口本企业自产货物的境外单位或个人的，用于维修本企业出口的自产货物的零部件（选项C）。
    - **委托加工货物视同自产**：
        - 与本企业所生产的货物名称、性能相同，或者是用本企业生产的货物再委托深加工收回的货物。
        - 出口给进口本企业自产货物的境外单位或个人。
        - 委托方与受托方必须签订委托加工协议，且主要原材料必须由委托方提供，受托方不垫付资金，只收取加工费，开具加工费（含代垫的辅助材料）的增值税专用发票。
2. **重要性**：准确判断视同自产货物，对于生产企业正确适用“免、抵、退”税管理办法，合理享受出口退税政策至关重要。若对视同自产货物认定错误，可能导致退税申报不准确，引发税务风险或无法充分享受应有的税收优惠。

### 题目
某农机生产企业为增值税一般纳税人。2025年5月，该企业有以下业务：
- 向各地农机销售公司销售农机整机，开具的增值税专用发票上注明金额800万元。
- 向各地农机修配站销售农机零配件，取得含税收入120万元。
- 购进钢材等材料取得的增值税专用发票上注明税额68万元。
- 取得运输企业（增值税一般纳税人）开具的增值税专用发票注明不含税的运费10万元。取得的发票均已在当月通过主管税务机关认证。

该企业当月应纳增值税（  ）万元。

A. 85.81
B. 26.81
C. 17.81
D. 16.91

### 答案
D

### 答案解析
- **销项税额**：
    - 销售农机整机适用9%的税率，销项税额为\(800×9\% = 72\)万元。
    - 销售农机零配件适用13%的税率，因为取得的是含税收入，所以要先换算为不含税收入，销项税额为\(120÷(1 + 13\%)×13\%≈13.81\)万元。
    - 两者相加，销项税额总计\(72 + 13.81 = 85.81\)万元。
- **进项税额**：
    - 购进钢材等材料的进项税额为68万元。
    - 运输费用的进项税额为\(10×9\% = 0.9\)万元。
    - 进项税额合计\(68 + 0.9 = 68.9\)万元。
- **应纳增值税**：
    - 应纳增值税 = 销项税额 - 进项税额，即\(85.81 - 68.9 = 16.91\)万元。
