#不得抵扣进项税情形

	
以下是对该增值税进项税额抵扣问题的系统整理，采用结构化排版以便清晰理解：  


### **一、核心考点：增值税进项税额不得抵扣的情形**  
**关键规则**：用于**免税项目**的购进货物、劳务、服务等，其进项税额不得抵扣。  
**原理**：免税项目无销项税额，若抵扣进项税会导致增值税链条断裂，违背“征多少扣多少”原则。  
### **二、选项详细解析**  
| **选项**    | **是否可抵扣进项税** | **解析依据**                                                                                            |     |
| --------- | ------------ | --------------------------------------------------------------------------------------------------- | --- |
| **A**     | 可抵扣          | 政策允许资源回收企业按规定抵扣合规专票注明的税款，属于正常抵扣场景。                                                                  |     |
| **A**     | 可抵扣          | 政策允许资源回收企业按规定抵扣合规专票注明的税款，属于正常抵扣场景。                                                                  |     |
| **B（当选）** | 不可抵扣         | - **免税政策**：蔬菜、部分鲜活肉蛋产品属于免税项目（《财政部 国家税务总局关于免征蔬菜流通环节增值税有关问题的通知》等）。<br>- **不得抵扣原因**：购进货物用于免税销售，进项税需转出。 |     |
| **C**     | 可抵扣          | 实物赔付中自行购进的修理劳务属于“用于生产经营”，未用于免税或非应税项目，符合进项税抵扣条件。                                                     |     |
| **D**     | 可抵扣          | - 以货易货属于“视同销售”，需确认销项税额，同时因货物用于生产经营流通环节，进项税可正常抵扣。<br>- 增值税链条完整，符合抵扣规则。                               |     |

### **三、延伸总结：进项税额不得抵扣的常见场景**  
1. **用于免税/简易计税项目**：如免税农产品销售、简易计税的建筑服务等。  
2. **用于集体福利/个人消费**：如外购货物用于职工食堂、职工福利。  
3. **非正常损失**：因管理不善等原因导致货物被盗、霉烂变质等，进项税需转出。  
4. **购进旅客运输服务（除特定情况外）**、贷款服务、餐饮服务等。  
### **四、记忆技巧**  
**口诀**：免税简计福人消，非正常损失不可抵。  
**逻辑**：抓住“进项税抵扣需匹配销项税”原则，若购进用途不产生销项税（如免税）或脱离增值税链条（如个人消费），则不得抵扣。
