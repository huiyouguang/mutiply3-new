# 第一部分 民事法律关系的基本原理
| 民事法律关系 | 非民事法律关系 |
| ---- | ---- |
| 平等主体之间的 | 有关单位(如国家机关、企业事业单位等)在从事管理活动时产生的社会关系；人自己的活动，如起床、睡觉、散步、读书等 |
| 人事关系 | 不能产生私法效果的人身关系；受纯粹的道德或宗教调整的恋爱、同居、朋友、网友、同学、同事、同乡、师生关系等 |
| 财产关系 | 不能产生私法效果的财产关系：<br>1. 好意施惠：<br>（1）邀请同看演出、比赛或承诺陪同旅游；<br>（2）火车到站叫醒；<br>（3）顺路投递邮件或顺便帮邻居清扫积雪；<br>（4）无偿指路；<br>（5）青年志愿者做帮工；<br>（6）请客吃饭；<br>（7）搭便车(搭顺风车)。<br>好意施惠不存在合同违约责任或缔约过失责任问题，但有可能因符合法律规定而产生侵权责任问题。<br>2. 戏谑行为(开玩笑、说大话、吹牛皮) |

### 6.2 宣告死亡
[民法典第 46 - 53 条,共 8 条;民法典总则编解释第 16、17 条;2006 - 3 - 2,单;2017 - 3 - 52,多]

#### 6.2.1 表格
| 功能 | 重在维护失踪人的权益。<br />宣告失踪，既影响人身关系(如配偶、子女关系)，又影响财产关系(财产发生继承) |
| ---- | ---- |
| 情形 | 1. 一般情形，下落不明满 4 年<br />2. 意外事件，下落不明满 2 年<br />3. 意外事件且经有关机关证明(如公安机关)该自然人不可能生存，随时(0 年) |
| 程序 | 申请：利害关系人<br />受理：下落不明人住所地的基层法院受理；住所地与经常居住地不一致的，由经常居住地的基层人民法院受理 |
| 公告 | 1. 一般情形，公告期 1 年<br />2. 意外事件，公告期 1 年<br />3. 意外事件且经有关机关证明(如公安机关)，公告期 3 个月 |
| 宣告 | 被宣告死亡的人，法院宣告死亡的判决作出之日(判决书上所载明的文书作出日期)视为其死亡日期；因意外事件下落不明宣告死亡的，意外事件发生之日视为其死亡的日期 |
