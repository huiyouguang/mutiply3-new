## 2 💸 任务：清税缴税

 

| 序号  | 未结事项               | 类型    | 核心处理步骤                                                              | 备注    |
| :-- | :----------------- | :---- | :------------------------------------------------------------------ | ----- |
| 1   | **未注销税控设备监控**      | 🔴 强制 | **首要任务**<br><br>需携带税控盘（金税盘）、营业执照、经办人身份证前往办税服务厅，办理税控设备**注销 / 发行缴销**。 | 税局办理  |
| 2   | **发票验旧信息监控**       | 🔴 强制 | 完成税控注销后，通过电子税务局【发票验旧 / 缴销】模块，将结存空白发票作废、已开发票验旧，或前往大厅缴销。              | 税局办理  |
| 3   | **已发起未办结的待办任务**    | 🔴 强制 | 点击【未申报查询】或【清算申报查询】，查看具体待办任务（如往期未申报、未报验），逐项完成申报。                     |       |
| 4   | **清算申报监控**         | 🔴 强制 | **关键步骤**<br>点击【清算企业所得税申报】，填写《企业所得税清算申报表》，完成清算年度的申报。                 | 4月1日  |
| 5   | **未申报监控（税、规费...）** | 🔴 强制 | 点击【未申报查询】，确认是否存在往期增值税、附加税、个税等未申报记录，全部补申报完毕。                         | 4月1日  |
| 6   | **未报送财务报表信息**      | 🔴 强制 | 通过【未申报查询】功能，补报对应属期的财务报表（资产负债表、利润表）。                                 | 3月25日 |
| 7   | **个人所得税未申报监控**     | 🔴 强制 | 切换至**自然人电子税务局（个税 APP / 网页端）**，办理个体工商户经营所得个税或个人所得税汇算清缴。              | 3月31日 |
| 8   | **清算报备监控**         | 🔴 强制 | 完成企业所得税清算报备，确认清算开始时间及清算期间。                                          |       |
| 9   | **手续费未办结监控**       | 🟡 提示 | 若有代扣代缴个税手续费返还，可在自然人电子税务局申请；若无则忽略此条。                                 | 2月已收  |
| 10  | **税收违法行为处理监控**     | 🔴 强制 | 联系主管税务机关，核实是否存在税务违章记录（如逾期未申报处罚），按要求处理并缴纳罚款。                         |       |

### 2.1 子任务

- [ ] 第一步：登录电子税务局首页、清税申报套餐、选好日期
- [ ] 第二步：等1-2天，税务变成清算状态后
- [ ] 第三步：清算申报、申报当期税务
- [ ] 第四步：申请税务注销登记，再等1-2工作日就能领清税证明
- [ ] 第五步：清税证明下载方法：选个人业务、法人扫码登录、首页搜注销登记、输入企业统一社会信用代码、下载文书
